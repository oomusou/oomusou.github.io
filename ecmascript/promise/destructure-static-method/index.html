使用 Object Destructuring 從 Static Method 取出 Free Function

2020-07-09

Object Destructuring 除了能從 Object 取出 Free Function 外，也能從 Class 的 Static Method 取出 Free Function。

Version

macOS Catalina 10.15.5
VS Code 1.46.1
Quokka 1.0.307
ECMAScript 2015

Static Method

class Foo {
  static add(x, y) {
    return x + y
  }
}

let { add } = Foo
add(1, 1) // ?

add() 為 Foo class 的 static method，可使用 Object Destructuring 將 add() 取出成為 free function。

Axios

import axios from 'axios'

axios.get('http://localhost:3000')
  .then(x => x.data) // ?

get() 為 axios 的 static method，因此我們可直接使用 axios.get()。

import axios from 'axios'

let { get: ajax } = axios

ajax('http://localhost:3000')
  .then(x => x.data) // ?

既然 get() 為 axios 的 static method，我們就可使用 Object Destructuring 取出 get()，且順便 alias 成 ajax()。

Proimse

let f = x => Promise.resolve(x)

f(1) // ?

Promise.resolve() 亦為 static method，可建立 Fulfilled Promise。

let { resolve } = Promise

let f = x => resolve(x)

f(1) // ?

既然 resolve() 為 Promise 的 static method，理論上我們可使用 Object Destructuring 將 resolve() 取出成 free function。

但實際執行卻出現錯誤，why not ?

let { resolve } = Promise

let f = x => resolve.call(Promise, x)

f(1) // ?

因為 Promise.resolve() 內部使用了 this，抽成 free function 後，this 就成了 undefined，因此我們必須將 this 找回。

可將 Promise 傳入 .call() 重新定義 this。

let { resolve } = Promise

let f = x => resolve.bind(Promise)(x)

f(1) // ?

可將 Promise 傳入 .bind() 回傳新 function。

let resolve = x => Promise.resolve(x)

let f = x => resolve(x)

f(1) // ?

.call() 與 .bind() 雖然可行，最後還得傳入 Promise，顯然並不好用。

其實最簡單的方式就是自己包 resolve()，內部再呼叫 Promise.resolve()。

Wink-fp

import { resolve } from 'wink-fp'

let f = x => resolve(x)

f(1) // ?

Wink-fp 已經內建 resolve()，可直接使用。

Conclusion

  • 只是不使用 this 的 static method，就可以使用 Object Destructuring 取出 free function
  • 若遇到使用 this static method，只能自已建立新 function 將 static method 包起來
