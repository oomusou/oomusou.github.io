使用 entries() 讓 for of Loop 也有 Index

2020-07-14

ECMAScript 2015 支援 for of Loop 後，幾乎都用 for of 取代 for 了，但可否在 for of 內也能取得 Index 呢 ?

Version

macOS Catalina 10.15.5
VS Code 1.47.0
Quokka 1.0.307
ECMAScript 2015

for

let data = [1, 2, 3]

let f = a => {
  let result = []

  for(let i = 0; i < a.length; i++)
    if (i !== a.length -1)
      result.push(a[i])

  return result
}

f(data) // ?

f() 要取得不包含最後一筆的 Array，因此必須取得 index 判斷是否為 arr.length -1，傳統使用 for loop 實現。

for of

let data = [1, 2, 3]

let f = a => {
  let result = []

  for(let [i, x] of a.entries())
    if (i !== a.length -1)
      result.push(a[i])

  return result
}

f(data) // ?

若要改用 for of，最大的挑戰是 index 怎麼取得 ?

可使用 ES6 的 Array.prototype.entries() 取得 index，由於 entries() 回傳為 array，可使用 array destructuring 直接解構出 index。

Conclusion

  • 搭配 entries() 之後，就不用擔心 for of loop 取不到 index 了
