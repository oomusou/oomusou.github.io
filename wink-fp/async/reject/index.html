使用 reject() 建立 Rejected Promise

2019-09-20

reject() 是 Promise.reject() 的 Function 版本，特別適合 Function Composition。

Version

macOS Mojave 10.14.6
VS Code 1.38.1
Quokka 1.0.250
ECMAScript 2018
Ramda 0.26.1
Wink-fp 0.0.11

Promise.reject()

let fetchData = () => Promise.resolve('Hello World');

fetchData().then(x => {
  console.log(x);
  return Promise.reject(new Error('Something wrong'));
}).catch(e => console.log(e.message));

若想在 fulfilled handler 內自行建立 rejected promise，可先建立 Error object，再由 Promise.reject() 包成 rejected promise。

由於 catch() callback 中的 e 為 Error object，必須使用 e.message 才能取得 Something wrong。

Throw

let fetchData = async () => 'Hello World';

fetchData().then(x => {
  console.log(x);
  throw new Error('Something wrong');
}).catch(e => console.log(e.message));

也可在 fulfilled handler 中直接使用 throw 建立 rejected promise，其中 throw 就是 Promise.reject() 的 syntatic sugar。

Try Catch

let fetchData = async () => 'Hello World';

try {
  let x = await fetchData();
  console.log(x);
  throw new Error('Something wrong');
} catch(e) {
  console.log(e);
}

throw 除了能用在 fulfilled handler 外，也能用在 try catch block 中，如此更符合 imperative 與 synchronous 習慣。

但別忘了 throw 是 syntatic sugar，當 throw 搭配 await 時，它產生的是包裹 Error object 的 rejected promise。

Functional

import { pipe, then, otherwise, always } from 'ramda'; 
import { resolve, log } from 'wink-fp';

let fetchData = () => resolve('Hello World');

let reject = x => Promise.reject(x);

let onFulfilled = pipe(
  log,
  always('Something wrong'),
  reject
);

let fn = pipe(
  fetchData,
  then(onFulfilled),
  otherwise(log)
);

fn();

若想改以 FP 方式實現 promise chain，可搭配 Ramda 的 pipe()、then() 與 otherwise()。

resolve() 與 log() 則由 Wink-fp 提供。

第 6 行

let reject = x => Promise.reject(x);

至於沒人提供的 reject()，則自行由 Promise.reject() 包成 function。

Wink-fp

import { pipe, then, otherwise, always } from 'ramda'; 
import { resolve, log, reject } from 'wink-fp';

let fetchData = () => resolve('Hello World');

let onFulfilled = pipe(
  log,
  always('Something wrong'),
  reject
);

let fn = pipe(
  fetchData,
  then(onFulfilled),
  otherwise(log)
);

fn();

Wink-fp 已經提供 reject()，可直接使用。

reject()
a -> Promise<a>
將任意值包成 rejected promise

a：data 為任意值

Promise<a>：包成 promise 回傳

Function Composition

import { compose, then, otherwise, always } from 'ramda'; 
import { resolve, reject, log } from 'wink-fp';

let fetchData = () => resolve('Hello World');

let onFulfilled = compose(
  reject,
  always('Something wrong'),
  log
);

let fn = compose(
  otherwise(log),
  then(onFulfilled),
  fetchData
);

fn();

既然能使用 pipe() 達成 pipeline，反向使用 compose() 就是 function composition 了。

Conclusion

  • ECMAScript 很多 API 可能不適合 function composition，但我們可以自己包成 function
  • 透過 function 化的 reject()，我們就可組合其他 function 產生 rejected Promise
