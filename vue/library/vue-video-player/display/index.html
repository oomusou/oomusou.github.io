使用 Vue-video-player 顯示 HLS

2020-06-02

Safari 的 <video> tag 可直接支援 HLS，但 Chrome 與 Firefox 則不行，透過 Vue-video-player 則可在 Safari、Chrome 與 Firefox 都支援 HLS。

Version

macOS Catalina 10.15.4
WebStorm 2020.1.1
Vue 2.6.11
Vue-video-player 5.0.2
