Data Binding with SVG

2022-10-17

Even when use SVG in <template>, we can also use Data Binding with SVG.

Version

Vue 2.6.14
HTML 5

SVG

Blue Rectangle is drawn by SVG, but Hello World is Data Binding by Vue.
