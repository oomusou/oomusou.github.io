使用 create-vue 建立 Vue 2

2023-06-19

Vue CLI 是基於 webpack，而 create-vue 則是基於 Vite，速度快很多。

Version

create-vue 3.7.0
Vue 2.7.7

Create Project

$ npm create vue@2
$ cd vue2-lab
$ npm install
$ npm run dev
  • npm create vue@2 之後一定要帶版本號 @2
  • create-vue 並不會安裝相關套件，必須手動下 npm install 安裝
  • 改用 npm run dev 啟動專案
