Tailwind CSS 與 CSS Binding

2020-08-04

Tailwind 的 Utility 其實就是一堆預先定義好的 CSS Class，若要能根據 Model 動態改變 Utility 使其 Reactive，就要使用 Vue 的 CSS Binding。

Version

macOS Catalina 10.15.6
WebStorm 2020.2
Vue 2.6.11
Tailwind 1.6.0

String

Tailwind Rocks 使用 Tailwind 的 text-red-600 utility 成為紅色。

<template>
  <div :class="fontColor">Tailwind Rocks</div>
</template>

<script>
export default {
  name: 'App',
  data: () => ({
    fontColor: 'text-red-600'
  })
}
</script>

第 2 行

<div :class="fontColor">Tailwind Rocks</div>

並沒有在 class attribute 直接使用 text-red-600，而是綁定到 fontColor model。

第 8 行

data: () => ({
  fontColor: 'text-red-600'
})

將 fontColor 指定為 text-red-600。

Object

也可將 class 綁定到 Object。

Static Binding

<template>
  <div :class="{ 'text-red-600': true }">Tailwind Rocks</div>
</template>

<script>
export default {
  name: 'App'
}
</script>

第 2 行

<div :class="{ 'text-red-600': true }">Tailwind Rocks</div>

將 Object 直接綁定到 class attribute，key 為 Tailwind utility，value 為 true 則表示套用該 utility，false 則為否。

Binding to Property

<template>
  <div :class="{ 'text-red-600': isActive }">Tailwind Rocks</div>
</template>

<script>
export default {
  name: 'App',
  data: () => ({
    isActive: true
  })
}
</script>

第 2 行

<div :class="{ 'text-red-600': isActive }">Tailwind

Object 的 value 為 isActive model。

第 8 行

data: () => ({
  isActive: true
})

true 與 false 由 isActive model 決定。

Binding to Object

<template>
  <div :class="fontColor">Tailwind Rocks</div>
</template>

<script>
export default {
  name: 'App',
  data: () => ({
    fontColor: { 'text-red-600': true }
  })
}
</script>

第 2 行

<div :class="fontColor">Tailwind Rocks</div>

也可整個 Object model 與 class attribute 綁定。

第 8 行

data: () => ({
  fontColor: { 'text-red-600': true }
})

fontColor model 為 Object。

Binding to Function

<template>
  <div :class="changeFontColor()">Tailwind Rocks</div>
</template>

<script>
let changeFontColor = () => ({
  'text-red-600': true
})

export default {
  name: 'App',
  methods: {
    changeFontColor
  }
}
</script>

第 2 行

<div :class="changeFontColor()">Tailwind Rocks</div>

若要最靈活，就是將 class attribute 直接綁定到 function。

Event binding 時 method 之後可省略 ()，但 css binding 時 method 之後一定要 ()

第 6 行

let changeFontColor = () => ({
  'text-red-600': true
})

該 function 的唯一要求是要回傳 Object。

Point-free

<template>
  <div :class="changeFontColor()">Tailwind Rocks</div>
</template>

<script>
import { pipe, objOf, T } from 'ramda'

let changeFontColor = pipe(
  T,
  objOf('text-red-600')
)

export default {
  name: 'App',
  methods: {
    changeFontColor
  }
}
</script>

第 8 行

let changeFontColor = pipe(
  T,
  objOf('text-red-600')
)

既然 changeFontColor 為 function，就可使用 Ramda 組合使其 point-free。

Array

也可將 class 綁定到 Array。

Binding to Array

<template>
  <div :class="[fontSize, fontColor]">Tailwind Rocks</div>
</template>

<script>
export default {
  name: 'App',
  data: () => ({
    fontSize: 'text-3xl',
    fontColor: 'text-red-600'
  })
}
</script>

第 2 行

<div :class="[fontSize, fontColor]">Tailwind Rocks</div>

若要同時具備多個 utility，可將眾多 model 以 Array 組合綁定。

第 8 行

data: () => ({
  fontSize: 'text-3xl',
  fontColor: 'text-red-600'
})

Model 提供 utility。

Binding to Object Array

也可以使用 Object Array。

<template>
  <div :class="[{fontSize: isFontSizeActive}, fontColor]">Tailwind Rocks</div>
</template>

<script>
export default {
  name: 'App',
  data: () => ({
    fontSize: 'text-3xl',
    fontColor: 'text-red-600',
    isFontSizeActive: false,
  })
}
</script>

第 3 行

<span :class="[{fontSize: isFontSizeActive}, fontColor]">Tailwind Rocks</span>

若並不是所有 utility 都要使用，則可使用 Object Array，使用 property 的 value 決定是否 enable。

第 8 行

data: () => ({
  fontSize: 'text-3xl',
  fontColor: 'text-red-600',
  isFontSizeActive: false,
})

另外提供 isFontSizeActive 決定是否啟用 text-red-600 utility。

Binding to Function

<template>
  <div :class="changeFontStyle()">Tailwind Rocks</div>
</template>

<script>
let changeFontStyle = () => [
  'text-3xl', 'text-red-600'
]

export default {
  name: 'App',
  methods: {
    changeFontStyle
  }
}
</script>

第 2 行

<div :class="changeFontStyle()">Tailwind Rocks</div>

最靈活依然是綁定 function。

Event binding 時 method 之後可省略 ()，但 css binding 時 method 之後一定要 ()

第 6 行

let changeFontStyle = () => [
  'text-3xl', 'text-red-600'
]

只要該 function 回傳 Array 即可。

Conclusion

  • CSS binding 可使用 Object 或 Array，亦可使用 method，只要能回傳 Object 或 Array 即可
  • 若要套用多 utility，除了使用 Array 外，也能使用 Object Array
