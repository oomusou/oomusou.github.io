Monad-tw Index.vue 之 JS 解析

2020-10-04

Index.vue 為 Monad-tw 的首頁，負責顯示 Featured Post 與 Latest Post。

Version

Monad-tw 0.0.1

Index Page

NavigationLink layout 本質為 slot component，包含 header 與 footer。

ContentHeader component 顯示 title 與 description，此為 content 的 header，而不是整體的 header。

FeaturedCard component 負責顯示 feature post 並可左右 slideshow。

CardItem component 負責顯示最新 post list。

下方的 footer 屬於 LavigationLink layout。

pages/Index.vue

<template>
  <navigation-link>
    <content-header :title="$static.metadata.siteName" :sub="$static.metadata.siteDescription" image="phoenix-han-Nqdh0G8rdCc-unsplash.jpg"></content-header>
    <div class="container mx-auto">
      <div class="flex flex-wrap my-4">
        <FeaturedCard v-if="isShowFeaturedCard" :records="$page.featured.edges"/>
        <CardItem v-for="edge in $page.entries.edges" :key="edge.node.id" :record="edge.node"/>
      </div>
    </div>
  </navigation-link>
</template>

<script>
import NavigationLink from '@/layouts/NavigationLink'
import ContentHeader from '@/components/Partials/ContentHeader'
import FeaturedCard from '@/components/Content/FeaturedCard'
import CardItem from '@/components/Content/CardItem'

let isShowFeaturedCard = function() {
  let { totalCount } = this.$page.featured
  return totalCount > 0
}

export default {
  metaInfo: {
    title: 'Hello World!'
  },
  components: {
    NavigationLink,
    CardItem,
    FeaturedCard,
    ContentHeader
  },
  computed: {
    isShowFeaturedCard
  }
}
</script>

Page Meta

title 被定義成 Hello World。

25 行

metaInfo: {
  title: 'Hello World!'
},

Gridsome 已經整合 Vue-meta，在 metaInfo 內定義 title 即可。

ContentHeader Component

顯示了整個 site 的 title 與 subtitle。

<content-header :title="$static.metadata.siteName" :sub="$static.metadata.siteDescription" image="phoenix-han-Nqdh0G8rdCc-unsplash.jpg"></content-header>

使用 ContentHeader component 顯示 title 與 description：

  • title prop 為 title
  • sub prop 為 subtitle
  • image prop 為底圖

gridsome.config.js

由於 title 與 subtitle 為整個 site 的資訊，因此在 gridsome.config.js 定義 siteName 與 siteDescription 。

101 行

query {
  metadata {
    siteName
    siteDescription
  }
}

在 gridsome.config.js 內的資訊，可在 metadata static query 透過 GraphQL 讀取。

可在 GraphQL Playground 使用 metadata static query，siteName 與 siteDescription 都在 metadata 內。

<content-header :title="$static.metadata.siteName" :sub="$static.metadata.siteDescription" image="phoenix-han-Nqdh0G8rdCc-unsplash.jpg"></content-header>

$static 可讀取 static query 的回傳結果，在 metadata 內可讀取 siteName 與 siteDescription。

FeaturedCard Component

以 slideshow 顯示 featured post。

第 6 行

<FeaturedCard v-if="isShowFeaturedCard" :records="$page.featured.edges"/>

使用 FeaturedCard component 顯示 featured post：

  • 使用 v-if() 判斷是否該要顯示 FeaturedCard component
  • records prop 傳入所有 featured post

42 行

  • filter：只顯示 featured 為 true 的 post
  • sortBy：根據 created 排序

以 totalCount 回傳 featured post 筆數。

將要顯示 featured post 資料寫在 edges/node。

可在 GraphQL Playground 使用 featured page query，totalCount 與所有 featured post 資料都在 edges/node 內。

19 行

let isShowFeaturedCard = function() {
  let { totalCount } = this.$page.featured
  return totalCount > 0
}

isShowFeaturedCard computed 會判斷 featured query 回傳的 totalCount 是否大於 0，若有 featured post 則顯示 FeaturedCard component。

第 6 行

<FeaturedCard v-if="isShowFeaturedCard" :records="$page.featured.edges"/>
  • 將 isShowFeaturedCard computed 指定給 v-if 控制顯示 FeaturedCard component
  • 將 $page.featured.edges 傳給 records prop 供顯示

CardItem Component

顯示 latest post。

第 7 行

<CardItem v-for="edge in $page.entries.edges" :key="edge.node.id" :record="edge.node"/>

使用 CardItem component 顯示 latest post：

  • 使用 v-for 重複顯示 latest post
  • records prop 傳入所有 latest post

67 行

query($page: Int) {
  entries: allBlog(perPage: 24, page: $page, sortBy:"created") @paginate {
    totalCount
    pageInfo {
      totalPages
      currentPage
    }
    edges {
      node {
        id
        title

          path
        }
      }
    }
  }
}

以 totalCount 回傳 latest post 筆數。

以 pageInfo 回傳 totalPages 與 currentPage。

將要顯示 latest post 資料寫在 edges/node。

可在 GraphQL Playground 使用 entries page query，totalCount 與所有 latest post 資料都在 edges/node 內。

其中 edges 為 Object Array，每一個 Object 即為一筆 post，而每個 Object 都有 node property，其下為 post 資訊。

第 7 行

<CardItem v-for="edge in $page.entries.edges" :key="edge.node.id" :record="edge.node"/>

將每筆 post 資訊 edge.node 傳給 record prop 供顯示。
