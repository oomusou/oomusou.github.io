Monad-tw 之 NavbarMobile Component 解析

2020-10-20

NavbarMobile Component 為 NavBar Component 內負責在 Mobile 顯示的實質 Navbar。

Version

Monad-tw 0.0.1

NavbarMobile Component

NavbarMobile component 負責顯示 mobile 的 navbar、search 與 dark mode toggle。

components/Navbar/NavbarMobile.vue

<template>
  <nav class="block md:hidden py-4 dark:text-gray-400">
    <div class="flex items-center mx-4">
      <!-- hamburger -->
      <div class="flex items-center mr-6">
        <a role="button" @click.prevent="onOpenNavbarModal" aria-label="Open Navigation" title="Open Navigation">
          <font-awesome :icon="['fas', 'bars']"/>
        </a>
      </div>

      <!-- title -->
      <div class="flex-grow text-center text-xl font-semibold tracking-tight">
        {{ title }}
      </div>

      <div>
        <ul class="flex">
          <!-- search -->
          <li class="mr-6">
            <search-button v-on="$listeners"/>
          </li>

          <!-- toggle -->
          <li>
            <theme-switcher v-on="$listeners" :theme="theme"/>
          </li>
        </ul>
      </div>
    </div>
  </nav>
</template>

<script>
import SearchButton from '@/components/Navbar/SearchButton'
import ThemeSwitcher from '@/components/Navbar/ThemeSwitcher'

let onOpenNavbarModal = function() {
  this.$emit('openNavbarModal')
}

let created = function() {
  this.title = this.$static.metadata.siteName
}

export default {
  components: {
    ThemeSwitcher,
    SearchButton
  },
  props: {
    theme: { default: '' },
  },
  data: () => ({
    title: ''
  }),
  methods: {
    onOpenNavbarModal
  },
  created
}
</script>

<!--@formatter:off-->
<static-query>
query {
  metadata {
    siteName
  }
}
</static-query>
<!--@formatter:on-->

第 2 行

<nav class="block md:hidden py-4 dark:text-gray-400">
  • block md:hidden：當 sm 時顯示， md 以上隱藏，這是 Tailwind 在 RWD 慣用技巧
  • dark:text-gray-400：dark mode 時文字為 灰色

第 3 行

<div class="flex items-center mx-4">
  <!-- hamburger -->
  <div/>

  <!-- title -->
  <div/>

  <!-- search & toggle -->
  <div/>
</div>
  • flex：使 hamburger、title 、search 與 toggle 水平顯示
  • items-center：使 hamber、title、search 與 toggle 垂直置中

第 4 行

<!-- hamburger -->
<div class="flex items-center mr-6">
  <a role="button" @click.prevent="onOpenNavbarModal" aria-label="Open Navigation" title="Open Navigation">
    <font-awesome :icon="['fas', 'bars']"/>
  </a>
</div>

顯示 hamburger：

  • flex items-center：確保 <a> 能垂直置中顯示

11 行

<!-- title -->
<div class="flex-grow text-center text-xl font-semibold         tracking-tight">
  {{ title }}
</div>

顯示 title：

  • flex-grow：title 在中間會佔據所有剩餘 width，故使用 flex-grow
  • text-center：title 水平置中顯示

18 行

<div>
  <ul class="flex">
    <!-- search -->
    <li class="mr-6">
      <search-button v-on="$listeners"/>
    </li>

    <!-- toggle -->
    <li>
      <theme-switcher v-on="$listeners" :theme="theme"/>
    </li>
  </ul>
</div>

顯示 search 與 toggle，因為 search 與 toggle 同時水平靠右，故再多一個 <div> 包一層。

  • flex：使 <li> 能水平顯示

41 行

let created = function() {
  this.title = this.$static.metadata.siteName
}

由 static query 讀取 siteName。

37 行

let onOpenNavbarModal = function() {
  this.$emit('openNavbarModal')
}

當 hamburger 按下時，會觸發 openNavModal event，將由 Navbar component 提供 event handler 顯示 NavbarModal component。

Conclusion

  • 相對於 NavbarDesktop，NavbarMobile 的 codebase 較小，主要都是 Flexbox 排版部分

Reference

oomusou, monad-tw
