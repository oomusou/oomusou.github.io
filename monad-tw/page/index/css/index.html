Monad-tw 之 Index.vue CSS 解析

2020-10-04

Index.vue 為 Monad-tw 的首頁，負責顯示 Featured Post 與 Latest Post，並且以 RWD 根據不同 Device 寬度改變 Layout。

Version

Monad-tw 0.0.1

Index Page

NavigationLink layout 本質為 slot component，包含 header 與 footer。

ContentHeader component 顯示 title 與 description，此為 content 的 header，而不是整體的 header。

FeaturedCard component 負責顯示 feature post 並可左右 slideshow。

CardItem component 負責顯示最新 post list。

下方的 footer 屬於 NavigationLink layout。

pages/Index.vue

<template>
  <navigation-link>
    <content-header :title="$static.metadata.siteName" :sub="$static.metadata.siteDescription" image="phoenix-han-Nqdh0G8rdCc-unsplash.jpg"></content-header>
    <div class="container mx-auto">
      <div class="flex flex-wrap my-4">
        <FeaturedCard v-if="isShowFeaturedCard" :records="$page.featured.edges"/>
        <CardItem v-for="edge in $page.entries.edges" :key="edge.node.id" :record="edge.node"/>
      </div>
    </div>
  </navigation-link>
</template>

<script>
import NavigationLink from '@/layouts/NavigationLink'
import ContentHeader from '@/components/Partials/ContentHeader'
import FeaturedCard from '@/components/Content/FeaturedCard'
import CardItem from '@/components/Content/CardItem'

let isShowFeaturedCard = function() {
  let { totalCount } = this.$page.featured
  return totalCount > 0
}

export default {
  metaInfo: {
    title: 'Hello World!'
  },
  components: {
    NavigationLink,
    CardItem,
    FeaturedCard,
    ContentHeader
  },
  computed: {
    isShowFeaturedCard
  }
}
</script>

<page-query>
query($page: Int) {

      node {

      totalPages

        id
        title
        image(width: 800)

        humanTime: created(format: "DD MMM YYYY")
        datetime: created
        category {
          id
          title
          path
        }
        author {
          id
          name
          image(width: 64, height: 64, fit: inside)
          path
        }
      }
    }
  }
}
</page-query>

<static-query>
query {
  metadata {
    siteName
    siteDescription
  }
}
</static-query>

NavigationLink Layout

NavigationLink layout 負責顯示 header 與 footer。

<navigation-link>
  <content-header :title="$static.metadata.siteName" :sub="$static.metadata.siteDescription" image="phoenix-han-Nqdh0G8rdCc-unsplash.jpg"></content-header>
  <div class="container mx-auto">
    <div class="flex flex-wrap my-4">
      <FeaturedCard v-if="isShowFeaturedCard" :records="$page.featured.edges"/>
      <CardItem v-for="edge in $page.entries.edges" :key="edge.node.id" :record="edge.node"/>
    </div>
  </div>
</navigation-link>

NavigationLink layout 負責所有 page 的 header 與 footer，其他 page 則以 slot 傳入 NavigationLink layout。

ContentHeader Component

ContentHeader component 負責顯示 title、subtitle 與圖片。

<content-header :title="$static.metadata.siteName" :sub="$static.metadata.siteDescription" image="phoenix-han-Nqdh0G8rdCc-unsplash.jpg"></content-header>

ContentHeader component 為 NavigationLink layout 的 slot 中第一個 component，沒有使用任何 CSS class，因此只以 block element 形式佔據一整行。

FeaturedCard & CardItem Component

  • FeaturedCard component 以 slideshow 顯示 featured post
  • CardItem component 顯示 latest post
<div class="container mx-auto">
  <div class="flex flex-wrap my-4">
    <FeaturedCard v-if="isShowFeaturedCard" :records="$page.featured.edges"/>
    <CardItem v-for="edge in $page.entries.edges" :key="edge.node.id" :record="edge.node"/>
  </div>
</div>

第 4 行

<div class="container mx-auto">

FeaturedCard component 與 CardItem component 都在 <div> 內：

  • container：寬度會根據各 breakpoint 的定義而改變
  • mx-auto：水平置中

第 5 行

<div class="flex flex-wrap my-4">

由於 FeaturedCard component 與 CardItem component 會以 RWD 根據寬度改變其 layout，因此使用了 Flexbox：

  • flex：以 Flexbox 對 <div> 內的 component 排版
  • flex-wrap：自動根據 width 而換行
  • my-4：設定 margin-top

Conclusion

  • Monad-tw 為完成度很高的 Starter，幾乎 Blog 所需功能都已具備，而且還使用 Gridsome 與 Tailwind CSS，深入研究可學到很多 Vue、GraphQL、Gridsome、Tailwind CSS … 實用技巧
  • Index.vue 由於使用了 flex 與 flex-wrap，因此 FeaturedCard 與 CardItem component 能以 RWD 根據不同 device 寬度而自動排版
