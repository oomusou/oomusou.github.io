Gridsome 之 Static Query

2020-11-01

使用 GraphQL 讀取 Site Meta 是 Gridsome 一大特色，在 HTML 內直接使用 $static 存取 Static Query，ECMAScript 則要在 Function 內 Destructure this 。

Version

Gridsome 0.7.21

Metadata Query

在使用 Gridsome 內建的 GraphQL Playground 時，會發現 metadata query 除了回傳 siteName 外，還可回傳 siteDescription、siteUrl 與 pathPrefix。

除了 siteName 外，其餘 3 個一開始都是 empty string。

gridsome.config.js

module.exports = {
  siteName: 'Gridsome',
  siteDescription: 'This is My Blog'
}

metadata query 的 data 可在根目錄的 gridsome.config.js 設定，自行加上 siteDescription，並 yarn serve 重新啟動 dev server。

GraphQL Playground

重新執行 metadata query，可發現 siteDescription 已經不是 empty string。

HTML Template

在 HTML template 顯示 siteDescription。

src/pages/About.vue

<template>
  <Layout>
    <h1>About us</h1>
    <p>{{ $static.metadata.siteDescription }}</p>
  </Layout>
</template>

@formatter:off
<static-query>
query {
  metadata {
    siteDescription
  }
}
</static-query>
@formatter:on

第 8 行

@formatter:off
<static-query>
query {
  metadata {
    siteDescription
  }
}
</static-query>
@formatter:on

加上 <static-query/> 定義 metadata query，僅回傳 siteDescription 即可。

第 4 行

<p>{{ $static.metadata.siteDescription }}</p>

在 HTML template 內使用 $static 存取 static query，後面接 GraphQL query 與 field 名稱。

ECMAScript

<template>
  <Layout>
    <h1>About us</h1>
    <p>{{ title }}</p>
  </Layout>
</template>

<script>
let mounted = function() {
  let { $static: { metadata: { siteDescription }}} = this
  this.title = siteDescription
}

export default {
  data: () => ({
    title: ''
  }),
  mounted
}
</script>

@formatter:off
<static-query>
query {
  metadata {
    siteDescription
  }
}
</static-query>
@formatter:on

第 9 行

let mounted = function() {
  let { $static: { metadata: { siteDescription }}} = this
  this.title = siteDescription
}

若要在 ECMAScript 內存取 static query 則不太一樣，需使用從 this 去 destructure 出 siteDescription。

Conclusion

  • 由於 site meta 定義於 gridsome.config.js 檔案中，因此使用 static query 讀取
  • HTML template 可直接使用 $static 存取 static query，但 ECMAScript 內必須 destructure this

Reference

Reed Barger, GraphQL Overview, Using Static Queries
Gridsom, Querying data
