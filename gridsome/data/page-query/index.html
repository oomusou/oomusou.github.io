Gridsome 之 Page Query

2020-09-13

Blog 每篇 Post 都是獨立 Markdown，透過 Template 與 Page Query 可將 Post 以 HTML 顯示。

Version

macOS Catalina 10.15.2
Gridsome 0.7.12

Page Query

首頁多了 Read Post，可以單獨顯示整篇 post。

一整頁單獨顯示整篇 post。

Gridsome Config

gridsome.config.js

module.exports = {
  siteName: 'Gridsome',
  plugins: [
    {
      use: '@gridsome/source-filesystem',
      options: {
        typeName: 'Post',
        baseDir: './markdown',
        pathPrefix: '/posts',
        path: '*.md',
      },
      templates: {
        Post: '/posts/:slug'
      }
    }
  ]
}

第 3 行

plugins: [
  {
    use: '@gridsome/source-filesystem',
    options: {
      typeName: 'Post',
      baseDir: './markdown',
      pathPrefix: '/posts',
      path: '*.md',
    },
    templates: {
      Post: '/posts/:slug'
    }
  }
]

設定 @gridsome/source-filesystem plugin：

  • options
    • typeName：設定 GraphQL 的 type 為 Post
    • baseDir：設定 Markdown 檔案所在目錄
    • pathPrefix：設定 route
    • path：baseDir 目錄下所有 .md 皆為 Markdown
  • templates
    • Post：設定 Post type 的 URI，其中 :slug 為 Markdown 檔名

GraphQL Playground

AllPost Query

query {
  allPost {
    edges {
      node {
        title
        path
      }
    }
  }
}

allPost query 包含 path field，可查詢每篇 post 的 URI。

Post Query

query($path: String!) {
  post(path: $path) {
    content
  }
}

{
  "path": "/posts/first-post"
}

Gridsome 另外提供 post query，以 path 為 argument，可回傳 post 完整內容。

Page

pages/Blog.vue

在 allPost query 加上 path field，另外加上 <g-link/>，將 edge.node.path 綁定到 to property。

Template

templates/Post.vue

<template>
  <Layout>
    <div v-html="$page.post.content"></div>
  </Layout>
</template>

<page-query>
query($path: String!) {
  post(path: $path) {
    content
  }
}
</page-query>

<script>
export default {
  name:"Post"
}
</script>

第 7 行

<page-query>
query($path: String!) {
  post(path: $path) {
    content
  }
}
</page-query>

由於 post query 帶 variable，因此必須改用 <page-query/>。

第 3 行

<div v-html="$page.post.content"></div>

由於從 <static-query/> 改用 <page-query/>，因此 HTML template 也要從 $static 改成 $page。

Conclusion

  • Post 可由 template 從 Markdown 以 HTML 顯示
  • 若 GraphQL 使用 variable，Vue 就要改用 page query
