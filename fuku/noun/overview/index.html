複合名詞簡介

2025-10-27

日文常將 兩個名詞 組合成一個新的 名詞，中間不必加 の。這裡的 名詞 是 廣義 的 名詞：動詞連用形、サ變名詞 也算 名詞。這種構造在日語中非常常見，是詞彙擴充的重要方式之一。

名詞 + 名詞

  • 雨雲(あまぐも) [0]：雨 + 雲
  • 鉄橋(てっきょう) [0]：鉄 + 橋
  • 日本語(にほんご) [0]：日本 + 語

動詞連用形 + 名詞

  • 飛(と)び出(だ)し [0]：飛び + 出し
  • 休(やす)み時間(じかん) [4]：休み + 時間
  • 話(はな)し方(かた) [0]：話し + 方

サ變名詞 + 名詞

  • 勉強会(べんきょうかい) [3]：勉強 + 会
  • 運動靴(うんどうぐつ) [5]： 運動 + 靴
  • 旅行計画(りょこうけいかく) [5]：旅行 + 計画

外來語 + 名詞

  • コンピューター教室(きょうしつ) [7]：コンピューター + 教室
  • テレビ番組(ばんぐみ) [4]：テレビ + 番組
  • スマホカバー [4]： スマホ + カバー
