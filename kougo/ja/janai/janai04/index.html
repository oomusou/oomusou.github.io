じゃないか = ではないか (確認)

2025-03-18

普通形 じゃないか 可表示 確認，用於說話者認為對方 應該 也知道的事實，並向對方 確認。其中 か 可 省略。語調 為 上揚。翻成成 ...不是嗎？

接続

  • 普 + じゃない (か) = 普 + ではないか
  • V + じゃない (か) = V + ではないか
  • いA + じゃない (か) = いA + ではないか
  • なA + じゃない (か) = なA + ではないか
  • N + じゃない (か) = N + ではないか

動詞

明日、試験があるではなか？
明日、試験があるじゃない？ (縮約)

明天有考試不是嗎？

  • 明日 (あした) [0]：名 明天
  • 試験 (しけん) [2]：名 考試
  • が：格助 表 主語
  • ある (ある) [1]：動I 自動 辞書形 有
    • あるではないか [1]：確認 有考試嗎不是嗎？
    • あるじゃない [1]：縮約 有考試嗎不是嗎？

い形容詞

この映画、面白いではないか？
この映画、面白いじゃない？ (縮約)

這部電影很有趣不是嗎？

  • この [0]：連体 這個
  • 映画 (えいが) [0]：名 電影
  • 面白(おもしろ)い [4]：い形容 有趣的
    • 面白いではないか [4]：確認 有趣的不是嗎？
    • 面白いじゃない [4]：縮約 有趣的不是嗎？

な形容詞

これ、すごく便利ではないか？
これ、すごく便利じゃない？ (縮約) 　

這個很方便不是嗎？

  • これ [0]：代 這個
  • すごく [2]：副 很
  • 便利 (べんり) [1]：な形容 便利的
    • 便利ではないか [1]：確認 便利的不是嗎？
    • 便利じゃない [1]：縮約 便利的不是嗎？

名詞

明日、祝日ではないか？
明日、祝日じゃない？ (縮約)　

明天是國定假日不是嗎？

  • 明日(あした) [0]：名 明天
  • 祝日 (しゅくじつ) [0]：名 國定假日
    • 祝日ではないか [0]：確認 國定假日不是嗎？
    • 祝日じゃない [0]：縮約 國定假日不是嗎？
