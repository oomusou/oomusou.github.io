そりゃ = それは (那是)

2025-01-15

それは 在 口語 常縮約成 そりゃ 。

接続

  • そりゃ = それは

例文

彼が勝つなんて、それはそうだ。
彼が勝つなんて、そりゃそうだ。 (縮約)

他會勝，那倒也是。

  • 彼(かれ) [1]：代 他
  • が：格助 表 主語
  • 勝(か)つ [1]：動I 自動 勝利
  • なんて：副助 竟然
  • それは：代 那是
    • そりゃ：縮約 那是
  • そうだ：慣 說的也是

温泉に行くなんて、それはいいね。
温泉に行くなんて、そりゃいいね。 (縮約)

去泡溫泉，那真不錯啊。

  • 温泉(おんせん) [0]：名 溫泉
  • に：格助 表 目的地
  • 行(い)く [0]：動I 自動 去
  • なんて：副助 竟然
  • それは：代 那是
    • そりゃ：縮約 那是
  • いい [1]：い形容 好的
  • ね：終助 表 稱讚

それはそうだが、簡単じゃないよ。
そりゃそうだが、簡単じゃないよ。 (縮約)

雖然那是沒錯，但可不容易啊。

  • それは：代 那是
    • そりゃ：縮約 那是
  • そう [1]：副 是這樣
  • だ：助動 是
  • が：接助 表 雖然
  • 簡単 (かんたん) [0]：な形容 簡單的
    • 簡単じゃない：ない形 不簡單的
  • よ：終助 表 告知
