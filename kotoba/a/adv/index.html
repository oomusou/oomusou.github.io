副詞化

2026-07-15

有 多種方式 可將 い形容詞、な形容詞、名詞 或 動詞 加以 副詞化。

接続

  • いA い + く
  • なA + に
  • N + に
  • N + 風に
  • N + の + ように
  • 漢字詞 + 的に
  • V普 + ように
  • Vて (部分動詞)

い形容詞

  • 早い -> 早く
  • 遅い -> 遅く
  • 強い -> 強く

な形容詞

  • 静か -> 静かに
  • 丁寧 -> 丁寧に
  • 簡単 -> 簡単に

名詞

  • 最後 -> 最後に

  • 一緒 -> 一緒に

  • 本当 -> 本当に

  • 子供 -> 子供風に

  • 冗談 -> 冗談風に

  • 王様 -> 王様風に

  • 鳥 -> 鳥のように

  • 子供 -> 子供のように

  • プロ -> プロのように

  • 基本 -> 基本的に

  • 一般 -> 一般的に

  • 具体 -> 具体的に

動詞

  • 分かる -> 分かるように

  • 忘れない -> 忘れないように

  • 聞こえる -> 聞こえるように

  • 座る -> 座って

  • 笑う -> 笑って

  • 急ぐ -> 急いで
