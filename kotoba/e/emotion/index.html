他人情緒

2026-06-22

嬉しい、楽しい 等 情緒形容詞、不能 直接用於 他人，要改用 動詞、そう 或 がる 表示。

接続

  • いA い + そうだ
  • なA + そうだ
  • いA い + がる
  • なA + がる

動詞

彼は試験に合格してとても喜んだ。

他考試及格了，非常高興。

嬉しい -> 喜ぶ

  • 彼(かれ) [1]：代 他
  • は：副助 表 主題
  • 試験(しけん) [2]：名 考試
  • に：格助 表 動作對象
  • 合格(ごうかく) [0]：名 合格
    • 合格する [0]：動III 自動 辞書形 合格
    • 合格して [0]：て形 合格
  • とても [0]：副 非常地
  • 喜(よろこ)ぶ [3]：動I 自動 辞書形 高興
    • 喜んだ [3]：た形 高興

彼は休日に音楽を楽しむ。

他在假日享受聽音樂的樂趣。

楽しい -> 楽しむ

  • 彼(かれ) [1]：代 他
  • は：副助 表 主題
  • 休日(きゅうじつ) [0]：名 假日
  • に：格助 表 發生時間
  • 音楽(おんがく) [1]：名 音樂
  • を：格助 接受 楽しむ 的 賓語
  • 楽(たの)しむ [3]：動I 他動 辞書形 享受

そう

彼はとても嬉しそうだ。

他看起來非常高興。

嬉しい -> 嬉しそう

  • 彼(かれ) [1]：代 他
  • は：副助 表 主題
  • とても [0]：副 非常地
  • 嬉(うれ)しい [3]：い形容 開心的
    • 嬉しそうだ [4]：樣態 好像很開心

彼女は新しい仕事が楽しそうだ。

她的新工作看起來很有趣。

楽しい -> 楽しそう

  • 彼女(かのじょ) [1]：代 她
  • は：副助 表 主題
  • 新(あたら)しい [4]：い形容 新的
  • 仕事(しごと) [0]：名 工作
  • が：格助 表 主語
  • 楽(たの)しい [3]：い形容 快樂的
    • 楽しそうだ [4]：樣態 好像很快樂

がる

彼は新しい携帯を自慢して、嬉しがっている。

他一直在炫耀新手機，表現得很開心。

嬉しい -> 嬉しがる -> 嬉しがっている

  • 彼(かれ) [1]：代 他
  • は：副助 表 主題
  • 新(あたら)しい [4]：い形容 新的
  • 携帯(けいたい) [0]：名 手機
  • を：格助 接受 自慢する 的 賓語
  • 自慢(じまん) [0]：名 炫耀
    • 自慢する [0]：動III 他動 辞書形 炫耀
    • 自慢して [0]：て形 炫耀
  • 嬉(うれ)しい [3]：い形容 開心的
    • 嬉しがる [5]：動I 自動 辞書形 表現出開心的樣子
    • 嬉しがって [5]：て形 表現出開心的樣子
    • 嬉しがっている [5]：狀態 表現得很開心

彼はみんなの前では楽しがっている。

他在人前表現得很開心。

楽しい -> 楽しがる -> 楽しがっている

  • 彼(かれ) [1]：代 他
  • は：副助 表 主題
  • みんな [3]：名 大家
  • の：格助 以 名詞 形容 名詞
  • 前(まえ) [1]：名 前面
  • で：格助 表 動作場所
  • は：副助 表 對比
  • 楽(たの)しい [3]：い形容 快樂的
    • 楽しがる [5]：他人情感 表現出快樂的樣子
    • 楽しがって [5]：て形 表現出快樂的樣子
    • 楽しがっている [5]：狀態 表現得很快樂
