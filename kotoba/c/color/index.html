顏色

2025-10-04

日語的 顏色 由於 歷史 因素，不同的顏色的 形容詞 有 不同 的講法。

基本顏色

基本顏色 為 狀態，可直接以 い形容詞 形容 名詞

  • 赤(あか) [1]：名 紅色 (明)
    • 赤い [0]：い形容 紅色的
  • 黒(くろ) [1]：名 黑色 (暗)
    • 黒い [2]：い形容 黑色的
  • 白(しろ) [1]：名 白色 (顯)
    • 白い [2]：い形容 白色的
  • 青(あお) [1]：名 藍色 (淡)
    • 青い [2]：い形容 藍色的

黄、茶

黄色 與 茶色 必須加上 色 當 名詞， 再加上 い 成為 い形容詞 形容 名詞

  • 黄色(きいろ) [0]：名 黃色
    • 黄色い [0]：い形容 黃色的
  • 茶色(ちゃいろ) [0]：名 茶色
    • 茶色(ちゃいろ)い [0]：い形容 茶色的
    • 茶色の [0]：修飾 茶色的

茶色 是 例外，茶色い 與 茶色の 皆可，但目前大都使用 茶色の

其他顏色

緑 與 紫 可直接當 名詞，其他顏色要加上 色 當 名詞，要使用 の 來 修飾 名詞

  • 緑(みどり) [1]：名 綠色
    • 緑の [1]：修飾 綠色的
  • 紫(むらさき) [2]：名 紫色
    • 紫の [2]：修飾 紫色的
  • 灰色(はいいろ) [0]：名 灰色
    • 灰色の [0]：修飾 灰色的
  • 桃色(ももいろ) [0]：名 粉紅色
    • 桃色の [0]：修飾 粉紅色的
  • 橙色(だいだいいろ) [0]：名 橘色
    • 橙色の [0]：修飾 橘色的
  • 金色(きんいろ) [0]：名 金色
    • 金色の [0]：修飾 金色的
  • 銀色(ぎんいろ) [0]：名 銀色
    • 銀色の [0]：修飾 銀色的
  • 水色(みずいろ) [0]：名 水藍色
    • 水色の [0]：修飾 水藍色的

外來語顏色

外來語顏色 可直接當 名詞，要使用 の 來 修飾 名詞

  • ピンク [1]：名 pink、粉紅色
    • ピンクの [1]：修飾 粉紅色的
  • オレンジ [2]：名 orange、橙色
    • オレンジの [2]：修飾 橙色的
  • グレー [2]：名 gray、灰色
    • グレーの [2]：修飾 灰色的
  • ブルー [2]：名 blue、藍色
    • ブルーの [2]：修飾 藍色的
  • レッド [1]：名 red、紅色
    • レッドの [1]：修飾 紅色的
  • イエロー [2]：名 yellow、黃色
    • イエローの [2]：修飾 黃色的
  • グリーン [2]：名 green、綠色
    • グリーンの [2]：修飾 綠色的
  • パープル [1]：名 purple、紫色
    • パープルの [1]：修飾 紫色的
