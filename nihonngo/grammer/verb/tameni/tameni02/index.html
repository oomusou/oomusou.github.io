N の ために (目的)

2023-09-20

名詞 の ために 可表示 目的，翻譯成 為了，且 に 可以省略。

接続

  • N + の + ために、V

例文

引っ越しのために、車を借りる。

為了搬家要借車子。

  • 引(ひ)っ越(こ)し [0]：名 搬家
  • の：格助 以 名詞 形容 名詞
  • ために [2]：接助 表 目的
  • 車(くるま) [0]：名 車
  • を：格助 接受 借りる 的 賓語
  • 借(か)りる [0]：動II 他動 借入
