Vます そう だ (好像)

2023-02-25

動詞ます形 ます そう だ 可根據 眼前景象 而做出 主觀判斷 的 好像，有 動作即將發生、迫切感強、關心程度高 的語感，常搭配 今にも、もうすぐ、これから … 等副詞。そう 後變成 な形容詞，可再接 名詞 或轉為 副詞 使用。在普通体可省略 だ。

接続

  • Vます ます + そう + だ
  • いA い + そう + だ
  • なA + そう + だ
  • Vない ない + なそう + だ (否定)
  • いA い + くなさそう + だ (否定)
  • なA + なさそう + だ (否定)
  • いA + なそう + だ (つまらない、危ない、少ない) (ない結尾)

例文

今にも雨が降りそうだ。

好像快要下雨了。
