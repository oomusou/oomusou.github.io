A は B に Vれる (動作者)

2023-07-11

格助詞 に 可表示 動詞可能形、動詞使役形、動詞受身形 與 動詞使役受身形 或 含被動意味動詞 的 動作者。

接続

  • N + は + A + に + Vできる (可能形)
  • A + は + B + に + Vせる (使役形)
  • B + は + A + に + Vれる (受身形)
  • B + は + A + に + Vされる (使役受身形)
  • B + は + A + に + V (含被動意味動詞)

動詞可能形

そんなマジックは子供にもできる。

這個魔術連小孩都會。

  • そんな [0]：連体 那樣的
  • マジック [1]：名 magic、魔術
  • は：副助 表 主題
  • 子供(こども) [0]：名 孩子
  • に：格助 表 動作者
  • も：副助 表 並立
  • する [0]：動III 他動 辞書形 做
    • できる [0]：可能形 會做

動詞使役形

子供に公園を散歩させる。

我讓孩子在公園散步。

  • 子供(こども) [0]：名 孩子
  • に：格助 表 動作者
  • 公園(こうえん) [0]：名 公園
  • を：格助 表 通過點
  • 散歩(さんぽ)する [0]：動I 自動 辞書形 散步
    • 散歩させる [0]：使役形 讓散步

動詞受身形

私は先生に褒められた。

我被老師誇獎了。

  • 私(わたし) [0]：代 我
  • は：副助 表 主題
  • 先生(せんせい) [3]：名 老師
  • に：格助 表 動作者
  • 褒(ほ)める [2]：動II 他動 辞書形 誇獎
    • 褒められる [4]：受身形 被誇獎

使役受身形

生徒は先生に走らされる。

學生被老師逼迫去跑步。

  • 生徒(せいと) [1]：名 學生
  • は：副助 表 主題
  • 先生(せんせい) [3]：名 老師
  • に：格助 表 動作者
  • 走(はし)る [2]：動I 自動 辞書形 跑
    • 走らされる [5]：使役受身形 被逼迫跑

含被動意味動詞

宝は海賊に見つかった。

寶藏被海盜找到了。

見つかる 雖然不是 受身形，但本身含有 被動意味

  • 宝(たから) [3]：名 寶藏
  • は：副助 表 主題
  • 海賊(かいぞく) [0]：名 海盜　
  • に：格助 表 動作者
  • 見(み)つかる [0]：動I 自動 辞書形 找到
    • 見つかった [0]：た形 找到
