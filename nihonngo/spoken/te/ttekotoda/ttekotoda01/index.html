N って、普 ってことだ (定義)

2023-11-30

慣用語 N とは、普 ということだ 在 口語 常縮約成 N って、普 ってことだ 表示 定義，翻譯成 所謂的...就是...的意思。

接続

  • N + って + 普 + ってことだ = N + とは + 普 + ということだ
  • N + って + Vる + ってことだ = N + とは + Vる + ということだ
  • N + って + いA + ってことだ = N + とは + いA + ということだ
  • N + って + なA + だ + ってことだ = N + とは + なA + だ + ということだ
  • N + って + なA + ってことだ = N + とは + なA + ということだ
  • N + って + N + だ + ってことだ = N + とは + N + だ + ということだ
  • N + って + N + ってことだ = N + って + N + ということだ

例文

友情とは互いを理解することということだ。
友情って互いを理解することってことだ。 (縮約)

所謂的友情，就是彼此理解的意思。

  • 友情(ゆうじょう) [0]：名 友情
  • とは [0]：慣 所謂的…就是
  • って [0]：縮約 表 とは
  • 互(たが)い [0]：名 互相
  • を：格助 接受 理解する 的 賓語
  • 理解(りかい)する [0]：動III 他動 辞書形 理解
    • 理解すること [0]：名 理解
  • ということだ [0]：慣 的意思
  • ってことだ [0]：縮約 表 ということだ
