だろう (推測)

2024-10-05

だろう 當結尾為 句號 ，且語調 降低 時可表示 沒依據的推測，約 70% 可能性。だろう 為 でしょう 的 普通体 。翻譯成 大概。

接続

  • 普 + だろう
  • Vる + だろう
  • いA + だろう
  • なA + だろう
  • N + だろう

動詞

明日は雨が降るだろう。

明天大概會下雨。

  • 明日(あした) [3]：名 明天
  • は：副助 表 主題
  • 雨(あめ) [1]：名 雨
  • が：格助 表 主語
  • 降(ふ)る [1]：動I 自動 辞書形 下雨
    • 降るだろう [1]：大概 大概會下雨
