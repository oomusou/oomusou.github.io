普 場合は、V (時)

2022-12-09

場合 是假設某種狀況的表達方式，後續句子表示的是作為其對策的方法，或其帶來的結果。由於 場合 是 名詞，所以和 名詞 修飾相同。其意義與 時 相同，但較於 正式。

接続

  • 普 + 場合 + は、V
  • いA + 場合 + は、V
  • なA + 場合 + は、V
  • N + の + 場合 + は、V
