はずだ (應該會做)

2024-11-23

普通形 はずだ 可表示說話者的 90%推測 的 推測的應該，在普通体可省略 だ。翻譯成 應該會做。

接続

  • 普 + はずだ
  • Vる + はずだ
  • いA + はずだ
  • なA + な + はずだ
  • N + の + はずだ
