普 時、V (時)

2023-03-22

普 時、V 可表示 在...時做了什麼事。由於 時 是 名詞，所以使用 な形容詞 必須加上 な，名詞 必須加上 の。當 普通形 使用 非過去式 時，表示 時 句子內的動作 尚未完成；使用 過去式 則表示 時 句子內的動作 已經完成。

接続

  • 普 + 時、V
  • Vる + 時、V
  • いA + 時、V
  • なA + な + 時、V
  • N + の + 時、V
