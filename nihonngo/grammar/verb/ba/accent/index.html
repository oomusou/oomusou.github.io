動詞条件形のアクセント

2022-09-12

動詞条件形 的 重音 與 動詞辭書形 相同。

動詞 I

い 段 -> え 段 + ば

  • 会います [3] -> 合えば [1]
  • 聞きます [3] -> 聞けば [0]
  • 急ぎます [4] -> 急げば [2]
  • 出します [3] -> 出せば [1]
  • 待ちます [3] -> 待てば [1]　
  • 呼びます [3] -> 呼べば [0]
  • 飲みます [3] -> 飲めば [1]
  • 降ります [3] -> 降れば [1]

動詞 II

え 段 -> れば

  • 晴れます [3] -> 晴れれば [2]
  • 見ます [2] -> 見れば [1]

動詞 III

  • 来(き)ます [2] -> 来(く)れば [1]
  • します [2] -> すれば [0]
  • 勉強します [6] -> 勉強すれば [0]

い形容詞

い -> ければ

  • 高い [2] -> 高ければ [2]
  • いい [1] -> よければ [2]

な形容詞

な -> なら

  • きれいな [1] -> 綺麗なら [1]
  • 真面目な [0] -> 真面目なら [0]

名詞

名詞なら

  • 雨 [1] -> 雨なら [1]
  • 無料 [0] -> 無料なら [0]
