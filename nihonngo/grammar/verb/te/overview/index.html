動詞て形變化

2022-01-24

本文整理 動詞て形 變化規則。

動詞 I

き -> いて

  • 書きます -> 書いて
  • 行きます -> 行って

ぎ -> いで

  • 急ぎます -> 急いで

いちり -> って

  • 会います -> 会って
  • 立ちます -> 立って
  • 帰ります -> 帰って

みにび -> んで

  • 飲みます -> 飲んで
  • 死にます -> 死んで
  • 遊びます -> 遊んで

し -> して

  • 貸します -> 貸して
  • 返します -> 返して
  • 無くします -> 無くして
  • 話します [4] -> 話して

動詞 II

え段 -> て

  • 開けます -> 開けて
  • 見せます -> 見せて
  • 止めます -> 止めて
  • 食べます -> 食べて
  • 出ます -> 出て
  • できます -> できて
  • 起きます -> 起きて
  • 借ります -> 借りて

動詞 III

  • 来(き)ます -> 来(き)て
  • します -> して
  • 勉強します -> 勉強して
