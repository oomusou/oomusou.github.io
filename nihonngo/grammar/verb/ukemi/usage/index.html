動詞受身形 (直接、間接、受害、創造、無動作者)

2022-10-06

可使用 動詞受身形 表示 直接被動、間接被動 、受害被動、創造被動 或 無動作者被動。

接続

  • A + は + B + に + Vれる
  • A + は + B + に + N + を + Vれる

例文

直接被動

直接受到行為者影響。若 被行為者 很明確可省略

私は先生に褒められた。

我被老師稱讚了。

  • 私(わたし) [0]：代 我
  • は：副助 表 主題
  • 先生(せんせい) [3]：名 先生
  • に：格助 表 對象
  • 褒(ほ)める [2]：動II 他動 辭書形 稱讚
    • 褒められる [4]：受身形 被稱讚
    • 褒められた [4]：た形 被稱讚
