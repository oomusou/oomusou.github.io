あげる (給)

2023-07-25

說話者 給 平輩 或 晚輩 物品時，可使用 あげる；但說話者給 長輩 或 上司 給 物品時，則要使用 さしあげる。

単語

あげる [0]：動II 他動 給。

動詞活用

  • ます形：あげます [3]
  • 辞書形：あげる [0]
  • ない形：あげない [0]
  • て形：あげて [0]
  • た形：あげた [0]
  • 可能形：あげられる [0]
  • 意向形：あげよう [3]
  • 命令形：あげろ [0]

  • 条件形：あげれば [0]
  • 受身形：あげられる [0]
  • 使役形：あげさせる [0]

文体

  • 丁寧形現在肯定：あげます
  • 丁寧形現在否定：あげません
  • 丁寧形過去肯定：あげました
  • 丁寧形過去否定：あげませんでした
  • 普通形現在肯定：あげる
  • 普通形現在否定：あげない
  • 普通形過去肯定：あげた
  • 普通形過去否定：あげなかった

接続

  • B + に + N + を + あげる
  • A + は + B + に + N + を + あげる
