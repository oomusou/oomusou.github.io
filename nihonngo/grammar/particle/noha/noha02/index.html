普 のは N (強調)

2023-03-29

格助詞 のは 可表示 強調，其中 の 為 もの 的 縮約，因此也可搭配 い形容詞 與 な形容詞。

接続

  • 普 + の + は + N
  • Vる + のは + N
  • いA + のは + N
  • なA + な + のは + N

例文

動詞

父が生まれたのは北海道の小さな村。

父親出生的地方是北海道的小村莊。

  • 父(ちち) [2]：名 父親
  • が：格助 表 主語
  • 生(う)まれる [0]：動II 自動 出生
  • のは：副助 表 強調
  • 北海道(ほっかいどう) [3]：名 北海道
  • の：格助 以 名詞 形容 名詞
  • 小(ちい)さな [1]：連体 小的
  • 村(むら) [2]：名 村莊

  • さん [0]：名 先生

  • 来(く)る [1]：動III 自動 辞書形 來
    • 来(き)た [1]：た形 來
  • のは：副助 表 強調
  • 去年(きょねん) [1]：名 去年

  • 去年(きょねん) [1]：名 去年
