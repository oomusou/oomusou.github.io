普 けれど、V (雖然)

2023-08-04

接助詞 けれど 可表示 雖然，可 逆接 兩個句子，其中句子可放 丁寧体，也可放 普通体，但通常放 普通體。亦可使用 けれども 或 けれど，其正式度 けれども > けれど > けど。

接続

  • 丁/普 + けれど、V
  • Vる + けれど、V
  • いA + けれど、V
  • なA + だ + けれど、V
  • N + だ + けれど、V

動詞

毎日試験勉強したけれど、不合格だった。

雖然每天準備考試，但是還是不及格。

  • 毎日(まいにち) [1]：名 每天
  • 試験(しけん) [2]：名 考試
  • 試験勉強(しけんべんきょう)する：動III 自動 辞書形 準備考試
    • 試験勉強した：た形 準備考試
  • けれど [1]：接助 表 雖然
  • 不合格(ふごうかく) [2]：名 不及格
  • だ [0]：助動 是
    • だった：た形 是

い形容詞

この部屋は狭いけれど、きれいだ。

這個房間雖窄，但很乾淨。

  • 部屋(へや) [2]：名 房間
  • は：副助 表 主語
  • 狭(せま)い [2]：い形容 狹窄的
  • けれど [1]：接助 表 雖然
  • きれい [1]：な形容 乾淨的
  • だ：普通形 是

日本語は難しいけれど、面白い。

日語雖難但很有趣。

  • 日本語(にほんご) [0]：名 日語
  • は：副助 表 主語
  • 難(むずか)しい [4]：い形容 難的
  • けれど [1]：接助 表 雖然
  • 面白(おもしろ)い [4]：い形容 有趣的

フランス料理は美味しいけれど、高い。

法國料理雖然好吃但很貴。

  • フランス [0]：名 France、法國
  • 料理(りょうり) [1]：名 料理
  • は：副助 表 主語
  • 美味(おい)しい [3]：い形容 好吃的
  • けれど [1]：接助 表 雖然
  • 高(たか)い [2]：い形容 貴的

日本の生活は忙しいけれど、毎日楽しい。

雖然日本的生活忙碌，但是每天都很快樂。

  • 日本(にほん) [2]：名 日本 (地名)
  • の：格助 以 名詞 形容 名詞
  • 生活(せいかつ) [0]：名 生活
  • は：副助 表 主語
  • 忙(いそが)しい [4]：い形容 忙碌的
  • けれど [1]：接助 表 雖然
  • 毎日(まいにち) [1]：名 每天
  • 楽(たの)しい [3]：い形容 高興的

な形容詞

日本の歌が大好きだけれど、歌があまり上手じゃない。

雖然很喜歡日本歌，但是唱歌我不太拿手。

  • 日本(にほん)の歌(うた) [2]：名 日文歌
  • が：格助 表 主語
  • 大好(だいす)き [1]：い形容 非常喜歡
  • だ：助動 是
  • けれど [1]：接助 表 雖然
  • 歌(うた) [2]：名 歌
  • が：格助 表 主語
  • あまり [0]：副 不太 (後接否定)
  • 上手(じょうず) [3]：な形容 拿手的
    • 上手じゃない：ない形 不拿手的

名詞

今日は日曜日だけれど、会社へ行く。

雖然今天是星期天，但是我去公司。

  • 今日(きょう) [1]：名 今天
  • は：副助 表 主題
  • 日曜日(にちようび) [3]：名 星期天
  • だ：助動 是
  • けれど [1]：接助 表 雖然
  • 会社(かいしあ) [0]：名 公司
  • 行(い)く [0]：動I 自動 辞書形 去

可笑しい日本語だけれど、よろしく。

雖然是很好笑的日語，但是請多多指教。

  • 可笑(おか)しい [3]：い形容 可笑的
  • 日本語(にほんご) [0]：名 日語
  • だ：助動 是
  • けれど [1]：接助 表 雖然
  • よろしく [0]：慣 多多指教
