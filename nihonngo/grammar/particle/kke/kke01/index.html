普 っけ (確認)

2023-10-27

普通形 っけ 可表示 不確定的記憶，進而想要 確認，此時語調為 上揚，翻譯成 是不是~來著？

接続

  • 普 + っけ
  • Vる + っけ
  • いA + っけ
  • なA + だ + っけ
  • N + だ + っけ

動詞

私なんかミスしたっけ？

我是不是做錯了什麼來著？

  • 私(わたし) [0]：代 我
  • 何(なん) [1]：名 什麼
  • か：副助 表 不確定
  • ミスする [0]：動III 自動 辞書形 做錯
    • ミスした [0]：た形 做錯了
    • ミスしたっけ [0]：確認 是不是做錯了來著？

この店、以前来たことがあったっけ？

這間店，我以前是不是有來過了來著？

  • この [0]：連体 這個
  • 店(みせ) [2]：名 店
  • 以前(いぜん) [1]：名 以前
  • 来(く)る [1]：動III 自動 辞書形 來
    • 来(き)た [1]：た形 來
    • 来たことがある [6]：經驗 有來過
    • 来たことがあった [6] ：た形 有來過了
    • 来たことがあったっけ [6]：確認 是不是有來過了來著？

  • 先月(せんげつ) [1]：名 上個月
  • の：格助 以 名詞 形容 名詞
  • 電気料金(でんきりょうきん) [4]：名 電費
  • は：副助 表 主題
  • もう [1]：副 已經
  • 払(はら)う [2]：動I 他動 辞書形 付錢
    • 払った [2]：た形 付了錢
    • 払ったっけ [2]：確認 是不是付了錢來著？
