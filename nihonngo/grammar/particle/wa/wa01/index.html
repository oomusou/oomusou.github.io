わ (告知)

2024-10-09

終助詞 わ 可表示 告知，相當於 よ，男性 使用會 語氣下降，女性 則 上升。唯 わ 偏 年長女性 或 關西腔 使用，有 上對下 的 語感，翻譯成 喔。

接続

  • 普 + わ
  • Vる + わ
  • いA + わ
  • なA + だ + わ
  • N + だ + わ
