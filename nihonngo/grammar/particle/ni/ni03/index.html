N に ある / いる (所有者)

2023-03-10

格助詞 に 可表示 ある、いる 的 所有者 。

接続

  • N + に + [は] + N + が + ある
  • N + に + [は] + N+ が + いる

例文

ある

あの人には不思議な魅力がある。

那個人有不可思議的魅力。

當 に 表 所有者 時常搭配 は 表 強調，但亦可 省略は

  • あの [0]：連体 那個
  • 人(ひと) [0]：名 人
  • に：格助 表 所有者
  • は：副助 表 主題
  • 不思議(ふしぎ) [0]：な形容 不可思議的
  • 魅力(みりょく) [0]：名 魅力
  • が：格助 表 主語
  • ある [1]：動III 自動 擁有

誰にでも秘密がある。

無論是誰都有秘密。

  • 誰(だれ) [1]：代 誰
  • に：格助 表 所有者
  • でも [1]：副助 無論
  • 秘密(ひみつ) [0]：名 秘密
  • が：格助 表 主語
  • ある [1]：動III 自動 擁有

楽器にも色々な種類がある。

樂器也有各式各樣的種類。

  • 楽器(がっき) [0]：名 樂器
  • に：格助 表 所有者
  • も：副助 表 也
  • 色々(いろいろ) [0]：な形容 各式各樣的
  • 種類(しゅるい) [1]：名 種類
  • が：格助 表 主語
  • ある [1]：動III 自動 擁有

いる

田中さんには子供が３人いる。

田中先生有三個小孩。

當 に 表 所有者 時常搭配 は 表 強調，但亦可 省略は

  • 田中(たなか) [0]：名 田中 (姓名)
  • さん [0]：名 先生
  • に：格助 表 所有者
  • は：副助 表 主題
  • 子供(こども) [0]：名 小孩
  • が：格助 表 主語
  • ３人(さんにん) [3]：名 三人
  • いる [0]：動I 自動 擁有
