普 じゃん = 普 じゃない (不滿)

2024-01-06

じゃん 在 口語 可表示 不滿，為 ではありませんか、ではないか 、 じゃないか 或 じゃない 的 縮約，語調為 下降，中文翻譯成 嘛！、啊！。

接続

  • 普 + じゃん = 普 + じゃない
  • Vる + じゃん = Vる + じゃない
  • いA + じゃん = いA + じゃない
  • なA + じゃん = なA + じゃない
  • N + じゃん = N + じゃない

動詞

ねぇ、君、玄関の前に車を止めて、困るじゃない！
ねぇ、君、玄関の前に車を止めて、困るじゃん！ (縮約)

喂！你！在玄關前面停車，我們很困擾啊！

  • ねぇ [1]：感 喂
  • 君(きみ) [0]：代 你
  • 玄関(げんかん) [1]：名 大門
  • の：格助 表 所在
  • 前(まえ) [1]：名 前面
  • に：格助 表 定著點
  • 車(くるま) [0]：名 車
  • を：格助 接受 止める 的 賓語
  • 止(と)める [0]：動II 他動 辞書形 停止　
    • 止めて [0]：て形 停止
  • 困(こま)る [0]：動I 自動 困擾
    • 困るじゃん [0]：不滿 困擾啊！

  • え [1]：感 哎
  • 明日(あした) [3]：名 明天
  • 買(か)い物(もの) [0]：名 購物
  • に：格助 表 行動目的
  • 行(い)く [0]：動I 自動 去
  • って：縮約 と 表 內容
  • 言(い)う [0]：動I 自他 辞書形 說
    • 言った [0]：た形 說了
    • 言ったじゃん [0]：不滿 不是說了嘛！
