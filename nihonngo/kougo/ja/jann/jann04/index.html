普 じゃん = 普 じゃない (確認)

2024-01-06

じゃん 在 口語 可表示 驚訝，為 ではありませんか、ではないか 、 じゃないか 或 じゃない 的 縮約，語調為 上升，中文翻譯成 ...不是嗎？

接続

  • 普 + じゃん = 普 + じゃない
  • V普 + じゃん = V普 + じゃない
  • い形容 + じゃん = い形容 + じゃない
  • な形容 + じゃん = な形容 + じゃない
  • N + じゃん = N + じゃない

動詞

ねぇ、隣に田中って言う人がいるじゃない？あの人、宝くじに当たったって。
ねぇ、隣に田中って言う人がいるじゃん？あの人、宝くじに当たったって。 (縮約)

喂，隔壁不是有一個叫做田中的人嗎？聽說他中了彩券耶。
