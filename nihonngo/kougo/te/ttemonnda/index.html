ってもんだ = というものだ (是所謂的)

2024-09-11

というものだ 在 口語 常縮約成 ってもんだ 表示 是所謂的。

接続

  • 普 + ってもんだ = 普 + というものだ
  • V + ってもんだ = V + というものだ
  • いA + ってもんだ = いA + というものだ
  • なA + (だ) + ってもんだ = なA + (だ) + というものだ
  • N + (だ) + ってもんだ = N + (だ) + というものだ

動詞

失敗から学んでこそ、成長するというものだ。
失敗から学んでこそ、成長するってもんだ。 (縮約)

只有從失敗中學習，才是所謂的成長。

  • 失敗(しっぱい) [0]：名 失敗
  • から [1]：格助 表 從
  • 学(まな)ぶ [0]：動I 他動 辞書形 學習
    • 学んで [0]：て形 學習
    • 学んでこそ [0]：只有才 只有學習才
  • 成長(せいちょう) [0]：名 成長
    • 成長する [0]：動III 自動 辞書形 成長
    • 成長するというものだ [0]：是所謂的 是所謂的成長
    • 成長するってもんだ [0]：縮約 是所謂的成長
