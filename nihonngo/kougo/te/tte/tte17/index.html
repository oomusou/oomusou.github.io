んだって = んだと言うことだ (聽說)

2024-07-27

慣用語 んだと言うことだ 在 口語 常縮約成 んだって 表示 聽說。

接続

  • 普 + んだって = 普 + んだと言うことだ
  • Vる + んだって = Vる + んだと言うことだ
  • いA + んだって = いA + んだと言うことだ
  • なA + な + んだって = なA + な + んだと言うことだ
  • N + な + んだって = N + な + んだと言うことだ

動詞

明日、雨が降るんだと言うことだ。
明日、雨が降るんだって。 (縮約)

聽說明天會下雨。

  • 明日(あした) [3]：名 明天
  • 雨(あめ) [1]：名 雨
  • が：格助 表 主語
  • 降(ふ)る [1]：動I 自動 辞書形 下雨
    • 降るんだと言うことだ [1]：聽說 聽說會下雨
    • 降るんだって [1]：縮約 聽說會下雨

い形容詞

この店の料理は美味しいんだと言うことだ。
この店の料理は美味しいんだって。 (縮約)

聽說這間店的料理很好吃。

  • この [0]：連体 這個
  • 店(みせ) [2]：名 店
  • の：格助 以 名詞 形容 名詞
  • 料理(りょうり) [1]：名 料理
  • は：副助 表 主題
  • 美味(おい)しい [0]：い形容 好吃的
    • 美味しいんだと言うことだ [0]：聽說 聽說好吃的
    • 美味しいんだって [0]：縮約 聽說好吃的

な形容詞

このレストランは有名んだと言うことだ。
このレストランは有名なんだって。 (縮約)

聽說這間餐廳很有名。

  • この [0]：連体 這個
  • レストラン [1]：名 restaurant、餐廳
  • は：副助 表 主題
  • 有名(ゆうめい) [0]：な形容 有名的
    • 有名とんだと言うことだ [0]：聽說 聽說有名的
    • 有名なんだって [0]：縮約 聽說有名的

名詞

来週の月曜日学校休みんだと言うことだ。
来週の月曜日学校休みなんだって。 (縮約)

聽說下星期一學校休息。

  • 来週(らいしゅう) [0]：名 下週
  • の：格助 以 名詞 形容 名詞
  • 月曜日(げつようび) [3]：名 星期一
  • 学校(がっこう) [0]：名 學校
  • 休(やす)み [3]：名 休息
    • 休みとんだと言うことだ [3]：聽說 聽說休息
    • 休みなんだって [3]：縮約 聽說休息
