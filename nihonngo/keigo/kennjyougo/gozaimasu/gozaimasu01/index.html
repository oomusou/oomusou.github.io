ございます (だ 謙讓語)

2024-11-16

丁寧体 的 です 與 普通体 的 だ 在 謙讓語 可使用 ございます 表示，多用於 商務 或 挨拶 。

接続

  • いA (ア段) い + (オ段) + う + ございます
  • いA (イ段) い + (拗音) + う + ございます
  • いA (其他) い + う + ございます
  • なA + で + ございます
  • N + で + ございます
