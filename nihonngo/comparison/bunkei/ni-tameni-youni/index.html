に、ために、ように

2022-11-13

に、ために、ように 都可表示 目的，但彼此在使用上有些微差異。

接続

  • Vます ます + に + 移動V
  • N + に + 移動V
  • 意志V + ため + に、V
  • N + の + ため + に、V
  • 非意志V + よう + に、V
  • Vできる + よう + に、V
  • Vない + よう + に、V

例文

に

に 可表示 移動動詞 的 目的

移動動詞 如 行く、来る、帰る、戻る、入る、出る、出かける、登る …等

日本へ友達に会いに来た。

我來日本見朋友。

  • 日本(にほん) [2]：名 日本 (地名)
  • へ：格助 表 移動方向
  • 友達(ともだち) [0]：名 朋友
  • に：格助 表 目的
  • 会(あ)う [1]：動I 自動 辞書形 見面
    • 会います [3]：ます形 見面
    • 会いに来る [1]：來 來見面　
    • 会いに来た [1]：た形 來見面

スーパーへ買い物に行く。

我去超市購物。

  • スーパー [1]：名 supermarket、超市
  • へ：格助 表 移動方向
  • 買(か)い物(もの) [0]：名 購物
  • に：格助 表 目的
  • 行(い)く [0]：動I 自動 去

ために

ために 可表示 意志動詞 的 目的

意志動詞 如 見る、行く、食べる、飲む、歌う、話す … 等

家を買うために、貯金してる。

為了買房子，正在存錢。

  • 家(いえ) [2]：名 家
  • を：格助 接受 買う 的 賓語
  • 買(か)う [0]：動I 他動 買
  • ために [2]：接助 表 目的
  • 貯金(ちょきん)する [0]：動III 自動 辞書形 存錢
    • 貯金して [0]：て形 存錢
    • 貯金している [0]：正在進行 正在存錢
    • 貯金してる [0]：縮約 正在存錢

大学に入るために、勉強してる。

為了進大學，正在唸書。

  • 大学(だいがく) [0]：名 大學
  • に：格助 表 進入點
  • 入(はい)る [1]：動I 自動 進大學　
  • ために [2]：接助 表 目的
  • に：格助 表 目的
  • 勉強(べんきょう)する [0]：動III 自他 辞書形 唸書
    • 勉強して [0]：て形 唸書
    • 勉強している [0]：正在進行 正在唸書
    • 勉強してる [0]：縮約 正在唸書

健康のために、毎日ジョギンぐしてる。

為了健康，每天慢跑。

  • 健康(けんこう) [0]：名 健康
  • の：格助 以 名詞 形容 名詞
  • ために [2]：接助 表 目的
  • 毎日(まいにち) [1]：名 每天
  • ジョギンぐする [0]：動III 自動 辭書形 慢跑
    • ジョギングして [0]：て形 慢跑
    • ジョギングしている [0]：習慣反覆 都慢跑
    • ジョギングしてる [0]：縮約 都慢跑

ように

ように 可表示 非意志動詞 的 目的

非意志動詞 如 動詞可能形、動詞ない形、分かる、見える、聞こえる、減る、乾く … 等

家が買えるように、貯金してる。

為了能買房子，正在存錢。

  • 家(いえ) [2]：名 家
  • が：格助 表 主語
  • 買(か)う [0]：動I 他動 辞書形 買
    • 買える [0]：可能形 能買
  • ように [1]：接助 表 目的
  • 貯金(ちょきん)する [0]：動III 自動 辭書形 存錢
    • 貯金して [0]：て形 存錢
    • 貯金している [0]：正在進行 正在存錢
    • 貯金してる [0]：縮約 正在存錢

  • に：格助 表 進入點
  • 入(はい)る [1]：動I 自動 辭書形 進大學
    • 入れる [3]：可能形 能進大學
  • ように [1]：接助 表 目的
  • 勉強(べんきょう)する [0]：動III 自他 辭書形 唸書

  • 遅刻(ちこく)する [0]：動III 自動 辭書形 遲到
