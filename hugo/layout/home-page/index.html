Creating index.html for Home Page Layout

2022-03-07

Since home page has a different look than other content page, Hugo uses a separate layout layouts/index.html for the home page to generate.

Version

Hugo 0.93.1

Create Markdown

  • hogo new : create markdown file by the default archetypes/default.md
  • _index.md : create markdown for home page

_index.md is created by Hugo CLI.

_index.md is created under the content folder.

_index.md for Home Page

content/_index.md

Modify _index.md as above to provide page variable and content.

_index.md is a special for home page content

layouts/index.html

Home Page

Conclusion

  • layouts/_default/single.html is the default layout for single page, whereas layouts/index.html is the layout for home page
