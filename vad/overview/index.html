VAD 全名為 Voice Activity Detection，可偵測人聲的開始與結束，可藉此作判斷並將人聲送往伺服器，本文介紹純 HTML 方式。

  • onSpeechStart：當偵測到人聲開始時會觸發 SpeechStart event
  • onSpeechEnd：當偵測到人聲結束時會處方 SpeechEnd event
