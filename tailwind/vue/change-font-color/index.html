如何根據條件動態改變字體顏色 ?

2020-04-24

根據 Data 動態判斷條件而改變字體顏色是實務上常見需求，在 Vue 與 Tailwind 合作下有更簡單的實作方式。

Class Binding

<template>
  <div :class="isHighTemp(temperature)">
    {{ temperature }}
  </div>
</template>

<script>
let isHighTemp = x => ({ 'text-red-400': x > 37.5 })

export default {
  name:'app',
  data:() => ({
    temperature: 38.5
  }),
  methods:{
    isHighTemp
  }
}
</script>

11 行

data:() => ({
  temperature: 38.5
}),

若 temperature data 超過 37.5 度，我們想將字體動態改成 紅色。

顏色屬於 CSS 範圍，所以必須使用 Vue 的 class binding。

第 2 行

<div :class="isHighTemp(temperature)">
  {{ temperature }}
</div>

將 isHighTemp() 使用 class binding 綁定 class，根據條件動態改變字體顏色。

第 8 行

let isHighTemp = x => ({ 'text-red-400': x > 37.5 })

完全不用自己寫 CSS，直接使用 Tailwind 的 text-red-400 utility 即可。

Point-free

<template>
  <div :class="isHighTemp(temperature)">
    {{ temperature }}
  </div>
</template>

<script>
import { pipe, gt, flip, objOf } from 'ramda'

let isHighTemp = pipe(
  flip(gt)(37.5),
  objOf('text-red-400')
)

export default {
  name:'app',
  data: () => ({
    temperature: 38.5
  }),
  methods: {
    isHighTemp
  }
}
</script>

第 10 行

let isHighTemp = pipe(
  flip(gt)(37.5),
  objOf('text-red-400')
)

isHighTemp() 可由 Ramda 組合而成，使 isHighTemp() 能 point-free。

Conclusion

  • 改變顏色雖然屬於 CSS 範圍，但我們完全不需要寫任何 CSS，只需使用 Tailwind 的 text-red-400 utility 即可，也由於 Vue 的 class binding 主要是回傳 object，因此可搭配 Ramda 的 objOf() 組合 object 後回傳
