使用 flex-wrap-reverse 自動反向換列換行

2021-01-11

實務上若想 Item 在 Main Axis 方向總寬度大於外層時就自動反向換列，可使用 flex-wrap-reverse 達成。

Version

Tailwind CSS 2.0.2

flex-wrap-reverse

Item 總寬度大於 box，Flexbox 雖然自動換列，卻是從反方向開始顯示。

<template>
  <div class="flex flex-wrap-reverse w-11/12 h-screen">
    <div class="w-72 h-56 m-2">1</div>
    <div class="w-72 h-56 m-2">2</div>
    <div class="w-72 h-56 m-2">3</div>
    <div class="w-72 h-56 m-2">4</div>
    <div class="w-72 h-56 m-2">5</div>
  </div>
</template>
  • flex：使用 Flexbox
  • flex-wrap-reverse：設定自動反向換列
  • 各 item 總寬度大於外層，但由於 flex-wrap-reverse，因此所有 item 自動反向換列

flex-wrap-reverse 本質是 CSS 的 flex-wrap: wrap-reverse。

Conclusion

  • flex-wrap-reverse 實務上較少使用
