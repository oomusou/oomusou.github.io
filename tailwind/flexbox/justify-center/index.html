使用 justify-center 水平置中

2020-10-20

水平置中為實務上常見需求，Tailwind CSS 可由多種方式實現。

Version

Tailwind CSS 1.9.0

text-center

Tailwind CSS 水平置中。

<template>
  <div class="text-center">Tailwind CSS</div>
</template>
  • text-center：<div> 為 block element 會佔據一整行，直接使用 text-center 讓 content 水平置中於 <div> 即可

text-center 本質是 CSS 的 text-align: center。

mx-auto

<template>
  <div class="max-w-sm mx-auto text-center">Tailwind CSS</div>
</template>
  • max-w-sm：設定 max-width 搭配 mx-auto，唯一要求是 max-width 必須比 content 略寬，在此使用了最小的 max-w-sm
  • mx-auto：自動調整 <div> 左右 margin，因此 <div> 可水平置中
  • text-center：max-w-none 與 mx-auto 只能使的 <div> 能水平置中，因此須在補上 text-center 讓 content 在視覺上也能水平置中

mx-auto 主要原理是先使 <div> 水平置中於 browser，再使用 text-center 水平置中於 <div>

  • max-w-sm 本質是 CSS 的 max-width: 24rem
  • mx-auto 本質是 CSS 的 margin-left: auto 與 margin-right: auto
  • text-center 本質是 CSS 的 text-align: center

可發現 mx-auto 產生左右超大 margin，這也是只有一個 <div> 就能 水平置中 的原因。

flex-grow

<template>
  <div class="flex">
    <div class="flex-grow"></div>
    <div>Tailwind CSS</div>
    <div class="flex-grow"></div>
  </div>
</template>

一個簡單的思路，分成 3 個 <div>，置中 <div> 只跟著 content 本身而變，兩旁 <div> 自然瓜分剩下 width，如此 Tailwind CSS 看起來如同水平置中：

  • flex：外層 <div> 使用 flex
  • flex-grow：兩旁 <div> 使用 flex-grow 瓜分剩下 width

flex-grow 本質是 CSS 的 flex-grow: 1。

justify-center

<template>
  <div class="flex justify-center">
    <div>Tailwind CSS</div>
  </div>
</template>

一樣使用 flex，直接使用 justify-center，則其中的 <div> 會自動水平置中。

justify-center 本質是 CSS 的 justify-content: center。

Conclusion

  • 在單純的情況下，使用 text-center 就可水平置中
  • m-auto 只需一個 <div> 即可，justify-center 則必須再包一層 <div>
  • 若不知 justify-center 也能由 flex-grow 兜出來
  • 實務上建議使用 justify-center，因為 Flexbox 最為萬用
