Tailwind CSS 使用 CheckBox

2020-11-30

CheckBox 必須點到 方框 才能選取，實務上會想要點擊 Label 也相當於選擇 CheckBox，可將 <input> 與 <label> 合作達成。

Version

Tailwind CSS 2.0.1

CheckBox in Label

當 cursor 指向 label 時也會出現 pointer，表示點下 label 亦能點選 checkbox。

<template>
  <label class="hover:cursor-pointer"><input class="hover:cursor-pointer" type="checkbox">Swimming</label>
  <label class="hover:cursor-pointer"><input class="hover:cursor-pointer" type="checkbox">Reading</label>
  <label class="hover:cursor-pointer"><input class="hover:cursor-pointer" type="checkbox">Shopping</label>
</template>

第 2 行

<label class="hover:cursor-pointer"><input class="hover:cursor-pointer" type="checkbox">Swimming</label>
  • 將 <input> 包在 <label> 內，如此當點下 <label> 時相當於選擇 checkbox
  • <label> 與 <input> 的 class 皆套用 hover:cursor-pointer

Extract Class

<template>
  <label class="check"><input class="check" type="checkbox">Swimming</label>
  <label class="check"><input class="check" type="checkbox">Reading</label>
  <label class="check"><input class="check" type="checkbox">Shopping</label>
</template>

<style>
.check {
  @apply hover:cursor-pointer
}
</style>

第 8 行

.check {
  @apply hover:cursor-pointer
}

若覺得 hover:cursor-pointer 在 <label> 與 <input> 重複出現，也可使用 @apply 抽出 check class。

第 2 行

<label class="check"><input class="check" type="checkbox">Swimming</label>

如此 <label> 與 <input> 都套用 check class。

Div Container

<template>
  <div class="check">
    <label><input type="checkbox">Swimming</label>
    <label><input type="checkbox">Reading</label>
    <label><input type="checkbox">Shopping</label>
  </div>
</template>

<style>
.check label, .check input {
  @apply hover:cursor-pointer
}
</style>

第 2 行

<div class="check">
  <label><input type="checkbox">Swimming</label>
  <label><input type="checkbox">Reading</label>
  <label><input type="checkbox">Shopping</label>
</div>

若覺得每個 <label> 與 <input> 都要加上 check class 很煩，也可將三個 <label> 與 <input> 統一放在另一個 <div> 內，如此只要在 <div> 加上 check class 即可。

10 行

.check label, .check input {
  @apply hover:cursor-pointer
}

將 label 與 input 都放在 .check 下一起套用 @apply hover:cursor-pointer。

Sibling Label && CheckBox

<template>
  <input type="checkbox" class="hover:cursor-pointer" id="swimming"><label class="hover:cursor-pointer" for="swimming">Swimming</label>
  <input type="checkbox" class="hover:cursor-pointer" id="reading"><label class="hover:cursor-pointer" for="reading">Reading</label>
  <input type="checkbox" class="hover:cursor-pointer" id="shopping"><label class="hover:cursor-pointer" for="shopping">Shopping</label>
</template>

第 2 行

<input type="checkbox" class="hover:cursor-pointer" id="swimming"><label class="hover:cursor-pointer" for="swimming">Swimming</label>
  • 若不喜歡 <input> 放在 <label> 內，也可將 <input> 與 <label> 放在同一層，如此 <label> 要以 for 指向 <input> 的 id
  • <input> 與 <label> 的 class 都套用 hover:cursor-pointer

Extract Class

<template>
  <input type="checkbox" class="check" id="swimming"><label class="check" for="swimming">Swimming</label>
  <input type="checkbox" class="check" id="reading"><label class="check" for="reading">Reading</label>
  <input type="checkbox" class="check" id="shopping"><label class="check" for="shopping">Shopping</label>
</template>

<style>
.check {
  @apply hover:cursor-pointer
}
</style>

第 8 行

.check {
  @apply hover:cursor-pointer
}

若覺得 hover:cursor-pointer 在 <label> 與 <input> 重複出現，也可使用 @apply 抽出 check class。

第 2 行

<input type="checkbox" class="check" id="swimming"><label class="check" for="swimming">Swimming</label>

如此 <label> 與 <input> 都套用 check class。

Div Container

<template>
  <div class="check">
    <input type="checkbox" id="swimming"><label for="swimming">Swimming</label>
    <input type="checkbox" id="reading"><label for="reading">Reading</label>
    <input type="checkbox" id="shopping"><label for="shopping">Shopping</label>
  </div>
</template>

<style>
.check input, .check label {
  @apply hover:cursor-pointer
}
</style>

第 2 行

<div class="check">
  <input type="checkbox" id="swimming"><label for="swimming">Swimming</label>
  <input type="checkbox" id="reading"><label for="reading">Reading</label>
  <input type="checkbox" id="shopping"><label for="shopping">Shopping</label>
</div>

若覺得每個 <input> 與 <labe> 都要加上 check class 很煩，也可將三個 <label> 與 <input> 統一放在另一個 <div> 內，如此只要在 <div> 加上 check class 即可。

第 10 行

.check input, .check label {
  @apply hover:cursor-pointer
}

將 label 與 input 都放在 .check 下一起套用 @apply hover:cursor-pointer。

Conclusion

  • Tailwind 雖然習慣將 utility 直接寫在 HTML 的 class 內，若相同的 utility 一再重複出現，也可使用 CSS selector 技巧與 @apply 將 utility 抽出新 class 避免重複
