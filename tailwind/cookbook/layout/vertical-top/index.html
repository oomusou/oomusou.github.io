如何垂直靠上 ?

2020-12-31

垂直靠上為實務上常見需求，Tailwind CSS 可由多種方式實現。

Version

Tailwind CSS 2.0.1

flex-grow

Tailwind CSS 垂直靠上於 <div>。

<template>
  <div class="flex flex-col h-32">
    <div>Tailwind CSS</div>
    <div class="flex-grow"/>
  </div>
</template>

一個簡單的思路，分成 2 個 <div>，在 Flexobx 中 <div> 未設定 height，會自動 shrink 跟著 content 本身而變，下方 <div> 自然瓜分剩下 height，如此 Tailwind CSS 看起來如同垂直靠上。

  • flex：外層 <div> 使用 Flexbox
  • flex-col：設定 main axis 為 column
  • h-32：要垂直靠上一定要設定 height
  • flex-grow：下方 <div> 使用 flex-grow 瓜分剩下 height

justify-start

Tailwind CSS 垂直靠上於 <div>。

<template>
  <div class="flex flex-col justify-start h-32">
    <div>Tailwind CSS</div>
  </div>
</template>

一樣使用 flex，Flexbox 內只有一個 <div> 即可。

  • flex：外層 <div> 使用 Flexbox
  • flex-col：設定 main axis 為 column
  • justify-start：由於 <div> 從上至下，因此 justify-start 就是垂直靠上
  • h-32：要垂直靠上一定要設定 height

justify-end

Tailwind CSS 垂直靠上於 <div>。

<template>
  <div class="flex flex-col-reverse justify-end h-32">
    <div>Tailwind CSS</div>
  </div>
</template>

一樣使用 flex，Flexbox 內只有一個 <div> 即可。

  • flex：外層 <div> 使用 Flexbox
  • flex-col-reverse ：使 <div> 從下至上
  • justify-end：由於 <div> 從下至上，因此 justify-end 就是垂直靠上
  • h-32：要垂直靠上一定要設定 height

實務上不會使用這種方式，只是展示亦可使用 justify-start 達成垂直靠上

items-start

Tailwind CSS 垂直靠上於 <div>。

<template>
  <div class="flex items-start h-32">
    <div>Tailwind CSS</div>
  </div>
</template>

一樣使用 flex，Flexbox 內只有一個 <div> 即可。

  • flex：外層 <div> 使用 Flexbox
  • items-start：由於沒使用 flex-col，因此目前 main axis 仍是 row，可使用 items-start 直接靠上於 cross axis，相當於垂直靠上
  • h-32：要垂直靠上一定要設定 height

Conclusion

  • 只要記住 justify-content 系列是根據 Flexbox 的 main axis；而 align-items 系列是根據與 cross axis
  • 若以寫法論，使用 items-start 直接根據 cross axis 垂直靠上最精簡
