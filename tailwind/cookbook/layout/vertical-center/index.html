如何垂直置中 ?

2020-12-30

垂直置中為實務上常見需求，Tailwind CSS 可由多種方式實現。

Version

Tailwind CSS 2.0.1

flex-grow

Tailwind CSS 垂直置中。

<template>
  <div class="flex flex-col h-32">
    <div class="flex-grow"/>
    <div>Tailwind CSS</div>
    <div class="flex-grow"/>
  </div>
</template>

一個簡單的思路，分成 3 個 <div>，在 Flexobx 中 <div> 未設定 height，會自動 shrink 跟著 content 本身而變，兩旁 <div> 自然瓜分剩下 height，如此 Tailwind CSS 看起來如同垂直置中。

  • flex：外層 <div> 使用 Flexbox
  • flex-col：設定 main axis 為 column
  • flex-grow：上下 <div> 使用 flex-grow 瓜分剩下 height
  • h-32：要垂直置中一定要設定 height

justify-between

Tailwind CSS 垂直置中。

<template>
  <div class="flex flex-col justify-between h-32">
    <div/>
    <div>Tailwind CSS</div>
    <div/>
  </div>
</template>

與 flex-grow 一樣使用三個 <div>，上下為 dummy div。

  • flex：外層 <div> 使用 Flexbox
  • flex-col：設定 main axis 為 column
  • justify-between：原本為將剩餘 height 自動平分給 item 間剩餘空間，因為上下 dummy div 沒有高度，平分剩餘空間後 Tailwind CSS 看起來像垂直置中
  • h-32：要垂直置中一定要設定 height

justify-center

Tailwind CSS 垂直置中。

<template>
  <div class="flex flex-col justify-center h-32">
    <div>Tailwind CSS</div>
  </div>
</template>

一樣使用 flex，Flexbox 內只有一個 <div> 即可。

  • flex：外層 <div> 使用 Flexbox
  • flex-col：設定 main axis 為 column
  • justify-center：直接將 <div> 垂直置中
  • h-32：要垂直置中一定要設定 height

justify-around

Tailwind CSS 垂直置中。

<template>
  <div class="flex flex-col justify-around h-32">
    <div>Tailwind CSS</div>
  </div>
</template>

一樣使用 flex，Flexbox 內只有一個 <div> 即可。

  • flex：外層 <div> 使用 Flexbox
  • flex-col：設定 main axis 為 column
  • justify-around：原本與 justify-between 一樣將剩餘高度自動平分給 item 間剩餘空間，但不同的是 justify-around 會考慮 item 的上下空間，因此平分後看起來 Tailwind CSS 像垂直置中
  • h-32：要垂直置中一定要設定 height

justify-evenly

Tailwind CSS 垂直置中。

<template>
  <div class="flex flex-col justify-evenly h-32">
    <div>Tailwind CSS</div>
  </div>
</template>

一樣使用 flex，Flexbox 內只有一個 <div> 即可。

  • flex：外層 <div> 使用 Flexbox
  • flex-col：設定 main axis 為 column
  • justify-evenly：原本與 justify-between 一樣將剩餘高度自動平分給 item 間剩餘空間，但不同的是 justify-around 會考慮 item 的上下空間，因此平分後 Tailwind CSS 看起來像垂直置中
  • h-32：要垂直置中一定要設定 height

justify-around 與 justify-evenly 差異是儘管包含 item 上下空間，但 justify-evenly 是真正 evenly 等高，但 justify-around item 間空間會是前後高度的兩倍

items-center

Tailwind CSS 垂直置中。

<template>
  <div class="flex items-center h-32">
    <div>Tailwind CSS</div>
  </div>
</template>

一樣使用 flex，Flexbox 內只有一個 <div> 即可。

  • flex：外層 <div> 使用 Flexbox
  • items-center：由於沒更改 flex-direction，因此目前 main axis 仍是 row，可使用 items-center 直接置中於 cross axis，相當於垂直置中
  • h-32：要垂直置中一定要設定 height

Conclusion

  • 只要記住 justify 系列是根據 Flexbox 的 main axis；而 items 系列是根據與 cross axis
  • 若以寫法論，使用 items-center 直接根據 cross axis 垂直置中最精簡
