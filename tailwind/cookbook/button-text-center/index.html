如何使 button 內的文字與 SVG 置中 ?

2020-08-23

若要使 <button> 同時有文字與圖片，該如何使其同時水平與垂直置中呢 ?

Version

macOS Catalina 10.15.6
WebStorm 2020.2
Vue 2.6.11
Tailwind CSS 1.7.1

inlin-flex items-center

View article 與箭頭剛好置中於 <button> 內。

<template>
  <div>
    <button class="inline-flex items-center px-6 py-3 text-white font-semibold bg-blue-700 rounded-md shadow-sm">
      <span>View article</span>
      <svg class="ml-3 w-5 h-5" fill="currentColor" viewBox="0 0 20 20">
        <path fill-rule="evenodd" d="M10.293 3.293a1 1 0 011.414 0l6 6a1 1 0 010 1.414l-6 6a1 1 0 01-1.414-1.414L14.586 11H3a1 1 0 110-2h11.586l-4.293-4.293a1 1 0 010-1.414z" clip-rule="evenodd"></path>
      </svg>
    </button>
  </div>
</template>

<script>
export default {
  name: 'app',
}
</script>

第 3 行

<button class="inline-flex items-center px-6 py-3 text-white font-semibold bg-blue-700 rounded-md shadow-sm">
  <span>View article</span>
  <svg></svg> 
</button>

將 <span> 與 <svg> 放在 <button> 內，在 <button> 內加上 inline-flex 與 items-center utility。

  • inline-flex：讓 <button> 成為 flex container，但使 <button> 成為 inline element，主要是使 <button> 不再是 block element 佔一行決定 width，而是由其內部的 <span> 與 <svg> 的 width 決定
  • items-center：由於 <button> 已經成為 flex container，items-center 使 <span> 與 <svg> 在 cross-axis 能置中，也就是 垂直置中

若不下 items-center，會發現 <svg> 略為偏上

Conclusion

  • Tailwind 巧妙地使用 flex 特性將文字與圖片置中，只要加上 inline-flex 與 items-center 即可
