Gridsome 安裝 Tailwind CSS

2020-11-03

Gridsome 基於 Vue，也可以使用 Tailwind CSS，與 Vue CLI 設定上大體類似，只有在 Gridsome Configuration 稍有不同。

Version

Gridsome 0.7.21
Tailwind CSS 1.9.6

Create Project

$ gridsome create gridsome-tailwind

使用 Gridsome CLI 建立 project。

Install Package

$ yarn add tailwindcss

使用 Yarn 安裝 Tailwind。

Add Tailwind CSS

src/assets/css/tailwind.css

@tailwind base;
@tailwind components;
@tailwind utilities;

新增 src/assets/css/tailwind.css。

Tailwind 會在 build-time 以標準 CSS 取代。

Tailwind Configuration

$ npx tailwindcss init

使用 tailwind init 在 project 根目錄建立 tailwind.config.js。

tailwind.config.js

module.exports = {
  purge: [
    './src/**/*.vue',
    './src/**/*.html'
  ],
  theme: {
    extend: {},
  },
  variants: {},
  plugins: [],
}
  • purge：直接設定 PuregeCSS 要管理的檔案

Gridsome Configuration

gridsome.config.js

let tailwindcss = require("tailwindcss")

module.exports = {
  siteName: 'Gridsome',
  plugins: [],
  css: {
    loaderOptions: {
      postcss: {
        plugins: [tailwindcss],
      },
    },
  }
}

第 6 行

css: {
  loaderOptions: {
    postcss: {
      plugins: [tailwindcss],
    },
  },
}

在 gridsome.config.js 設定 PostCSS 使用 Tailwind。

PostCSS Configuraton

postcss.config.js

module.exports = {
  plugins: [
    require('tailwindcss')
  ]
}

在 project 根目錄建立 postcss.config.js，由 PostCSS 啟動 Tailwind。

Main

main.js

import DefaultLayout from '~/layouts/Default.vue'
import '@/assets/css/tailwind.css'

export default function (Vue, { router, head, isClient }) {
  // Set default layout as a global component
  Vue.component('Layout', DefaultLayout)
}

第 2 行

import '@/assets/css/tailwind.css'

在 main.js 引入 tailwind.css。

Component

src/pages/Index.vue

<template>
  <div>
    <h1 class="text-4xl font-bold text-red-700">Hello, world!</h1>
  </div>
</template>

<script>
export default {
  name: 'index'
}
</script>

第 3 行

<h1 class="text-4xl font-bold text-red-700">Hello, world!</h1>

直接在 <h1/> 套用 Tailwind 的 utility，可發現其可讀性很高，就算不熟 Tailwind，也能由其 class name 了解其意義。

Development Mode

$ yarn serve

啟動 development mode 的 DevServer。

可發現 Tailwind CSS 已經成功套用。

Production Mode

$ yarn build

使用 production mode 編譯 Gridsome 並啟動 PurgeCSS。

在 dist/css 目錄下，可發現 CSS 只有 4 kb，因為 PurgeCSS 只取出用到部分。

Conclusion

  • Tailwind 基於 PostCSS，搭配 PurgeCSS 後，才可大幅降低 CSS size
  • Tailwind 不斷精簡其設定步驟，目前已經非常精簡
