使用 group-hover 讓子層改變 Style

2021-04-17

hover variant 原本只能改變自己的 Style，若要改變子層的 Style，則要使用 group-hover variant。

Version

Tailwind CSS 2.1.1

group-hover

一開始 box 為灰色。

當滑鼠 hover 過水平 block 則 box 變成紅色。

<template>
  <div class="group">
    <div class="w-20 h-20 mx-auto bg-gray-300 group-hover:bg-red-500"/>
  </div>
</template>

第 2 行

<div class="group">

設定父層水平 block style：

  • group：設定要感應 hover 的父層

第 3 行

<div class="w-20 h-20 mx-auto bg-gray-300 group-hover:bg-red-500"/>

設定子層 box style：

  • w-20：設定子層 box 寬度
  • h-20：設定子層 box 高度
  • mx-auto：設定子層 box 水平置中
  • bg-gray-300：設定子層 box 背景顏色為灰色
  • group-hover:bg-red-500：當父層 hover 時，子層 box 背景改為紅色

Conclusion

  • 若要藉由 hover 改變子層 style，則在父層使用 group，子層加上 group-hover 即可
