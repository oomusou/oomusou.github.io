使用 h-{size} 設定 Height

2020-07-30

若要設定 Height，可使用 Tailwind 的 h-{size} 系列 Utility。

Version

macOS Catalina 10.15.4
WebStorm 2019.3.4
Vue 2.6.11
Tailwind 1.2.0

Auto

<template>
  <div>
    <img class="h-auto" src="./beach-work.jpg"/>
  </div>
</template>
<script>
export default {
  name:'app'
}
</script>

使用 h-auto 由 browser 根據 margin、padding 與 border 決定 width。

h-auto 本質是 CSS 的 height: auto。

Screen Height

<template>
  <div>
    <img class="h-screen" src="./beach-work.jpg"/>
  </div>
</template>
<script>
export default {
  name:'app',
}
</script>

使用 h-screen 由整個垂直 viewpoint 決定高度。

h-screen 本質是 CSS 的 height: 100vw。

Fixed Height

<template>
  <div>
    <img class="h-64" src="./beach-work.jpg"/>
  </div>
</template>
<script>
export default {
  name:'app',
}
</script>

使用 h-{size} 設定固定高度，數字越大表高度越大。

h-64 本質是 CSS 的 height: 16rem。

Fluid Height

<template>
  <div>
    <img class="h-full" src="./beach-work.jpg"/>
  </div>
</template>
<script>
export default {
  name:'app',
}
</script>

使用 h-full 設定高度為 100%。

h-full 本質是 CSS 的 height: 100%。

Conclusion

  • h-{size} 系列 utility 讓我們可以直接對 HTML element 設定 height，而不用另外撰寫 CSS
