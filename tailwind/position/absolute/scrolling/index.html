2021-12-27

Line 20

<div class="absolute w-20 h-20 bg-red-500">box</div>
  • absolute : use absolute position
  • w-20 : set block width
  • h-20 : set block height
  • bg-red-500 : set background color

When using absolute, the element is floated to make a new layer and shrink the width as content. This is the same as fixed.

But when scrolling the page, the position of the red box is changed. This is the difference from fixed.

Line 41

<div class="absolute top-0 right-0 w-20 h-20 bg-red-500">box</div>

Other utilities are the same, but we add top and right to specfify position :

  • top-0：set coordinate from top
  • right-0：set coordinate from right

We can also specify top, bottom, left ,and right on absolute as fixed. But absolute is only position on the first page, not for scrolling, and fixed is locked regardless of scrolling.

Summary

Fixed Absolute
New Layer O O
Shrink width as content O O
Position on the browser O X
Position on the first page X O
Scrollable X O

Conclusion

  • fixed and absolute are very similiar and but absolute doesn’t lock the position when scrolling the page
