使用 Absolute Position 實現定位

2021-03-22

使用 Absolute Position 時搭配 top、bottom、left 與 right 定位時，會往父層尋找具備定位設定的父層。

Version

Tailwind CSS 2.0.3

Flow Layout

紅色 box 位於三層 <div> 內，且每層 <div> 都有其 padding。

<template>
  <div class="p-4">
    <div class="p-4">
      <div class="p-4">
        <div class="w-32 h-32 bg-red-600">
          box
        </div>
      </div>
    </div>
  </div>
</template>

box 外共有三層 <div>，且每層都有 p-4 padding。

第 5 行

<div class="w-32 h-32 bg-red-600">
  box
</div>

設定 box style：

  • w-32：設定 box width
  • h-32：設定 box height
  • bg-red-600：設定 box 背景顏色

Absolute Solution

紅色 box 從原處浮起成新 layer，第三層 <div> 則內縮只剩下 padding 部分。

<template>
  <div class="p-4">
    <div class="p-4">
      <div class="p-4">
        <div class="absolute w-32 h-32 bg-red-600">
          box
        </div>
      </div>
    </div>
  </div>
</template>

第 5 行

<div class="absolute w-32 h-32 bg-red-600">
  box
</div>

設定 box style：

  • absolute：如同 fixed 一樣，將從原處浮起成立新 layer，因此不佔原本空間，導致 <div> 內縮只剩下 padding

Position at 3rd div

紅色 box 左側緊貼第三層 <div>。

<template>
  <div class="p-4">
    <div class="p-4">
      <div class="relative p-4">
        <div class="absolute left-0 w-32 h-32 bg-red-600">
          box
        </div>
      </div>
    </div>
  </div>
</template>

第 5 行

<div class="absolute left-0 w-32 h-32 bg-red-600">
  box
</div>

設定 box style：

  • absolute：使用 absolute position
  • left-0：設定 left 為 0，但此時還不知以誰為定位基準，將往父層尋找具備定位設定的父層

若使用 fixed，則恆以 window 為基準

第 4 行

<div class="relative p-4">

設定第三層 <div> style：

  • relative：設定此層為 relartive 定位

absolute 找到第三層 <div> 而停止，left 將以第三層為標準定位

Position at 2nd div

紅色 box 左側緊貼第二層 <div>。

<template>
  <div class="p-4">
    <div class="relative p-4">
      <div class="p-4">
        <div class="absolute left-0 w-32 h-32 bg-red-600">
          box
        </div>
      </div>
    </div>
  </div>
</template>

第 3 行

<div class="relative p-4">

設定第二層 <div> style：

  • relative：設定此層為 relartive 定位

absolute 找到第二層 <div> 而停止，left 將以第二層為標準定位

Position at 1st div

紅色 box 左側緊貼第一層 <div>。

<template>
  <div class="relative p-4">
    <div class="p-4">
      <div class="p-4">
        <div class="absolute left-0 w-32 h-32 bg-red-600">
          box
        </div>
      </div>
    </div>
  </div>
</template>

第 2 層

<div class="relative p-4">

設定第一層 <div> style：

  • relative：設定此層為 relartive 定位

absolute 找到第一層 <div> 而停止，left 將以第一層為標準定位

Position at Window

紅色 box 左側緊貼 browser。

<template>
  <div class="p-4">
    <div class="p-4">
      <div class="p-4">
        <div class="absolute left-0 w-32 h-32 bg-red-600">
          box
        </div>
      </div>
    </div>
  </div>
</template>

並沒有任何 <div> 設定 relative，將定位到最上層的 window。

當父層沒有任何 <div> 使用 relative 時，absolute 相當於 fixed

Conclusion

  • 由於 Tailwind 自帶 CSS reset，因此定位在第一層 <div> 與定位在 window 看起來相同
  • 當只有設定 absolute 時，將從原處浮起新 layer，此時等效於 fixed
  • 當 absolute 搭配 top、bottom、left 或 right 定位時，會往父層尋找具備定位設定的父層
  • 當 absolute 搭配 top、bottom、left 或 right 定位時，若在父層都找不到任何有定位的父層，則定位在 window，此時等效於 fixed
