使用 Tailwind CSS 建立 Button

2020-10-11

初學者使用 Tailwind CSS 時，會發會原本 <button> 會不正常，因為 Tailwind CSS 自帶 CSS Reset，將原本 <button> 預設的 Property 都 Reset 了。

Version

Tailwind CSS 1.8.7

Button

<template>
  <button>Submit</button>
</template>

在沒有使用 Tailwind CSS 下，在 HTML 僅使用簡單 <button> 就可建立 button。

因為 browser 已經為 <button> 提供很多預設 property 設定。

Tailwind Button

在 Tailwind 下若使用簡單 <button>，會發現 browser 下已經不像 button，因為 Tailwind 內建 CSS Reset 了，<button> 已經不具備預設 property。

<template>
  <button class="bg-gray-200 border-2 rounded border-solid border-gray-600 px-3 py-1 m-2">Submit</button>
</template>

若要做出與 browser 預設 button 相同外觀，可自行加入適當 utility：

  • bg-gray-200： 設定 button 背景顏色
  • border-2： 設定 border width
  • rounded：設定 border 具有圓角
  • border-solid：設定 border 有實體線
  • border-gray-600：設定 border 顏色
  • px-3 py-1：設定 padding
  • m-2：設定 margin

Real World Button

實務上幾乎不會使用 browser 的預設 button，而會自行採用 CSS 加以修飾 style。

<template>
  <button class="bg-blue-500 text-white rounded px-3 py-1 m-2">Submit</button>
</template>
  • bg-blue-500：設定背景顏色
  • text-white：設定文字顏色
  • rounded：設定 border 具有圓角
  • px-3 py-1：設定 padding
  • m-2：設定 margin

Extract Component

<template>
  <button class="btn btn-blue">Submit</button>
</template>

<style scoped>
.btn {
  @apply rounded px-4 py-2; 
}

.btn-blue {
  @apply bg-blue-500 text-white;
}
</style>

若 style 在多處使用，也可抽成 CSS component。

第 6 行

.btn {
  @apply rounded px-4 py-2; 
}

使用 @apply 將基礎共用部分的 rounded py-2 px-4 抽成 btn component。

10 行

.btn-blue {
  @apply bg-blue-500 text-white;
}

將顏色部分抽成 btn-blue component。

第 2 行

<button class="btn btn-blue">Submit</button>

如此 <button> 可組合 btn 與 btn-blue 建立 button。

Global Component

assets/css/tailwind.css

@tailwind base;
@tailwind components;
@tailwind utilities;

@layer components {
  .btn {
    @apply rounded px-4 py-2; 
  }

  .btn-blue {
    @apply bg-blue-500 text-white;
  }
}

也可在 tailwind.css 建立 btn 與 btn-blue global component，如此所有 HTML 都可直接使用。

第 5 行

@layer components {
  .btn {
    @apply rounded px-4 py-2; 
  }

  .btn-blue {
    @apply bg-blue-500 text-white;
  }
}

由於 btn 與 btn-blue 屬於 component，所以寫在 @layer components 下。

App.vue

<template>
  <button class="btn btn-blue">Submit</button>
</template>

如此 HTML 可直接使用 btn 與 btn-blue。

Conclusion

  • Tailwind 由於自帶 CSS reset，因此 <button> 預設 property 都已喪失
  • 實務上 button 都會自行以 CSS 重新修飾 style，所以喪失預設 property 也沒關係
  • 如 button 的 style 需重複使用，可使用 @apply 抽成 CSS component，亦可放在 tailwind.css 成為 global component
  • 在抽成 component 時，也必須遵守 Functional CSS 精神，將 CSS 拆成最小單位，以利 Function Composition
