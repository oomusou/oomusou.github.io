使用 inline 將 Block 轉成 Inline

2020-12-08

HTML 將 Element 分成 Inline、Block 與 Inline-block Element，透過 inline Utility 可將 Block Element 強制轉成 Inline Element。

Version

Tailwind CSS 2.0.1

Inline Element

代表性的 Inline Element 有：

  • <span>
  • <a>
  • <input>
  • <img>

Block Element

代表性 Block Element 有：

  • <div>
  • <ul>、<li>
  • <p>
  • <h1>

div

<template>
  <div>Tailwind</div>
  <div>Rocks</div>
</template>

由於 <div> 是 block，因此使用多個 <div> 時會換行。

inline

<template>
  <div class="inline">Tailwind</div>
  <div class="inline">Rocks</div>
</template>

若要使 <div> 如同 <span> 那樣不換行，可使用 inline 轉換。

inline 本質是 CSS 的 display: inline。

Conclusion

  • 透過 inline 可將原本 block 轉成 inline

Reference

MDN, display
Tailwind CSS, Inline
