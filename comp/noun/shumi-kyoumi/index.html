趣味、興味

2022-11-11

趣味 與 興味 的漢字看起來很像，但意思不太一樣。趣味 是 興趣，興味 是 感興趣。

単語

  • 趣味(しゅみ) [1]：名 興趣
  • 興味(きょうみ) [1]：名 感興趣

接続

  • 趣味 + は + N + です
  • N + に + 興味 + が + ある

例文

趣味

趣味はテニスです。

我的興趣是網球。

  • 趣味(しゅみ) [1]：名 興趣
  • は：副助 表 主題
  • テニス [1]：名 tennis、網球
  • です：助動 是

興味

テニスに興味がある。

我對網球感興趣。

  • テニス [1]：名 tennis、網球
  • に：格助 表 動作對象
  • 興味(きょうみ) [1]：名 感興趣
  • が：格助 表 主語
  • ある [1]：動I 自動 有
