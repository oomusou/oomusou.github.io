の、こと

2025-01-12

動詞 之後為了接 格助詞，可接上 形式名詞 の、こと 成為名詞，大部分情況下使用 の 或 こと 皆可，但有些情況只能接 の 或 こと，由 格助詞 之後的 動詞 決定。

接続

  • V + の + 格助 + V
  • V + こと + 格助 + V

の

の 源自於 もの 代表 實體，之後接 五感 或 行動 類 動詞。

  • 五感：見る、見える、聞く、聞こえる、感じる
  • 行動：待つ、手伝う、助ける、辞める、止める

こと

こと 源自於 事 代表 抽象，之後接 腦 或 言語 類 動詞。

  • 腦：思う、願う、望む、祈る、信じる、考える、決める
  • 言語：言う、話す、聞く、書く、伝える、相談する、約束する

の / こと

除了上述情況外，使用 の 或 こと 皆可，の 用於 口語，こと 用於 書面。
