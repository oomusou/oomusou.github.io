使用 flex 1 水平均分寬度

2020-10-16

水平平分寬度也是實務上常見需求，有多種寫法，但以 flex: 1 最為精簡。

Version

CSS 3

Width

三個 <div> 水平均分所有寬度。

<template>
  <div class="row">
    <div class="column red">1</div>
    <div class="column green">2</div>
    <div class="column blue">3</div>
  </div>
</template>

<style scoped>
.row {
  display: flex
}

.column {
  width: 33.33%;
  color: white;
}

.red {
  background-color: red;
}

.green {
  background-color: green;
}

.blue {
  background-color: blue;
}
</style>

14 行

.column {
  width: 33.33%;
  color: white;
}

最直覺是使用 width，每個 <div> 在 Flexbox 的 width 為 33.33%，如此剛好三分水平寬度。

10 行

.row {
  display: flex
}

此時並沒有 Flexbox，只是使用 flex 讓 <div> 不要佔據一整行，改成水平並列而已。

這種寫法雖可行，但指定 width 顯然不是最優寫法，尤其若不是 width 所提供寬度時

Grid

<template>
  <div class="row">
    <div class="column red">1</div>
    <div class="column green">2</div>
    <div class="column blue">3</div>
  </div>
</template>

<style scoped>
.row {
  display: grid;
  grid-template-columns: repeat(3, minmax(0, 1fr));
}

.column {
  color: white;
}

.red {
  background-color: red;
}

.green {
  background-color: green;
}

.blue {
  background-color: blue;
}
</style>

10 行

.row {
  display: grid;
  grid-template-columns: repeat(3, minmax(0, 1fr));
}

也可使用 Grid 做水平均分，直接在 row 以 grid-template-columns 決定要均分 column 即可，每個 column 都不需再設定任何 style。

flex-grow

<template>
  <div class="row">
    <div class="column red">1</div>
    <div class="column green">2</div>
    <div class="column blue">3</div>
  </div>
</template>

<style scoped>
.row {
  display: flex
}

.column {
  flex-grow: 1;
  color: white;
}

.red {
  background-color: red;
}

.green {
  background-color: green;
}

.blue {
  background-color: blue;
}
</style>

14 行

.column {
  flex-grow: 1;
  color: white;
}

也可以三個 <div> 都使用 flex-grow，如此剛好均分所有寬度，也不必將 width 寫死。

flex: 1

<template>
  <div class="row">
    <div class="column item1">1</div>
    <div class="column item2">2</div>
    <div class="column item3">3</div>
  </div>
</template>

<style scoped>
.row {
  display: flex
}

.column {
  flex: 1;
  color: white;
}

.red {
  background-color: red;
}

.green {
  background-color: green;
}

.blue {
  background-color: blue;
}
</style>

14 行

.column {
  flex: 1;
  color: white;
}

更精簡方式是使用 flex: 1，如此一次指定 flex-grow 與 flex-shrink，更為精簡。

Conclusion

  • 實務上建議使用 flex-1 最為精簡
