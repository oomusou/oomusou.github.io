CSS 之 Attribute Selector

2020-07-16

僅管是相同 Tag，只要其 Attribute 的 Value 不一樣，就能使用 Attribute Selector 選擇該 Tag。

Version

macOS Catalina 10.15.5
WebStorm 2020.1.3
Vue 2.6.11
CSS 3

Select Class Attribute

想選擇所有使用 class attribute 的 tag。

<template>
  <div>
    <div class="nickname">Oomusou</div>
    <div class="first-name">Sam</div>
    <div>Xiao</div>
  </div>
</template>

<script>
export default {
  name: 'App',
}
</script>

<style scoped>
[class] {
  color: red;
}
</style>

第 3 行

<div class="nickname">Oomusou</div>
<div class="first-name">Sam</div>
<div>Xiao</div>

Oomusou 使用 nickname class，Sam 則使用 first-name class，而 Xiao 並沒有使用任何 class。

16 行

[class] {
  color: red;
}

若要選擇 Oomusou 與 Sam，可選擇所有 tag 中具有 class attribute。

Select Custom Attribute

其實 attribute selector 不只能用在 class attribute 而已，也能用在自訂 attribute。

<template>
  <div>
    <div my-name="nickname">Oomusou</div>
    <div my-name="first-name">Sam</div>
    <div>Xiao</div>
  </div>
</template>

<script>
export default {
  name: 'App',
}
</script>

<style scoped>
[my-name] {
  color: red;
}
</style>

第 3 行

<div my-name="nickname">Oomusou</div>
<div my-name="first-name">Sam</div>
<div>Xiao</div>

Oomusou 與 Sam 使用了自訂的 my-name attribute，而 Xioa 則有沒使用任何 attribute。

16 行

[my-name] {
  color: red;
}

若要選擇 Oomusou 與 Sam，可選擇所有 tag 中具有 my-name attribute。

Select Class Attribute on Tag

Attribute selector 也可只用在指定 tag 下。

<template>
  <div>
    <div class="nickname">Oomusou</div>
    <p class="first-name">Sam</p>
    <div>Xiao</div>
  </div>
</template>

<script>
export default {
  name: 'App',
}
</script>

<style scoped>
p[class] {
  color: red;
}
</style>

第 3 行

<div class="nickname">Oomusou</div>
<p class="first-name">Sam</p>
<div>Xiao</div>

Oomusou 與 Sam 都使用了 class attribute，唯 Oomusou 在 <div> 下，而 Sam 在 <p> 下。

16 行

p[class] {
  color: red;
}

若要選擇 Sam，可選擇所有 tag 中為 <p> 並使用 class attribute，syntax 為 [] 前搭配指定 tag。

[attribute=”value”]

Attribute selector 可搭配各種 operator 使用，如以 = 選擇 attribute 等於 指定 value。

<template>
  <div>
    <div class="nickname">Oomusou</div>
    <div class="first-name">Sam</div>
    <div>Xiao</div>
  </div>
</template>

<script>
export default {
  name: 'App',
}
</script>

<style scoped>
[class="nickname"] {
  color: red;
}
</style>

第 3 行

<div class="nickname">Oomusou</div>
<div class="first-name">Sam</div>
<div>Xiao</div>

Oomusou 使用 nickname class，而 Sam 使用 first-name class。

16 號

[class="nickname"] {
  color: red;
}

若要選擇 Oomusou，可選擇所有 tag 中 class attribute 為 nickname。

Class Selector

Attribute selector 搭配 =，實務上會直接使用 class selector 較精簡。

<template>
  <div>
    <div class="nickname">Oomusou</div>
    <div class="first-name">Sam</div>
    <div>Xiao</div>
  </div>
</template>

<script>
export default {
  name: 'App',
}
</script>

<style scoped>
.nickname {
  color: red;
}
</style>

16 行

.nickname {
  color: red;
}

Attribute selector 搭配 =，其等效於 class selector。

[attribute~=”value”]

以 ~= 選擇 attribute 包含 指定 value。

<template>
  <div>
    <div class="name nickname">Oomusou</div>
    <div class="name first-name">Sam</div>
    <div class="last-name">Xiao</div>
  </div>
</template>

<script>
export default {
  name: 'App',
}
</script>

<style scoped>
[class~="name"] {
  color: red;
}
</style>

第 3 行

<div class="name nickname">Oomusou</div>
<div class="name first-name">Sam</div>
<div class="last-name">Xiao</div>

Oomusou 與 Sam 擁有多個 class，但都有 name class。

16 行

[class~="name"] {
  color: red;
}

若要選擇 Oomusou 與 Sam，可選擇所有 tag 中 class attribute 包含 name class。

[attribute|=”value”]

以 |= 選擇 attribute 以指定 value 開頭，其後接著 hyphen -。

<template>
  <div>
    <div class="top-header">Oomusou</div>
    <div class="top-text">Sam</div>
    <div>Xiao</div>
  </div>
</template>

<script>
export default {
  name: 'App',
}
</script>

<style scoped>
[class|="top"] {
  color: red;
}
</style>

第 3 行

<div class="top-header">Oomusou</div>
<div class="top-text">Sam</div>
<div>Xiao</div>

Oomusou 與 Sam 的 class 都以 top- 開頭。

16 行

[class|="top"] {
  color: red;
}

若要選擇 Oomusou 與 Sam，可選擇所有 tag 中 class attribute 以 top 開頭，且其後為 -。

[attribute^=”value”]

以 ^= 選擇 attribute 以指定 value 開頭。

<template>
  <div>
    <div class="topheader">Oomusou</div>
    <div class="toptext">Sam</div>
    <div>Xiao</div>
  </div>
</template>

<script>
export default {
  name: 'App',
}
</script>

<style scoped>
[class^="top"] {
  color: red;
}
</style>

第 3 行

<div class="topheader">Oomusou</div>
<div class="toptext">Sam</div>
<div>Xiao</div>

Oomusou 與 Sam 的 class 都以 top 開頭。

16 行

[class^="top"] {
  color: red;
}

若要選擇 Oomusou 與 Sam，可選擇所有 tag 中 class attribute 以 top 開頭。

|= vs. ^=

  • 以指定 value 開頭，接著為 -
  • 以指定 value 開頭即可

[attribute$=”value”]

以 $= 選擇 attribute 以指定 value 結尾。

<template>
  <div>
    <div class="nickname">Oomusou</div>
    <div class="first-name">Sam</div>
    <div>Xiao</div>
  </div>
</template>

<script>
export default {
  name: 'App',
}
</script>

<style scoped>
[class$="name"] {
  color: red;
}
</style>

第 3 行

<div class="nickname">Oomusou</div>
<div class="first-name">Sam</div>
<div>Xiao</div>

Oomusou 與 Sam 的 class 都以 name 結尾。

16 行

[class$="name"] {
  color: red;
}

若要選擇 Oomusou 與 Sam，可選擇所有 tag 中 class attribute 以 name 結尾。

[attribute*=”value”]

以 *= 選擇 attribute 包含部分 value 即可。

<template>
  <div>
    <div class="nickname">Oomusou</div>
    <div class="first-name">Sam</div>
    <div class="name">Xiao</div>
  </div>
</template>

<script>
export default {
  name: 'App',
}
</script>

<style scoped>
[class*="name"] {
  color: red;
}
</style>

第 3 行

<div class="nickname">Oomusou</div>
<div class="first-name">Sam</div>
<div class="name">Xiao</div>

Oomusou、Sam 與 Xiao 的 class 中，都包含 name 字串。

16 行

[class*="name"] {
  color: red;
}

若要選擇 Oomusou 、 Sam 與 Xiao，可選擇所有 tag 中 class attribute 包含 name 字串。

Conclusion

  • Attribute selector 若用在 等於，其等效於 class selector，實務上也會使用 class selector
  • Attribute selector 主要用在 不等於，可搭配各種 operator 判斷
