如何垂直置中 ?

2021-03-03

垂直置中為實務上常見需求，CSS 可由多種方式實現。

Version

CSS 3

flex-grow: 1

CSS 垂直置中於父層 box。

<template>
  <div class="box">
    <div class="empty"/>
    <div>CSS</div>
    <div class="empty"/>
  </div>
</template>

<style scoped>
.box {
  display: flex;
  flex-direction: column;
  height: 100px;
}

.empty {
  flex-grow: 1;
}
</style>

第 2 行

<div class="box">
  <div class="empty"/>
  <div>CSS</div>
  <div class="empty"/>
</div>
  • 外層 <div> 使用 box class
  • 兩側佔位部分使用 empty class

10 行

.box {
  display: flex;
  flex-direction: column;
  height: 100px;
}

設定父層 box 的 style：

  • display: flex：子層 item 使用 Flexbox 排列
  • flex-direction: column：設定 main axis 為 column
  • height: 100px：若要設定垂直置中，則要設定 height

16 行

.empty {
  flex-grow: 1;
}
  • flex-grow: 1：表示空白部分剩餘 height 將由此 <div> 平分，因此看起來為垂直置中

justify-content: space-between

CSS 垂直置中於父層 box。

<template>
  <div class="box">
    <div/>
    <div>CSS</div>
    <div/>
  </div>
</template>

<style scoped>
.box {
  display: flex;
  flex-direction: column;
  justify-content: space-between;
  height: 100px;
}
</style>

10 行

.box {
  display: flex;
  flex-direction: column;
  justify-content: space-between;
  height: 100px;
}

與 flex-grow 一樣使用三個 <div>，兩旁為 dummy div。

  • display: flex：子層 item 使用 Flexbox 排列
  • flex-direction: column：設定 main axis 為 column
  • justify-content: space-between：原本為將剩餘高度自動平分給 item 間剩餘空間，因為兩側 dummy div 沒有高度，平分剩餘空間後 CSS 看起來像垂直置中
  • height: 100px：若要垂直置中，則要設定 height

justify-content: center

CSS 垂直置中於父層 box。

<template>
  <div class="box">
    CSS
  </div>
</template>

<style scoped>
.box {
  display: flex;
  flex-direction: column;
  justify-content: center;
  height: 100px;
}
</style>

第 8 行

.box {
  display: flex;
  flex-direction: column;
  justify-content: center;
  height: 100px;
}

設定父層 box 的 style：

  • display: flex：子層 item 使用 Flexbox 排列
  • flex-direction: column：設定 main axis 為 column，也因為 main axis 為 column，本來 Flexbox 會使 <div> 寬度由 content 決定，高度撐滿 100px，現在改成高度由 content 決定，寬度撐滿一整列
  • justify-content: center：直接將 <div> 置中於 main axis，因為目前 main axis 為 column，相當於垂直置中
  • height: 100px：若要垂直置中，則要設定 height

justify-content: space-around

CSS 垂直置中於父層 box。

<template>
  <div class="box">
    CSS
  </div>
</template>

<style scoped>
.box {
  display: flex;
  flex-direction: column;
  justify-content: space-around;
  height: 100px;
}
</style>

第 8 行

.box {
  display: flex;
  flex-direction: column;
  justify-content: space-around;
  height: 100px;
}

設定父層 box 的 style：

  • display: flex：子層 item 使用 Flexbox 排列
  • flex-direction: column：設定 main axis 為 column，也因為 main axis 為 column，本來 Flexbox 會使 <div> 寬度由 content 決定，高度撐滿 100px，現在改成高度由 content 決定，寬度撐滿一整列
  • justify-content: space-around：原本與 space-between 一樣將剩餘高度自動平分給 item 間剩餘空間，但不同的是 space-around 會考慮 item 的前後空間，因此平分後看起來 CSS 像垂直置中

justify-content: space-evenly

CSS 垂直置中於父層 box。

<template>
  <div class="box">
    CSS
  </div>
</template>

<style scoped>
.box {
  display: flex;
  flex-direction: column;
  justify-content: space-evenly;
  height: 100px;
}
</style>

第 8 行

.box {
  display: flex;
  flex-direction: column;
  justify-content: space-evenly;
  height: 100px;
}

設定父層 box 的 style：

  • display: flex：子層 item 使用 Flexbox 排列
  • justify-content: space-evenly：原本與 space-between 一樣將剩餘高度自動平分給 item 間剩餘空間，但不同的是 space-around 會考慮 item 的前後空間，因此平分後 CSS 看起來像垂直置中

space-around 與 space-evenly 差異是儘管包含 item 前後空間，但 space-evenly 是真正 evenly 等高，但 space-around item 間空間會是前後高度的兩倍

align-items: center

CSS 垂直置中於父層 box。

<template>
  <div class="box">
    CSS
  </div>
</template>

<style scoped>
.box {
  display: flex;
  align-items: center;
  height: 100px;
}
</style>

第 8 行

.box {
  display: flex;
  align-items: center;
  height: 100px;
}

設定父層 box 的 style：

  • display: flex：子層 item 使用 Flexbox 排列
  • align-items: center：由於沒更改 flex-direction，因此目前 main axis 仍是 row，可使用 align-items: center 直接置中於 cross axis，相當於垂直置中
  • height: 100px：若要垂直置中，則要設定 height

align-content: center

CSS 垂直置中於父層 box。

<template>
  <div class="box">
    CSS
  </div>
</template>

<style scoped>
.box {
  display: flex;
  flex-wrap: wrap;
  align-content: center;
  height: 100px;
}
</style>

第 8 行

.box {
  display: flex;
  flex-wrap: wrap;
  align-content: center;
  height: 100px;
}

設定父層 box 的 style：

  • display: flex：子層 item 使用 Flexbox 排列
  • flex-wrap：啟動換列機制，會抑制 align-items: stretch 預設 stretch 整個父層 box 高度，僅收縮成與 content 同高，因此 flex line 也收縮不再與父層 box 同高
  • align-content: center：因為 flex line 不再與父層 box 同高，因此 align-content: center才有機會以 flex line 對父層 box 垂直置中
  • height: 100px：若要垂直置中，則要設定 height

Conclusion

  • 只要記住 justify-content 系列是根據 Flexbox 的 main axis；而 align-items 系列是根據與 cross axis
  • align-content: center 也可以垂直置中，但必須搭配 flex-wrap: wrap
  • 若以寫法論，使用 align-items: center 最精簡
