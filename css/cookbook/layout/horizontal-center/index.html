如何水平置中 ?

2020-12-22

水平置中為實務上常見需求，CSS 可由多種方式實現。

Version

CSS 3

text-align: center

CSS 水平置中於 browser。

<template>
  <div class="item">CSS</div>
</template>

<style scoped>
.item {
  text-align: center;
}
</style>

第 6 行

.item {
  text-align: center;
}
  • text-align: center：<div> 為 block element 會佔據一整行，直接使用 text-align: center 讓 content 水平置中於 <div>

margin: auto

<template>
  <div class="item">CSS</div>
</template>

<style scoped>
.item {
  width: 40px;
  margin: auto;
  text-align: center;
}
</style>
  • width: 40px：width 必須比 content 內容更寬
  • margin-left: auto、margin-right: auto：自動調整 <div> 左右 margin，因此 <div> 可水平置中
  • margin: auto：相當於同時設定 margin-left: auto、margin-right: auto
  • text-align: center：margin-left: auto 與 margin-right: auto 只能使 <div> 水平置中，因為 <div> 為 block 有設定 width 比 content 略寬，因此預設 content 會水平靠左，必須靠 text-align: center 才能在 <div> 水平置中

當使用 margin: auto 水平置中時，只能用於 block 且有設定 width，而不能用於 inline 與 inline-block

flex-grow: 1

<template>
  <div class="box">
    <div class="empty"/>
    <div>CSS</div>
    <div class="empty"/>
  </div>
</template>

<style scoped>
.box {
  display: flex
}

.box .empty {
  flex-grow: 1
}
</style>

第 2 行

<div class="box">
  <div class="empty"/>
  <div>CSS</div>
  <div class="empty"/>
</div>
  • 外層 <div> 使用 box class
  • 兩側佔位部分使用 empty class

10 行

.box {
  display: flex
}
  • display: flex：使用 Flexbox

14 行

.box .empty {
  flex-grow: 1
}
  • flex-grow: 1：表示空白部分剩餘 width 將由此 <div> 平分，因此看起來為水平置中

justify-content: center

<template>
  <div class="box">
    <div>CSS</div>
  </div>
</template>

<style scoped>
.box {
  display: flex;
  justify-content: center;
}
</style>

第 7 行

.box {
  display: flex;
  justify-content: center;
}
  • display: flex：使用 Flexbox
  • justify-content: center：水平置中

Conclusion

  • 在單純的情況下，在 <div> 使用 text-align: center 就可水平置中
  • margin：auto 只需一個 <div> 即可，justify-content: center 則必須再包一層 <div>
  • 若不知 justify-content: center 也能由 flex-grow: 1 兜出來
  • 實務上建議使用 justify-content: center，因為 Flexbox 最為萬用
