如何垂直靠上 ?

2020-12-29

垂直靠上為實務上常見需求，CSS 可由多種方式實現。

Version

CSS 3

flex-grow: 1

CSS 垂直靠上於 <div>。

<template>
  <div class="box">
    <div>CSS</div>
    <div class="empty"/>
  </div>
</template>

<style scoped>
.box {
  display: flex;
  flex-direction: column;
  height: 100px;
}

.box .empty {
  flex-grow: 1;
}
</style>

第 2 行

<div class="box">
  <div>CSS</div>
  <div class="empty"/>
</div>
  • 外層 <div> 使用 box class
  • 下方佔位部分使用 empty class

第 9 行

.box {
  display: flex;
  flex-direction: column;
  height: 100px;
}
  • display: flex：外層 <div> 使用 Flexbox
  • flex-direction: column：設定 main axis 為 column
  • height: 100px：若要設定垂直靠上，則要設定 height

justify-content: flex-start

CSS 垂直靠上於 <div>。

<template>
  <div class="box">
    <div>CSS</div>
  </div>
</template>

<style scoped>
.box {
  display: flex;
  flex-direction: column;
  justify-content: flex-start;
  height: 100px;
}
</style>

第 8 行

.box {
  display: flex;
  flex-direction: column;
  justify-content: flex-start;
  height: 100px;
}
  • display: flex：外層 <div> 使用 Flexbox
  • flex-direction: column：設定 main axis 為 column，也因為 main axis 為 column，本來 Flexbox 會使 <div> 寬度由 content 決定，高度撐滿 100px，現在改成高度由 content 決定，寬度撐滿一整列
  • justify-content: flex-start：直接將 <div> 靠上於 main axis，因為目前 main axis 為 column，相當於垂直靠上
  • height: 100px：若要垂直靠上，則要設定 height

justify-content: flex-end

CSS 垂直靠上於 <div>。

<template>
  <div class="box">
    <div>CSS</div>
  </div>
</template>

<style scoped>
.box {
  display: flex;
  flex-direction: column-reverse;
  justify-content: flex-end;
  height: 100px;
}
</style>

第 8 行

.box {
  display: flex;
  flex-direction: column-reverse;
  justify-content: flex-end;
  height: 100px;
}
  • display: flex：外層 <div> 使用 Flexbox
  • flex-direction: column-reverse ： main axis 為 column，但改從下往上排列
  • justify-content: flex-end： 因為由下往上排列，flex-end 相當於垂直靠上

實務上不會使用這種方式，只是展示亦可使用 justify-content: flex-end 達成垂直靠上

align-items: start

CSS 垂直置中於 <div>，但 <div> 寬度只由 content 決定，而非佔據一整列。

<template>
  <div class="box">
    <div>CSS</div>
  </div>
</template>

<style scoped>
.box {
  display: flex;
  align-items: start;
  height: 100px;
}
</style>

第 8 行

.box {
  display: flex;
  align-items: start;
  height: 100px;
}
  • display: flex：外層 <div> 使用 Flexbox
  • align-items: start：由於沒更改 flex-direction，因此目前 main axis 仍是 column，可使用 align-items: start 直接靠上於 cross axis，相當於垂直靠上
  • height: 100px：若要垂直靠上，則要設定 height

Conclusion

  • 只要記住 justify-content 系列是根據 Flexbox 的 main axis；而 align-items 系列是根據與 cross axis
  • 若以寫法論，使用 align-items: start 直接根據 cross axis 垂直靠上最精簡
