どういう意味 (什麼意思？)

2025-09-07

疑問詞 どういう意味 可詢問 什麼意思？，用於 無法理解對方的話時，或有聽懂對方說的話，但是對其內容 感到不悅。

例文

今更無理だんて、どういう意味ですか。

現在才說沒辦法，是什麼意思呢？

  • 今更(いまさら) [0]：副 現在才
  • 無理(むり) [0]：な形容 辦不到的
  • なんて [0]：副 竟然
  • どういう意味(いみ) [0]：疑 什麼意思
  • です [0]：助動 是
    • ですか [0]：疑問 是嗎？

全部嘘だったって、どういう意味？

你說全部都是謊話是什麼意思？

  • 全部(ぜんぶ) [1]：副 全部
  • 嘘(うそ) [1]：名 謊話
  • だ [0]：助動 是
    • だった [1]：た形 是
  • という [0]：動I 自他 辞書形 說
    • といって [0]：て形 說
    • といっている [0]：維持狀態 說著
    • といっていた [0]：た形 說了
    • って [0]：縮約 說了
