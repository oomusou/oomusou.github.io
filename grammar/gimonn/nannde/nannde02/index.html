なんで かというと (要說為什麼)

2025-06-05

なんで 普通形 かというと 可表示 要說為什麼，其後常接 からだ 或 ためだ。

接続

  • なんで + 普 + かというと
  • なんで + Vる + かというと
  • なんで + いA + かというと
  • なんで + なA + なの + かというと
  • なんで + N + なの + かというと

動詞

なんで日本語を勉強するかというと、アニメが好きだから。

要說我為什麼學日語，因為我喜歡動畫。

  • なんで [1]：疑 為什麼
  • 日本語(にほんご) [0]：名 日語
  • を：格助 接受 勉強する 的 賓語
  • 勉強(べんきょう) [0]：名 唸書
    • 勉強する [0]：動III 他動 辞書形 唸書
    • 勉強するかというと [0]：要說為什麼 要說為什麼學日語
  • アニメ [1]：名 animation、動畫
  • が：格助 表 內容
  • 好(す)き [2]：な形容 喜歡
    • 好きだから [2]：原因 因為喜歡
