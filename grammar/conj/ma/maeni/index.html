まえに (之前)

2024-03-20

接続詞 まえに 可表示 動詞 發生在 動詞辞書形 / 名詞 之前。要注意句子的時態不論是 過去還是 非過去，動詞 都使用 動詞辞書形。

接続

  • Vる + まえ + に
  • N + の + まえ + に
  • 數量詞 + まえ + に

動詞

日本へ来るまえに日本語を勉強した。

來日本之前先學了日語。

  • 日本(にほん) [2]： 名 日本
  • へ：格助 表 移動方向
  • 来る(くる) [1]：動III 自動 來
  • まえに [1]：接 之前
  • 日本語(にほんご) [0]：名 日語
  • を：格助 接受 勉強する 的 賓語
  • 勉強(べんきょう)する [0]：動III 自他 辞書形 唸書
    • 勉強した：た形 唸書
