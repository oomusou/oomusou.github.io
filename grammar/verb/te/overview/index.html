動詞て形變化

2025-01-20

本文整理 動詞て形 變化規則。原本為 Vます ます て，但因為有些字 不易發音，因而產生 音便。

動詞 I

い音便

き -> いて

  • 書きます -> 書いて

書きて 太難唸，因此音便成 書いて

ぎ -> いで

  • 急ぎます -> 急いで

急ぎて 太難唸，因此音便成 急いで

促音便

いちり -> って

  • 会います -> 会って

会いて 太難唸，因此音便成 会って

  • 立ちます -> 立って

立ちて 太難唸，因此音便成 立って

  • 帰ります -> 帰って

帰りて 太難唸，因此音便成 帰って

撥音便

みにび -> んで

  • 飲みます -> 飲んで

飲みて 太難唸，因此音便成 飲んで

  • 死にます -> 死んで

死にて 太難唸，因此音便成 死んで

  • 遊びます -> 遊んで

遊びて 太難唸，因此音便成 遊んで

無音便

し -> して

  • 貸します -> 貸して
  • 返します -> 返して
  • 無くします -> 無くして
  • 話します -> 話して

例外

  • 行きます -> 行って

行いて 太難唸，因此音便成 行って

動詞 II

え段 -> て

  • 開けます -> 開けて
  • 見せます -> 見せて
  • 止めます -> 止めて
  • 食べます -> 食べて
  • 出ます -> 出て
  • できます -> できて
  • 起きます -> 起きて
  • 借ります -> 借りて

動詞 III

  • 来(き)ます -> 来(き)て
  • します -> して
  • 勉強します -> 勉強して
