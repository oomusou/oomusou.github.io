動詞て形のアクセント

2022-02-17

動詞て形 的 重音 可藉由規則從 辭書形 加以推導。

平板形動詞

辭書形 0 為 平板形動詞，て形 也是 0。

  • 行く [0] -> 行って [0]
  • 遊ぶ [0] -> 遊んで [0]
  • 貸す [0] -> 貸して [0]
  • 借りる [0] -> 借りて [0]
  • する [0] -> して [0]
  • 勉強する [0] -> 勉強して [0]

起伏形動詞

辭書形 非 0 為 起伏形動詞，表示第 n 個假名後要唸成低音。

1 號音

辭書形 為 1，則 て形 也是 1。

  • 書く [1] -> 買いて [1]
  • 会う [1] -> 会って [1]
  • 立つ [1] -> 立って [1]
  • 帰る [1] -> 帰って [1]
  • 飲む [1] -> 飲んで [1]
  • 来(く)る [1] -> 来(き)て [1]

非 1 號音

辭書形 為 1 以外，則 重音 在 て 往前數第 3 個假名 (包含 て)。

  • 食べる [2] -> 食べて [1]
  • 起きる [2] -> 起きて [1]
  • 覚える [3] -> 覚えて [2]
  • 降りる [2] -> 降りて [1]
  • 掛ける [2] -> 掛けて [1]
  • 見せる [2] -> 見せて [1]
