てある (維持狀態)

2024-08-31

動詞て形 ある 可表示 有人 為了某種目的而進行的某一行為，而其結果依然 維持 該 狀態，使用的動詞為 他動詞。

接続

  • N1 + に + N2 + が + Vて + ある
  • N2 + は + N1 + に + Vて + ある

桌子上放著留言。

  • に：格助 表 存在位置
  • メモ [1]：名 memo、筆記
  • が：格助 表 主語
