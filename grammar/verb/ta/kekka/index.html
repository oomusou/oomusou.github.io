結果 (結果)

2026-07-15

動詞た形 結果 / 名詞 の 結果 可表示經過一連串 行動 或 過程 後所得到的 結果。翻譯成 的結果。

難度

  • N3

情境

結果 用於敘述經過某個 動作 或 過程 後所產生的 客觀結果，前後常帶有 因果關係，語氣較為 正式，多用於 書面語 或 正式場合。

接続

  • Vた + 結果
  • N + の + 結果

動詞

彼が努力した結果、合格した。

他努力的結果，及格了。

  • 彼(かれ) [1]：代名詞 他
  • が：格助 表 主語
  • 努力(どりょく) [1]：名 努力
    • 努力する [1]：動III 自動 辞書形 努力
    • 努力した [1]：た形 努力了
    • 努力した結果(けっか) [1]：名 結果 努力了結果
  • 合格(ごうかく) [0]：名 及格
    • 合格する [0]：動III 自動 辞書形 及格
    • 合格した [2]：た形 及格了

名詞

戦争の結果、多くの人が亡くなった。

戰爭的結果，很多人去世了。

  • 戦争(せんそう) [0]：名 戰爭
    • 戦争の結果(けっか) [0]：戰爭的結果
  • 多(おお)く [1]：名 多數
  • の：格助 表 修飾
  • 人(ひと) [0]：名 人
  • が：格助 表 主語
  • 亡(な)くなる [3]：動I 自動 去世
    • 亡くなった [3]：た形 去世了
