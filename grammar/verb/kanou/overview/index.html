動詞可能形變化

2022-06-09

本文整理 動詞可能形 變化規則。

動詞 I

い 段 -> え 段 + る

  • 歌います -> 歌える
  • 弾きます -> 弾ける
  • 泳ぎます -> 泳げる
  • 直します -> 直せる
  • 持ちます -> 持てる
  • 遊びます -> 遊べる
  • 読みます -> 読める
  • 走ります -> 走れる

動詞 II

ます -> られる

  • 調べます -> 調べられる
  • 覚えます -> 覚えられる
  • 降ります -> 降りられる

動詞 III

  • 来ます -> 来(こ)られる
  • します -> できる
  • 勉強します -> 勉強できる

當動詞轉成 可能形 後都變成 動詞 II，其 普通形 可直接從 禮貌形 ます る 完成

例外

  • 分かる 本身已經有 可能 的意思，所以不用 分かれる
