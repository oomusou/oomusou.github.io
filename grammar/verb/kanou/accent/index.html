動詞可能形のアクセント

2022-06-09

動詞可能形 的 重音 可藉由規則從 辭書形 加以推導。

平板形動詞

辭書形 0 為 平板形動詞，可能形 也是 0。

  • 行く [0] -> 行ける [0]
  • 遊ぶ [0] -> 遊べる [0]
  • 吸う [0] -> 吸える [0]
  • 買う [0] -> 買える [0]
  • 入れる [0] -> 入れられる [0]
  • 浴びる [0] -> 浴びられる [0]
  • 借りる [0] -> 借りられる [0]
  • する [0] -> できる [0]
  • 呼ぶ [0] -> 呼べる [0]
  • 勉強する [0] -> 勉強できる [0]

起伏形動詞

辭書形 非 0 為 起伏形動詞，表示第 n 個假名後要唸成 低音，可能形 則是在倒數第二個為 重音。

  • 書く [1] -> 書ける [2]
  • 話す [2] -> 話せる [3]
  • 待つ [1] -> 待てる [2]
  • 読む [1] -> 読める [2]
  • 飲む [1] -> 飲める [2]
  • 帰る [1] -> 帰られる [4]
  • 取る [1] -> 取れる [2]
  • 急ぐ [2] -> 急げる [3]
  • 会う [1] -> 会える [2]
  • 食べる [2] -> 食べられる [4]
  • 見る [1] -> 見られる [3]
  • 起きる [2] -> 起きられる [4]
  • 降る [1] -> 降れる [2]
  • 来(く)る [1] -> 来(こ)られる [3]

可能形 變化後的動詞為 動II
