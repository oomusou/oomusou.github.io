ようが まいが (不管)

2026-05-04

動詞意向形 が 動詞 まいが 可表示 不管，表達 無論做或不做，結果都是 相同 的 堅定強烈情緒。。翻譯成 不管 要不要、不管 是不是。

接続

  • Vよう + が + V + まい + が
  • Vよう + が + Vる + まい + が (動I/動II/動III)
  • Vよう + が + Vない ない + まい + が (動II/動III)
  • Vよう + が + する -> するまい / すまい / しまい + が
  • Vよう + が + 来る -> 来(く)るまい / 来(こ)まい + が

例文

彼が来ようが来まいが、会議は始める。

不管他要來不來，會議都會開始。

  • 彼(かれ) [1]：代 他
  • が：格助 表 主語
  • 来(く)る [1]：動III 自動 辞書形 來
    • 来(こ)よう [2]：意向形 要來
    • 来ようが [2]：不管 不管要來
  • 来(く)る [1]：動III 自動 辞書形 來
    • 来(こ)まい [2]：否定意志 不來
    • 来まいが [2]：不管 不管不來
  • 会議(かいぎ) [1]：名 會議
  • は：副助 表 主題
  • 始(はじ)める [0]：動II 他動 開始
