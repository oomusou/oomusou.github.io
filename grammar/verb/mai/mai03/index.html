ようか まいか (猶豫)

2026-05-04

動詞意向形 か 動詞 まいか 可表示 猶豫，用來描述 陷入猶豫、兩難 的心境。後面常搭配 考える、迷う、悩む … 等 動詞。翻譯成 要做 還是不做？、要不要？、是否？。

接続

  • Vよう + か + V + まい + か
  • Vよう + か + Vる + まい + か (動I/動II/動III)
  • Vよう + か + Vない ない + まい + か (動II/動III)
  • Vよう + か + する -> するまい / すまい / しまい + か
  • Vよう + か + 来る -> 来(く)るまい / 来(こ)まい + か

考える

彼に話そうか話すまいか考えている。

我正在考慮要不要跟他說。

  • 彼(かれ) [1]：代 他
  • に：格助 表 動作對象
  • 話(はな)す [2]：動I 他動 辞書形 說
    • 話そう [3]：意向形 要說
    • 話そうか [0]：猶豫 要說嗎
  • 話(はな)す [2]：動I 他動 辞書形 說
    • 話すまい [4]：否定意志 不要說
    • 話すまいか [4]：猶豫 不要說嗎？
  • 考(かんが)える [4]：動II 他動 辞書形 考慮
    • 考えて [3]：て形 考慮
    • 考えている [0]：現在進行 考慮中

迷う

仕事をやめようかやめるまいか、迷ってる。

我正在猶豫要不要辭掉工作。

  • 仕事(しごと) [0]：名 工作
  • を：格助 接受 やめる 的 賓語
  • やめる [0]：動II 他動 辞書形 辭職
    • やめよう [3]：意向形 要辭職
    • やめようか [3]：猶豫 要辭職嗎？
  • やめる [0]：動II 他動 辞書形 辭職
    • やめるまい [4]：否定意志 不要辭職
    • やめるまいか [4]：猶豫 不要辭職嗎？
  • 迷(まよ)う [2]：動I 自動 辞書形 猶豫
    • 迷って [2]：て形 猶豫
    • 迷っている [2]：現在進行 猶豫中

悩む

本当のことを話そうか話すまいか、一晩中悩んだ。

我整晚都在煩惱要不要說出真相。

  • の：格助 表 修飾
  • こと [2]：名 事情
  • を：格助 接受 xx 的 賓語
  • 話(はな)す [2]：動I 他動 辞書形 說
    • 話そう [3]：意向形 要說
    • 話そうか [3]：猶豫 要說嗎？
