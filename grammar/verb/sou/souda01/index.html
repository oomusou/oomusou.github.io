そうだ (傳聞)

2024-11-17

普通形 そうだ 可表示 不參雜 任何自己的意見，就把從他處得到的訊息 傳達 給聽者的 傳聞。可在句首加上 によると、によねば、では 顯示訊息的 出處 ，可省略 だ。相當於 ということだ、とのことだ。翻譯成 聽說。

接続

  • 普 + そうだ
  • Vる + そうだ
  • いA + そうだ
  • なA + だ + そうだ
  • N + だ + そうだ
