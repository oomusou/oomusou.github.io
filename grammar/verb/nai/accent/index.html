動詞ない形のアクセント

2022-02-17

動詞ない形 的 重音 可藉由規則從 辭書形 加以推導。

平板形動詞

辭書形 0 為 平板形動詞，ない形 也是 0。

  • 行く [0] -> 行かない [0]
  • 遊ぶ [0] -> 遊ばない [0]
  • 吸う [0] -> 吸わない [0]
  • 買う [0] -> 買わない [0]
  • 入れる [0] -> 入れない [0]
  • 浴びる [0] -> 浴びない [0]
  • 借りる [0] -> 借りない [0]
  • する [0] -> しない [0]
  • 呼ぶ [0] -> 呼ばない [0]

起伏形動詞

辭書形 非 0 為 起伏形動詞，表示第 n 個假名後要唸成低音，ない形則是在 な 的前一個假名為 重音。

  • 書く [1] -> 書かない [2]
  • 話す [2] -> 話さない [3]
  • 待つ [1] -> 待たない [2]
  • 読む [1] -> 読まない [2]
  • 飲む [1] -> 飲まない [2]
  • 帰る [1] -> 帰らない [3]
  • 取る [1] -> 取らない [2]
  • 急ぐ [2] -> 急がない [3]
  • 会う [1] -> 会わない [2]
  • 食べる [2] -> 食べない [2]
  • 見る [1] -> 見ない [2]
  • 起きる [2] -> 起きない [2]
  • 降る [1] -> 降りない [2]
  • 来(く)る [1] -> 来(こ)ない [1]
  • 勉強する [0] -> 勉強しない [0]
