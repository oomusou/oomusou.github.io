動詞ない形變化

2022-02-17

本文整理 動詞ない形 變化規則。

動詞 I

い段 -> あ段 + ない

  • 行きます -> 行かない
  • 返します -> 返さない
  • 待ちます -> 待たない
  • 飲みます -> 飲まない
  • 帰ります -> 帰らない
  • 取ります -> 取らない
  • 急ぎます -> 急がない
  • 呼びます -> 呼ばない
  • 立ちます -> 立たない

い 段 -> わない

  • 会います -> 会わない
  • 買います -> 買わない
  • 手伝います -> 手伝わない
  • 吸います -> 吸わない
  • 使います -> 使わない
  • 払います -> 払わない

動詞 II

え 段 -> ない

  • 教えます -> 教えない
  • 開けます -> 開けない
  • 出かけます -> 出かけない
  • 見せます -> 見せない
  • 止めます -> 止めない
  • 入れます -> 入れない
  • 食べます -> 食べない
  • います -> いない
  • 見ます -> 見ない
  • 出ます -> 出ない
  • できます -> できない
  • 起きます -> 起きない
  • 借ります -> 借りない
  • 降ります -> 降りない
  • 浴びます -> 浴びない

動詞 III

  • 来(き)ます -> 来(こ)ない
  • します -> しない
  • 勉強します -> 勉強しない
