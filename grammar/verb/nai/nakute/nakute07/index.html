なくても結構 (可以不)

2024-08-08

普通形ない形 ない なくても結構 可表示 可以不，做的話 比較好，但不做也 沒關係 的語感。

接続

  • 普 ない + なくても結構
  • Vない ない + なくても結構
  • いA い + く + なくても結構
  • なA + で / じゃ + なくても結構
  • N + で / じゃ + なくても結構

動詞

名前を書かなくても結構。

可以不寫名字。

寫或不寫都可以，寫的話最好，不寫也沒關係

  • 名前(なまえ) [0]：名 名字
  • を：格助 接受 書く 的 賓語
  • 書(か)く[1]：動I 他動 辞書形 寫
    • 書かない [2]：ない形 不寫
    • 書かなくても結構 [0]：可以不 可以不寫

い形容詞

美味しくなくても結構。

可以不好吃的。

  • 美味(おい)しい [3]：い形容 好吃的
    • 美味しくなくても結構 [4]：可以不 可以不好吃的

な形容詞

漢字が上手じゃなくても結構。

可以不擅長寫漢字。

  • 漢字(かんじ) [0]：名 漢字
  • が：格助 表 主語
  • 上手(じょうず) [3]：な形容 拿手的
    • 上手じゃなくても結構 [5]：可以不 可以不拿手的

名詞

レポートは日本語じゃなくても結構。

報告可以不用日語寫。

  • レポート [2]：名 report、報告
  • は：副助 表 主題
  • 日本語(にほんご) [0]：名 日語
    • 日本語じゃなくても結構 [0]：可以不 可以不用日語
