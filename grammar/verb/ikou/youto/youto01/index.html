ようと (即使)

2025-11-16

動詞意向形 と (も) 可表示 即使，為 ても 的 書面 用語。

接続

  • Vよう + と (も)
  • いA い + かろう + と (も)
  • なA + だろう / であろう + と (も)
  • N + だろう / であろう + と (も)

動詞

残念な結果になろうとも、受け止めます。

即使是遺憾的結果，我也會接受。

  • 残念(ざんねん) [3]：な形容 遺憾的
  • な：格助 以 な形容詞 形容 名詞
  • 結果(けっか) [0]：名 結果
  • に：格助 表 變化結果
  • なる [1]：動I 自動 辞書形 變成
    • なろう [2]：意向形 變成
    • なろうとも [1]：即使 即使變成
  • 受(う)け止(と)める [4]：動II 他動 辞書形 接受
    • 受け止めます [5]：ます形 接受

い形容詞

値段が高かろうと、私は買います。

即使價錢貴，我也要買。

  • 値段(ねだん) [0]：名 價錢
  • が：格助 表 主語
  • 高(たか)い [2]：い形容 貴的
    • 高かろうと [4]：即使 即使貴的
  • 私(わたし) [0]：代 我
  • は：副助 表 主題
  • 買(か)う [0]：動I 他動 辞書形 買
    • 買います [3]：ます形 買

な形容詞

困難だろうと、必ず道は開けます。

即時很困難，道路一定會開的。

  • 困難(こんなん) [1]：な形容 困難的
    • 困難だろうと [6]：即使 即使困難的
  • 必(かなら)ず [0]：副 一定
  • 道(みち) [0]：名 道路
  • は：副助 表 主題
  • 開(あ)ける [0]：動II 他動 辞書形 開
    • 開けます [3]：ます形 開

名詞

金持ちだろうと、真の愛は買えません。

即使是有錢人，也不能買到真愛。

  • 金持(かねも)ち [3]：名 有錢人
    • 金持ちだろうと [6]：即使 即使有錢人
  • 真(しん) [0]：名 真實
  • の：格助 以 名詞 形容 名詞
  • 愛(あい) [1]：名 愛
  • は：副助 表 主題
  • 買(か)う [0]：動I 他動 辞書形 買
    • 買える [0]：可能形 能買
    • 買えます [3]：ます形 能買
    • 買えません [4]：否定 不能買
