動詞意向形變化

2022-08-21

本文整理 動詞意向形 變化規則。

動詞 I

い 段 -> お 段 + う

  • 買います -> 買おう
  • 歩きます -> 歩こう
  • 急ぎます -> 急ごう
  • 直します -> 直そう
  • 待ちます -> 待とう
  • 遊びます -> 遊ぼう
  • 休みます -> 休もう
  • 乗ります -> 乗ろう

動詞 II

え 段 -> よう

  • 換えます -> 換えよう
  • 覚えます -> 覚えよう
  • 見ます -> 見よう

動詞 III

  • 来(き)ます -> 来(こ)よう
  • します -> しよう
  • 勉強します -> 勉強しよう
