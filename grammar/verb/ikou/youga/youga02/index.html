ようが (無論)

2025-05-07

疑問詞 動詞意向形 が 可表示 無論，為 ても 的 書面 用語。

接続

  • W + Vよう + が
  • W + いA い + かろう + が
  • W + なA + だろう / であろう + が
  • W + N + だろう / であろう + が

動詞

何回失敗しようが、挑戦を恐れません。

無論失敗多少次，我都不怕挑戰。

  • 何回(なんかい) [1]：名 多少次
  • 失敗(しっぱい)する [0]：動III 自動 辞書形 失敗
    • 失敗しよう [6]：意向形 失敗
    • 失敗しようが [6]：無論 無論失敗
  • 挑戦(ちょうせん) [0]：名 挑戰
  • を：格助 接受 恐れる 的 賓語
  • 恐(おそ)れる [3]：動II 他動 辞書形 害怕
    • 恐れます [4]：ます形 害怕
    • 恐れません [5]：否定 不害怕

どんな結果になろうとが、受け止めます。

無論任何結果，我都會接受。

  • どんな [1]：連体 任何的
  • 結果(けっか) [0]：名 結果
  • に：格助 表 變化結果
  • なる [1]：動I 自動 辞書形 變成
    • なろう [2]：意向形 變成
    • なろうとが [2]：無論 無論變成
  • 受(う)け止(と)める [0]：動III 他動 辞書形 接受
    • 受け止めます [5]：ます形 接受

会社が何と言おうが、僕は諦めない。

無論公司怎麼說，我都不會放棄。

  • 会社(かいしゃ) [0]：名 公司
  • が：格助 表 主語
  • 何(なん) [1]：代 什麼
  • と：格助 表 內容
  • 言(い)う [0]：動I 他動 辞書形 說
    • 言おう [2]：意向形 說
    • 言おうが [2]：無論 無論說
  • 僕(ぼく) [1]：代 我
  • は：副助 表 主題
  • 諦(あきら)める [4]：動II 他動 辞書形 放棄
    • 諦めない [4]：ない形 不放棄

い形容詞

どんなに忙しかろうが、毎日運動します。

無論多麽地忙，每天都運動。

  • どんなに [1]：副 無論如何地
  • 忙(いそが)しい [4]：い形容 忙碌的
    • 忙しかろうが [6]：無論 無論忙碌的
  • 毎日(まいにち) [1]：名 每天
  • 運動(うんどう)する [0]：動III 自動 辞書形 運動
    • 運動します [6]：ます形 運動

な形容詞

いかに困難だろうが、必ず道は開けます。

無論如何困難，道路一定會開的。

  • いかに [2]：副 無論如何地
  • 困難(こんなん) [1]：な形容 困難的
    • 困難だろうが [6]：無論 無論困難的
  • 必(かなら)ず [0]：副 一定
  • 道(みち) [0]：名 道路
  • は：副助 表 主題
  • 開(あ)ける [0]：動II 他動 辞書形 開
    • 開けます [3]：ます形 開

名詞

どんな金持ちだろうが、真の愛は買えません。

無論是任何有錢人，都買不到真愛。

  • どんな [1]：連体 任何的
  • 金持(かねも)ち [3]：名 有錢人
    • 金持ちだろうが [6]：即使 即使有錢人
  • 真(しん) [0]：名 真實
  • の：格助 以 名詞 形容 名詞
  • 愛(あい) [1]：名 愛
  • は：副助 表 主題
  • 買(か)う [0]：動I 他動 辞書形 買
    • 買える [0]：可能形 能買
    • 買えます [3]：ます形 能買
    • 買えません [4]：否定 不能買
