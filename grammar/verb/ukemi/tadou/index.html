B は A に N を Vれる [他動] (B 被 A 做)

2023-02-09

可使用 B は A に N を 動詞受身形 表示 A 對 B 的所有物所做的某種行為。在大多數場合，這一行為都會讓 B 感到 困擾 和 麻煩。

接続

  • B + は + A + に + N + を + Vれる = A + が + B + の + N + を + Vる

例文

彼が私のパソコンを壊した。

他把我的電腦弄壞了。

主動 陳述 事實

  • 彼(かれ) [0]：代 他
  • が：格助 表 主語
  • 私(わたし) [0]：代 我
  • の：格助 表 所有
  • パソコン [0]：名 電腦
  • を：格助 接受 壊す 的 賓語
  • 壊(こわ)す [2]：動I 他動 辞書形 弄壞
    • 壊した [2]：た形 弄壞

私は彼にパソコンを壊された。

我被他他弄壞了電腦。

被動 強調 被害

  • 私(わたし) [0]：代 我
  • は：副助 表 主題
  • 彼(かれ) [0]：代 他
  • に：格助 表 對象
  • パソコン [0]：名 電腦
  • を：格助 接受 壊す 的 賓語
  • 壊(こわ)す [2]：動I 他動 辞書形 弄壞
    • 壊される [4]：受身形 被弄壞
    • 壊された [4]：た形 被弄壞

私のパソコンは彼に壊された。 (X)
作為 主題提示 的 不是所有物，而是對這一行為 感到困惑的人 (所有者)
