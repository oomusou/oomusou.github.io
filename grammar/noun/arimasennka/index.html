N は ありませんか (否定疑問句)

2022-09-20

は ありますか 與 は ありませんか 意思相同，但 は ありませんか 語氣更委婉。

接続

  • N + は + ありませんか

例文

A：2、3日に旅行をしようと思っているんですが。どこかいい所はありませんか。

B：そうですね。2、3日なら、箱根か日光がいいと思う。

A：我想去旅行兩三天，有什麼好地方嗎？

B：這個嘛。兩三天的話，箱根或日光不錯。
