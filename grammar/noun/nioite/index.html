において (於)

2026-07-15

在某個 場所、場面、領域、方面、時代、期間 等 進行 某件事或 存在 某種情況。為 に 或 で 的 書面語，漢字為 於く。翻譯成 於 或 在。

難度

  • N1

情境

用於 正式 場合或 書面語，表示某事發生的 場所、領域 或 時間點，語氣較 に、で 更為 生硬 且 鄭重。

接続

  • N + において
  • N + における + N

において

結婚式は、教会において行われます。

結婚儀式於教會舉辦。

において 的 語源 為 に 於いて，本意為 在 場所，衍生為 書面語 的 場所 用法

  • 結婚式(けっこんしき) [3]：名 結婚儀式
  • は：副助 表 主題
  • 教会(きょうかい) [0]：名 教會
    • 教会において [0]：書面 於教會
  • 行(おこな)う [0]：動I 他動 辞書形 舉行
    • 行われる [0]：受身形 被舉行
    • 行われます [5]：ます形 被舉行

における

車内における煙草のご利用はご遠慮ください。

在車子裡面請不要抽菸。

  • 車内(しゃない) [1]：名 車內
    • 車内における [1]：書面 在車內的
  • 煙草(たばこ) [0]：名 香菸
  • の：格助 表 修飾
  • ご利用(りよう) [0]：名 使用
  • は：副助 表 主題
  • 遠慮(えんりょ) [0]：名 謝絕
    • ご遠慮ください [0]：敬語 請不要
