つもりだ (自認為)

2025-07-20

つもりだ 可表示 自認為 。當主題為 第一人稱 時，其 自認為 與 事實無關；當主題為 二三人稱 時，其 自認為 與 事實不符。

接続

  • Vた + つもりだ
  • Vて + いるつもりだ
  • いA + つもりだ
  • なA + つもりだ
  • N + の + つもりだ

動詞
