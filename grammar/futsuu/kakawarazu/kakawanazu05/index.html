にかかわらず (無關)

2026-06-29

名詞 にかかわらず 可表示 無關，不管情況如何 結果都一樣。

難度

  • N2

情境

用於表示不把前項的 差異、條件 或 狀況 當成 判斷標準，後項的結果或規則都 一樣成立，常見於 公告、規定、說明 或　較正式 的表達。

接続

  • N + にかかわらず

例文

年齢にかかわらず、誰でも参加できます。

無關年齡，任何人都可以參加。

にかかわらず 的 語源 為 に + 関わる -> 関わらない -> 関わらず，本意是 不受前面條件所影響，也就是 無關

  • 年齢(ねんれい) [0]：名 年齡
    • 年齢にかかわらず [5]：無關 無關年齡
  • 誰でも [1]：代 任何人
  • 参加(さんか) [0]：名 參加
    • 参加する [0]：動III 自動 辞書形 參加
    • 参加できる [0]：可能形 可以參加
    • 参加できます [0]：ます形 可以參加
