といわれている (據說)

2025-07-20

普通形 といわれている 可表示 大家都這麼說。翻譯成 據說。

接続

  • 普 + といわれている
  • Vる + といわれている
  • いA + といわれている
  • なA + だ + といわれている
  • N + だ + といわれている

動詞

昔、天狗が住んでいたといわれている。

據說從前有天狗住在那裡。

  • 昔(むかし) [0]：名 以前
  • 天狗(てんぐ) [0]：名 天狗
  • 住(す)む [1]：動I 自動 辞書形 住
    • 住んで [1]：て形 住
    • 住んでいる [1]：狀態 住著
    • 住んでいた [1]：た形 住著
    • 住んでいたといわれている [1]：據說 據說住著
