し、し (特色列舉)

2025-06-14

丁寧形 / 普通形 し、丁寧形 / 普通形 し 可表示 兩個以上類似的事情的 特色列舉，只使用一個 し 亦可，表示還有 其他特色 未說完。

接続

  • 丁 / 普 + し
  • Vます / Vる / VIIIます ます / + し
  • いA + (です) + し
  • なA + です / だ + し
  • N + です / だ + し

動詞

鈴木さんはピアノも弾けるし、歌えるし、ダンスもできる。

鈴木先生既會彈鋼琴，又會唱歌，還會跳舞。

し 由於不只敘述一件事情，因此 主語 常搭配助詞 も

  • 鈴木(すずき) [0]：名 鈴木
  • さん [0]：名 先生
  • は：副助 表 主題
  • ピアノ [0]：名 piano、鋼琴
  • も：副助 表 也
  • 弾(ひ)く [0]：動I 他動 辞書形 彈琴
    • 弾ける [0]：可能形 會彈琴
    • 弾けるし [0]：特色列舉 會彈琴
  • 歌(うた)う [0]：動I 他動 辞書形 唱
    • 歌える [0]：可能形 會唱
    • 歌えるし [0]：特色列舉 會唱
  • ダンス [1]：名 dance、跳舞
  • も：副助 表 也
  • できる [2]：動II 自動 會

い形容詞

中森さんは若いし、体も大きいし、それに力も強い。

中森先生既年輕，身材又高大，而且力氣還很大。

  • 中森(なかもり) [0]：名 中森
  • さん [0]：名 先生
  • は：副助 表 主題
  • 若(わか)い [2]：い形容 年輕的
    • 若いし [2]：特色列舉 年輕的
  • 体(からだ) [0]：名 身體
  • も：副助 表 也
  • 大(おお)きい [3]：い形容 大的
    • 大きいし [3]：特色列舉 大的
  • それに [0]：接 而且
  • 力(ちから) [3]：名 力
  • も：副助 表 也
  • 強(つよ)い [2]：い形容 強壯的

な形容詞

田中さんは真面目だし、中国語も上手だし、それに経験もある。

田中先生很認真，中文也很好，而且還有經驗。

  • 田中(たなか) [0]：名 田中
  • さん [0]：名 先生
  • は：副助 表 主題
  • 真面目(まじめ) [0]：な形容 認真的
    • 真面目だし [0]：特色列舉 認真的
  • 中国語(ちゅうごくご) [0]：名 漢語
  • も：副助 表 也
  • 上手(じょうず) [3]：な形容 拿手的
    • 上手だし [3]：特色列舉 拿手的
  • それに [0]：接 而且
  • 経験(けいけん) [0]：名 經驗
  • も：副助 表 也
  • ある：動III 自動 有

名詞

今日は雨だし、外に出たくない。

今天下雨，不想出門。

  • 今日(きょう) [1]：名 今天
  • は：副助 表 主題
  • 雨(あめ) [1]：名 雨
    • 雨だし [1]：特色列舉 雨
  • 外(そと) [1]：名 外面
  • に：格助 表 目的地
  • 出(で)る [1]：動II 自動 辞書形 出門
    • 出ます [2]：ます形 出門
    • 出たい [2]：想做 想出門
    • 出たくない [4]： ない形 不想出門
