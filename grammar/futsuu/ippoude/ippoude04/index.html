一方だ (越來越)

2026-05-20

普通形 一方だ 可表示 越來越，可用於 好事 或 壞事，在 普通體 可省略 だ。

接続

  • Vる + 一方だ
  • いA い + くなる + 一方だ
  • なA + になる + 一方だ
  • N + の + 一方だ

動詞

最近、物価は上がる一方だ。

最近，物價越來越上漲。

  • 最近(さいきん) [0]：名 最近
  • 物価(ぶっか) [0]：名 物價
  • は：副助 表 主題
  • 上(あ)がる [0]：動I 自動 辞書形 上漲
    • 上がる一方(いっぽう)だ [6]：越來越 越來越上漲

い形容詞

景気は悪くなる一方だ。

景氣越來越差。

  • 景気(けいき) [0]：名 景氣
  • は：副助 表 主題
  • 悪(わる)い [2]：い形容 不好的
    • 悪くなる一方(いっぽう)だ [8]：越來越 越來越不好的

な形容詞

状況は複雑になる一方だ。

情況越來越複雜。

  • 状況(じょうきょう) [0]：名 狀況
  • は：副助 表 主題
  • 複雑(ふくざつ) [0]：な形容 複雜
    • 複雑になる一方(いっぽう)だ [10]：越來越 越來越複雜

名詞

都市の人口は増加の一方だ。

都市的人口越來越增加。

  • 都市(とし) [1]：名 都市
  • の：格助 表 所屬
  • 人口(じんこう) [0]：名 人口
  • は：副助 表 主題
  • 増加(ぞうか) [0]：名 增加
    • 増加の一方(いっぽう)だ [7]：越來越 越來越增近
