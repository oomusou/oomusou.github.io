べきだ (應該)

2025-08-04

普通形 べきだ 可表示 義務上的應該，或 建議上的應該，此時相當於 たほうがいい，但語氣較 強烈。翻譯成 應該。

接続

  • 普 + べきだ
  • Vる + べきだ
  • いA い + くある + べきだ
  • なA + である + べきだ
  • N + である + べきだ

動詞

毎朝、歯を磨くべきだ。

每天早上都應該刷牙。

  • 毎朝(まいあさ) [0]：名 每天早上
  • 歯(は)を磨(みが)く [0]：慣 刷牙
    • 歯を磨くべきだ [6]：應該 應該刷牙

試験の前に、しっかり勉強するべきだ。
試験の前に、しっかり勉強すべきだ。 (縮約)

考試前應該好好地唸書。

するべきた 可縮約成 すべきだ

  • 前(まえ) [1]：名 前
  • に：格助 表 發生時間

    • 勉強するべきだ [7]：應該 應該唸書
