べき (應該)

2025-08-04

普通形 べき 名詞 可表示 義務 上的 應該，或 建議上的應該，此時相當於 たほうがいい，但語氣上較 強烈。翻譯成 應該。

接続

  • 普 + べき + N
  • V普 + べき + N
  • いA い + くある + べき + N
  • なA + である + べき + N
  • N + である + べき + N

動詞

学校には守るべきルールがあるから、勝手な行動はできない。

因為學校裡有應該遵守的規則，所以不能任意行動。

  • 学校(がっこう) [0]：名 學校
  • に：格助 表 存在位置
  • は：副助 表 主題
  • 守(まも)る [2]：動I 他動 辞書形 遵守
    • 守るべき [1]：應該 應該遵守
    • 守るべきルール [1]：應該 應該遵守的規則
  • ルール [1]：名 rule、規則
  • が：格助 表 主語
  • ある [1]：動I 自動 存在
  • から [1]：接助 表 原因
  • 勝手(かって) [0]：な形容 隨便的
  • な：格助 以 な形容詞 形容 名詞
  • 行動(こうどう) [0]：名 行動
  • は：副助 表 對比
  • できる [2]：動II 自動 辞書形 能
    • できない [2]：ない形 不能

い形容詞

彼は若くあるべき姿勢を大切にしている。

他很重視年輕人應該有的態度。

  • 彼(かれ) [1]：代 他
  • は：副助 表 主題
  • 若(わか)い [2]：い形容 年輕的
    • 若くあるべき [2]：應該 應該年輕
    • 若くあるべき姿勢(しせい) [2]：應該 應該年輕的態度
  • 姿勢(しせい) [0]：名 態度
  • を：格助 接受 大切にする 的 賓語
  • 大切(たいせつ)にする [0]： 慣 重視
    • 大切にして [0]：て形 重視
    • 大切にしている [0]：狀態 重視著

な形容詞

私たちは公平であるべき社会を目指している。

我們正在以理應公平的社會為目標。

  • 私(わたし)たち [0]：代 我們
  • は：副助 表 主題
  • 公平(こうへい) [0]：な形容 公平的
    • 公平であるべき(かいしゃ) [0]：應該 應該公平
    • 公平であるべき会社(かいしゃ) [0]：應該 應該公平的社會
  • 社会(しゃかい) [0]：名 社會
  • を：格助 接受 目指す 的 賓語
  • 目指(めざ)す [2]：動I 他動 以 為目標
    • 目指して [2]：て形 以 為目標
    • 目指している [2]：現在進行 以 為目標
