まま (持續狀態)

2026-01-31

普通形 まま 可表示動作實行後的 持續狀態，詞性 轉為 な形容詞 或 副詞。翻譯成 就這樣。

接続

  • Vた + まま
  • Vない + まま
  • いA + まま
  • なA + な + まま
  • N + の + まま

動詞

彼女はいつも電気をつけたまま寝ています。

她總是把燈開著就這樣睡覺。

動詞 接 まま 時，肯定 會用 動詞た形，表示 動作 已經做了 之後的 持續狀態

  • 彼女(かのじょ) [1]：代 她
  • は：副助 表 主題
  • いつも [1]：副 總是
  • 電気(でんき) [1]：名 電燈
  • を：格助 接受 つける 的 賓語
  • つける [2]：動II 他動 辞書形 開燈
    • つけた [1]：た形 開了燈
    • つけたまま [1]：持續狀態 開了燈就這樣
  • 寝(ね)る [0]：動II 自動 辞書形 躺下睡覺
    • 寝て [0]：て形 躺下睡覺
    • 寝ている [0]：狀態 躺下睡覺著
    • 寝ています [4]：ます形 躺下睡覺著

シャワーを浴びていないまま寝ました。

沒洗澡就這樣睡著。

動詞 接 まま 時，否定 直接用 動詞ない形 即可，不是使用 なかった

  • シャワー [1]：名 shower、淋浴
  • を：格助 接受 浴びる 的 賓語
  • 浴(あ)びる [0]：動II 他動 辞書形 淋浴
    • 浴びて [0]：て形 淋浴
    • 浴びている [0]：狀態 淋浴著
    • 浴びていない [4]：ない形 不淋浴著
    • 浴びていないまま [4]：持續狀態 不淋浴著就這樣
  • 寝(ね)る [0]：動II 自動 辞書形 躺下睡覺
    • 寝ます [2]：ます形 躺下睡覺
    • 寝ました [2]：た形 躺下睡覺

い形容詞

弱いままでいいのか！

就這樣弱好嗎？

い形容詞 接 まま 時，直接以 い形容詞 形容 まま 即可

  • 弱(よわ)い [2]：い形容 弱的
    • 弱いまま [2]：持續狀態 就這樣弱的
  • で：格助 表 最低限度 的 就
  • いい [1]：い形容 好
  • のか [1]：んですか 的 口語，表 究竟

な形容詞

いくら頑張っても､ 貧乏なままです。

不管怎樣努力， 就這樣貧窮。

な形容詞 接 まま 時，要加上 な 形容 まま

  • いくら [1]：副 不管怎樣
  • 頑張(がんば)る [3]：動I 自動 辞書形 努力
    • 頑張って [3]：て形 努力
    • 頑張っても [3]：即使 即使努力
  • 貧乏(びんぼう) [1]：な形容 貧窮的
    • 貧乏なまま [1]：持續狀態 就這樣貧窮
  • です [0]：助動 是

名詞

この部屋は靴のまま入ってください。

這間房間請就這樣穿著鞋子進來。

名詞 接 まま 時，要加上 の 形容 まま

  • この [0]：連体 這個
  • 部屋(へや) [2]：名 房間
  • は：副助 表 主題
  • 靴(くつ) [1]：名 鞋子
    • 靴のまま [1]：持續狀態 就這樣穿鞋子
  • 入(はい)る [1]：動I 自動 辞書形 進入
    • 入って [1]：て形 進入
    • 入ってください [1]：請託 請進入
