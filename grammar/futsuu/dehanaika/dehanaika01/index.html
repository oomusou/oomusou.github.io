ではないか？ (驚訝)

2025-07-25

普通形 ではないか 可表示 驚訝，用於當說話者發現或遇到一個沒有預期到的事情時，表達自己的驚訝。其中 か 可以 省略。語調 為 上升。翻譯成 ...不是嗎？

接続

  • 普 + ではない (か)
  • Vる + ではない (か)
  • いA + ではない (か)
  • なA + ではない (か)
  • N + ではない (か)

動詞

これ、山田さんが作ったの？なかなか上手にできてるではないか？

這是山田先生做的嗎？做得超棒不是嗎？

  • これ [0]：代 這
  • 山田(やまだ) [0]：姓氏 山田
  • さん [0]：名 先生
  • が：格助 表 主語
  • 作(つく)る [2]：動I 自動 辞書形 做
    • 作った [2]：た形 做
    • 作ったの [2]：關心好奇 做嗎？
  • なかなか [0]：副 相當
  • 上手(じょうず) [3]：な形 拿手的
    • 上手に [3]：副 拿手地
  • できる [2]：動II 自動 辞書形 完成
    • できて [1]：て形 完成
    • できている [1]：結果狀態 完成
    • できてる [1]：縮約 完成
    • できてるではないか？ [1]：驚訝 完成不是嗎？

い形容詞

山田君の作った料理、結構美味しいではないか？

山田先生做的料理，超好吃的不是嗎？

  • 山田(やまだ) [0]：姓氏 山田
  • 君(くん) [1]：名 君
  • の：格助 以 名詞 形容 名詞
  • 作(つく)る [2]：動I 他動 辞書形 做
    • 作った [2]：た形 做了
  • 料理(りょうり) [1]：名 料理
  • 結構(けっこう) [0]：副 很
  • 美味(おい)しい [3]：い形容 好吃的
    • 美味しいではないか？ [3]：驚訝 好吃的不是嗎？

な形容詞

え？これ、君の彼女？すごくきれいではないか？

誒？這是你的女友嗎？超漂亮的不是嗎？

  • え：感 誒
  • これ [0]：代 這
  • 君(きみ) [0]：代 你
  • の：格助 表 所有
  • 彼女(かのじょ) [1]：名 女友
  • すごい [2]：い形容 非常的
    • すごく [2]：副 非常地
  • きれい [1]：な形容 漂亮的
    • きれいではないか？ [1]：驚訝 漂亮的不是嗎？

名詞

おお、山田君ではないか？久しぶり。

啊，這不是山田君嗎？好久不見。

  • おお [1]：感 啊
  • 山田(やまだ) [0]：姓氏 山田
  • 君(くん) [1]：名 君
    • 君ではないか？ [1]：驚訝 不是君嗎？
  • 久(ひさ)しぶり [0]：慣 好久不見
