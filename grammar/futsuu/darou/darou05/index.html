のだろう (推測)

2024-10-06

のだろう 當結尾為 句號 ，且語調 降低 時可表示 有依據的推測，約 70% 可能性，因此前面需要有一句話作為上下文。のだろう 為 のです だろう 的 結合體 。翻譯成 應該是...吧。

接続

  • 普 + のだろう
  • Vる + のだろう
  • いA + のだろう
  • なA + な + のだろう
  • N + な + のだろう

    • あった [1]：た形 發生
