だろうか (自言自語)

2024-10-05

普通形 だろうか 可表示 自言自語 時 對於事情的可能性，抱有懸念、擔心 的語氣，翻譯成 不知道是否...啊。

接続

  • 普 + だろうか
  • Vる + だろうか
  • いA + だろうか
  • なA + だろうか
  • N + だろうか

動詞

あの子は字がうまく書けるようになるだろうか。

那孩子不知道是否學會寫字啊。

  • あの [0]：連体 那個
  • 子(こ) [0]：名 孩子
  • は：副助 表 主題
  • 字(じ) [1]：名 字
  • が：格助 表 主語
  • うまく [1]：副 順利地
  • 書(か)く [1]：動I 他動 辞書形 寫
    • 書ける [2]：可能形 會寫
    • 書けるようになる [2]：能力變化 變得會寫
    • 書けるようになるだろうか [2]：自言自語 不知道是否會寫字啊

あの子にはもう一年ぶりだなあ。うまく話しができるようになっただろうか。

已經一年沒有看到那孩子了。不知道他現在是否會講話了啊。

  • あの [0]：連体 那個

  • に：格助 表 動作對象
  • は：副助 表 主題

    • 一年ぶり [2]：接尾 經過…之後又…
  • だ：助動 是
  • なあ [1]：終助 表 感嘆
  • うまく [1]：副 順利地
  • 話(はな)す [2]：動I 他動 辞書形 講

    • で きるようになった [2]：た形 變得會了
    • で きるようになっただろうか [2]：自言自語 不知道是否會講話了啊

今度の計画はうまくいくだろうか。

這次的計畫不知道會不會順利啊。

  • だろうか [2]：自言自語 不知道是否…啊
