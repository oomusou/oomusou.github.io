そうな (樣態)

2025-06-13

そう 裝飾之後成為 な形容詞，可修飾 名詞。

接続

  • Vます ます + そうな
  • いA い + そうな
  • なA + そうな

動詞

エンジンが壊れそうな音がしている。

引擎發出好像要壞掉的聲音。

  • エンジン [1]：名 engine、引擎
  • が：格助 表 主語
  • 壊(こわ)れる [0]：動II 自動 辞書形 壞掉
    • 壊れそう [4]：樣態 好像要壞掉
    • 壊れそうな [4]：な形容 好像要壞掉的
  • 音(おと) [0]：名 聲音
  • が：格助 表 主語
  • する [0]：動III 自動 辞書形 發出聲音
    • して [0]：て形 發出聲音
    • している [0]：現在進行 在發出聲音

い形容詞

面白そうな建物ですね。

好像很有趣的建築啊。

  • 面白(おもしろ)い [4]：い形容 有趣的
    • 面白そう [5]：樣態 好像有趣的
    • 面白そうな [5]：な形容 好像有趣的
  • 建物(たてもの) [2]：名 建築
  • です [0]：助動 是
  • ね：終助 表 稱讚

な形容詞

とても静かそうな所だ。

好像很安靜的地方。

  • とても [0]：副 很
  • 静(しず)か [1]：な形容 安靜的
    • 静かそう [4]：樣態 好像安靜的
    • 静かそうな [4]：な形容 好像安靜的
  • 所(ところ) [0]：名 地方
  • だ：助動 是
