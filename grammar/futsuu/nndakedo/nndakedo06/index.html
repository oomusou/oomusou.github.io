んだけど！ (驚訝)

2025-11-18

普通形 んだけど 若置於 句尾 可表示 驚訝 ，最後會加上 ！。

接続

  • 普 + んだけど + ！
  • Vる + んだけど + ！
  • いA + んだけど + ！
  • なA + な + んだけど + ！
  • N + な + んだけど + ！

例文

知らない男が後ろに写ってるんだけど！

後面有個生男人入鏡欸！

  • 知(し)る [0]：動I 他動 辞書形 認識
    • 知らない [0]：ない形 不認識的
  • 男(おとこ) [3]：名 男人
  • が：格助 表 主語
  • 後(うし)ろ [0]：名 後面
  • に：格助 表 存在位置
  • 写(うつ)る [2]：動I 自動 辞書形 入鏡
    • 写って [2]：て形 入鏡
    • 写っている [2]：狀態 入鏡著
    • 写ってる [2]：縮約 入鏡著
    • 写ってるんだけど！[2]：驚訝 入鏡著！
