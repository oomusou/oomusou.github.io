とすると (仮定)

2025-06-18

普通形 とすると 可表示 仮定，語氣比 条件 的 と 強，通常指 可能性較低 的事情。翻譯成 假設。

接続

  • 普 + とすると
  • Vる + とすると
  • いA + とすると
  • なA + だ + とすると
  • N + だ + とすると

動詞

行くとすると、今しかない。

假設要去，只能現在了。

  • 行(い)く [0]：動I 自動 辞書形 去
    • 行くとすると [0]：仮定 假設要去
  • 今（いま）[1]：名 現在
  • しかない [3]：副助 只能

い形容詞

うるさいとすると、文句が出る。

假設太吵，會被抱怨。

  • うるさい [3]：い形容 吵雜的
    • うるさいとすると [3]：仮定 假設吵雜的
  • 文句(もんく) [1]：名 抱怨
  • が：格助 表 主語
  • 出(で)る [1]：動II 自動 出現

な形容詞

便利だとすると、人気が出る。

假設方便，就會受歡迎。

  • 便利(べんり) [1]：な形容 方便
    • 便利だとすると [1]：仮定 假設方便
  • 人気(にんき) [0]：名 人氣
  • 出(で)る [1]：動II 自動 出現

名詞

明日が雨だとすると、試合は中止だ。

假設明天下雨，比赛就會取消。

  • 明日(あした) [0]：名 明天
  • が：格助 表 主語
  • 雨(あめ) [1]：名 雨
    • 雨だとすると [1]：仮定 假設下雨
  • 試合(しあい) [0]：名 比賽
  • は：副助 表 主題
  • 中止(ちゅうし) [0]：名 取消
  • だ：助動 是
