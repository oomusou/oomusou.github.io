がり (個性)

2026-06-22

形容詞 がり 或 形容詞 がり屋 可形容人 個性 ，常用在表達 個人感情 或 身體感覺 的 形容詞。

難度

  • N3

情境

用於描述某人經常表現出的 感情反應 或 身體感覺傾向，將其視為 較固定 的 個性 或 特徵。

接続

  • いA い + がり
  • いA い + がり屋
  • なA + がり
  • なA + がり屋

がり

彼は暑がりだから、いつもエアコンをつけている。

他很怕熱，所以總是開著冷氣。

較短 的 形容詞 用 がり

  • 彼(かれ) [1]：代 他
  • は：副助 表 主題
  • 暑(あつ)い [2]：い形容 熱的
    • 暑がり [3]：名 怕熱的人
  • だ：助動 是
  • から [1]：接助 表 原因
  • いつも [1]：副 總是
  • エアコン [0]：名 冷氣
  • を：格助 接受 つける 的 賓語
  • つける [2]：動II 他動 辞書形 開啟
    • つけて [1]：て形 開啟
    • つけている [1]：持續狀態 開著

がり屋

彼女は恥ずかしがり屋なので、人前で話すのが苦手だ。

她很害羞，所以不擅長在眾人面前說話。

較長 的 形容詞 用 がり屋

  • 彼女(かのじょ) [1]：代 她
  • は：副助 表 主題
  • 恥(は)ずかしい [4]：い形容 害羞的
    • 恥ずかしがり屋(や) [4]：名 容易害羞的人
  • なので：接助 表 原因
  • 人前(ひとまえ) [0]：名 眾人面前
  • で：格助 表 動作場所
  • 話(はな)す [2]：動I 他動 辞書形 說話
    • 話すの [2]：名 說話
  • が：格助 表 主語
  • 苦手(にがて) [0]：な形容 不擅長的
  • だ：助動 是
