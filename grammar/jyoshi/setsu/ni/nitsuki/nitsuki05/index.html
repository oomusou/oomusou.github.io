につきまして (關於)

2026-06-28

接助詞 につきまして 可表示 關於，為 について 的 書面語。

難度

  • N3

情境

以較 正式、商務 的方式針對某個 主題 進行 說明、通知 或 報告 時使用。

接続

  • N + につきまして + V
  • N + につきまして + の + N
  • N + につきまして + は + V

例文

新しい企画につきまして、ご説明いたします。

關於新的企劃，將為您說明。

につきまして 的 語源 為 に付く -> に付きます -> につきまして，本意為 附著在某個對象之上，並以此為主題來說，也就是 關於

  • 新(あたら)しい [4]：い形容 新的
  • 企画(きかく) [0]：名 企劃
  • につきまして [0]：接助 關於
  • 説明(せつめい) [0]：名 說明
    • 説明する [0]：動III 他動 辞書形 說明
    • ご説明する [0]：謙讓語 說明
    • ご説明いたす [0]：丁重語 說明
    • ご説明いたします [9]：ます形 說明

会議の日程につきましては、後日改めてご連絡いたします。

關於會議的日程，日後會再另行通知您。

  • 会議(かいぎ) [1]：名 會議
  • の：格助 表 所屬

    • ご連絡する [0]：謙讓語 聯絡

料金につきましての詳細は、こちらの資料をご覧ください。
