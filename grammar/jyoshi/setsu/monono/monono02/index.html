とはいうものの (雖然)

2026-05-11

接助詞 とはいうものの 可表示前項 雖然 成立，但結果並 不如預期 的語氣。為 けど 的 書面語，比 ものの 更 正式 。翻譯成 雖然。

接続

  • 普 + とはいうものの
  • V普 + とはいうものの
  • いA + とはいうものの
  • なA + だ + とはいうものの
  • N + だ + とはいうものの

動詞

日本に住んでいるとはいうものの、日本語が話せません。

雖然住在日本，但是不會說日語。

  • 日本(にほん) [2]：地名 日本
  • に：格助 表 地著點
  • 住(す)む [1]：動I 自動 辞書形 居住
    • 住んで [1]：て形 居住
    • 住んでいる [1]：狀態 居住著
    • 住んでいるとはいうものの [1]：雖然 雖然居住著
  • 日本語(にほんご) [0]：名 日語
  • が：格助 表 主語
  • 話(はな)す [2]：動II 他動 辞書形 說
    • 話せる [3]：可能形 會說
    • 話せます [4]：ます形 會說
    • 話せません [5]：ない形 不會說

い形容詞

背が高いとはいうものの、足が速いわけではない。

雖然個子高，但並不代表跑得快。

  • 背(せ) [1]：名 身高
  • が：格助 表 主語
  • 高(たか)い [2]：い形容 高的
    • 高いとはいうものの：雖然 雖然個子高
  • 足(あし)が速(はや)い [5]：慣 跑得快
    • 足が速いわけではない：並非 並非跑得快

な形容詞

家が豪華だとはいうものの、お金持ちではない。

雖然房子很豪華，但並不是有錢人。

  • 家(いえ) [2]：名 家
  • が：格助 表 主語
  • 豪華(ごうか) [1]：な形容 豪華的
    • 豪華だとはいうものの [1]：雖然 雖然豪華的
  • お金持(かねも)ち [0]：名 有錢人
    • お金持ちではない：ない形 不是有錢人

名詞

趣味だとはいうものの、ほとんどプロレベルだ。

雖然只是興趣，但幾乎是專業等級。

  • 趣味(しゅみ) [1]：名 興趣
    • 趣味であるものの [1]：雖然 雖然是興趣
  • ほとんど [2]：副 幾乎
  • プロ [1]：名 pro、專業
  • レベル [1]：名 level、等級
  • だ：助動 斷定
