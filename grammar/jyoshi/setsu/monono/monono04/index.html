からよかったものの (幸好)

2026-05-13

接助詞 からよかったものの 可表示結果還算 幸運，但其實情況原本很 危險 或 嚴重 的語氣 。用於 口語。翻譯成 幸好 不然。

接続

  • 普 + からよかったものの
  • V普 + からよかったものの
  • いA + からよかったものの
  • なA + だ + からよかったものの
  • N + だ + からよかったものの

動詞

間に合ったからよかったものの、あと少し遅れていたら大変だった。

幸好有趕上，不然再晚一點就慘了。

からよかったものの 的 語源 為 から + よかった + ものの，本意是 雖然還好，衍伸為 幸好 不然

  • 間(ま)に合(あ)う [3]：動I 自動 辞書形 趕上
    • 間に合った [3]：た形 趕上了
    • 間に合ったからよかったものの [3]：幸好 幸好趕上了，不然就
  • あと [1]：副 再
  • 少(すこ)し [2]：副 稍微
  • 遅(おく)れる [0]：動II 自動 辞書形 遲到
    • 遅れて [0]：て形 遲到
    • 遅れている [0]：現在進行 遲到
    • 遅れていた [0]：過去進行 遲到
    • 遅れていたら [6]：条件 遲到的話
  • 大変(たいへん) [0]：な形容 嚴重的
  • だ：助動 斷定
    • だった [1]：た形 斷定

い形容詞

幸好你還年輕，不然一直這樣亂來身體會壞掉。

  • 若(わか)い [2]：い形容 年輕的
    • 若いからよかったものの [0]：幸好 幸好年輕，不然
  • そんな [0]：連体 那樣的
  • 無理(むり) [1]：名 亂來
  • を：格助 接受 続ける 的 賓語
  • 続(つづ)ける [0]：動II 他動 辞書形 繼續
    • 続けた [0]：た形 繼續
    • 続けたら [0]：条件 繼續的話
  • 体(からだ) [0]：名 身體
  • を：格助 接受 壊す 的 賓語
  • 壊(こわ)す [2]：動I 他動 弄壞
  • よ：終助 表 告知
