が (雖然)

2025-05-06

接助詞 が 可 逆接 兩個句子表示 雖然，其中句子可放 丁寧体，也可放 普通体。

接続

  • 丁/普 + が
  • V + が
  • いA + が
  • なA + だ + が
  • N + だ + が

動詞

毎日試験勉強したが、不合格だった。

雖然每天準備考試，但是還是不及格。

  • 毎日(まいにち) [1]：名 每天
  • 試験(しけん) [2]：名 考試
  • 試験勉強(しけんべんきょう)する：動III 自他 辞書形 準備考試
    • 試験勉強した：た形 準備考試
  • が：接助 表 雖然
  • 不合格(ふごうかく) [2]：名 不及格
  • だ：助動 是
    • だった：た形 是

い形容詞

この部屋は狭いが、キレイだ。

這個房間雖窄，但很乾淨。

  • 部屋(へや) [2]：名 房間
  • は：副助 表 主語
  • 狭(せま)い [2]：い形容 狹窄的
  • が：接助 表 雖然
  • キレイ [1]：な形容 乾淨的
  • だ：助動 是

な形容詞

日本の歌が大好きだが、歌があまり上手じゃない。

雖然很喜歡日本歌，但是唱歌我不太拿手。

  • 日本(にほん)の歌(うた) [2]：名 日文歌
  • が：格助 表 主語
  • 大好(だいす)き [1]：非常喜歡
  • だが：接助 表 雖然
  • 歌(うた) [2]：名 歌
  • が：格助 表 主語
  • あまり [0]：副 不太 (後接否定)
  • 上手(じょうず) [3]：な形容 拿手的
    • 上手じゃない：ない形 不拿手的

名詞

今日は日曜日だが、会社へ行く。

雖然今天是星期天，但是我去公司。

  • 今日(きょう) [1]：名 今天
  • は：副助 表 主語
  • 日曜日(にちようび) [3]：名 星期天
  • だが：接助 表 雖然
  • 会社(かいしゃ) [0]：名 公司
  • 行(い)く [0]：動I 自動 去
