に (行動目的)

2025-07-24

格助詞 に 可表示 行く、来る、帰る、戻る、入る、出る、出かける、登る、通う、参加する … 的 行動目的。に 之前可接 名詞 或 動詞ます形 ます，因為 動詞ます形 ます 可視為 名詞。

接続

  • N + に + V
  • Vます ます + に + V

行く

東京へ日本料理を食べに行く。

去東京吃日本料理。

  • 東京(とうきょう) [0]：地名 東京
  • へ：格助 表 移動方向
  • 日本料理(にほんりょうり) [4]：名 日本料理
  • の：格助 以 名詞 修飾 名詞
  • 食(た)べる [2]：動II 他動 辞書形 吃
    • 食べます [3]：ます形 吃
    • 食べ [0]：名 吃
  • に：格助 表 行動目的
  • 行(い)く [0]：去 去

  • 紅茶(こうちゃ) [0]：名 紅茶
  • を：格助 接受 買う 的 賓語
  • 買(か)う [0]：動I 他動 辞書形 買

  • に：格助 表 行動目的
