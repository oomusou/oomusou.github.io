や (列舉)

2025-09-22

格助詞 や 可表示同性質 名詞 的 列舉。常搭配副助詞 など，但亦可省略。

接続

  • N + や
  • N + や + N + など

例文

電車やバスが動かなかったら、歩いて帰る。
電車やバスなどが動かなかったら、歩いて帰る。

如果電車或巴士等等都停駛，就走路回家。

  • 電車(でんしゃ) [0]：名 電車
  • や：格助 表 列舉
  • バス [1]：名 巴士
  • など [1]：副助 表 類推
  • が：格助 表 主語
  • 動(うご)く [0]：動I 自動 辞書形 動
    • 動かない [0]：ない形 不動
    • 動かなかった [0]：た形 不動
    • 動かなかったら [0]：如果 如果不動
  • 歩(ある)く [2]：動I 自動 辞書形 走路
    • 歩いて：て形 方法手段 走路
  • 帰(かえ)る [1]：動I 回家
