ようと まいと (無論是否)

2026-03-20

疑問詞 意向形 と 動詞 まいと 可表示 無論是否，為 どようと 的 進階版 。翻譯成 無論是否 都。

接続

  • Vよう + と + V + まいと
  • Vよう + と + Vる + まいと (動I/動II/動III)
  • Vよう + と + Vない ない + まいと (動II/動III)
  • Vよう + と + Vする る + まいと (する)
  • Vよう + と + Vします ます + まいと (する)

辞書形

雨が降ろうと降るまいと、キャンプに行くつもりです。

無論是否下雨，我都打算去露營。

  • 雨(あめ) [1]：名 雨
  • が：格助 表 主語
  • 降(ふ)る [1]：動I 自動 辞書形 下雨
    • 降ろう [2]：意向形 要下雨
    • 降ろうと [2]：無論是否 無論是否下雨
  • 降(ふ)る [1]：動I 自動 辞書形 下雨
    • 降るまい [3]：否定意向 不要下雨
      　* 降るまいと [3]：無論是否 無論是否下雨
  • キャンプ [1]：名 camp、露營
  • に：格助 表 行動目的
  • 行く(いく) [0]：動I 自動 辞書形 去
    • 行くつもりだ [3]：打算 打算去
    • 行くつもりです [3]：丁寧形 打算去

ない形

信じようと信じまいと、嘘をついていません。

無論你是否相信，我都沒有說謊。

  • 信(しん)じる [3]：動II 他動 辞書形 相信
    • 信じよう [4]：意向形 要相信
    • 信じるようと [4]：無論是否 無論是否相信
  • 信(しん)じる [3]：動II 他動 辞書形 相信
    • 信じまい [4]：否定意向 不相信
    • 信じまいと [4]：無論是否 無論是否相信
  • 嘘(うそ) [1]：名 謊言
  • を：格助 接受 つく 的 賓語
  • つく [1]：動I 他動 辞書形 說謊
    • ついて [1]：て形 說謊
    • ついている [1]：現在進行 在說謊
    • ついています [5]：ます形 在說謊
    • ついていません [6]：ない形 不在說謊

する

勉強しようと勉強すまいと、結果は変わらない。
勉強しようと勉強しまいと、結果は変わらない。

無論是否唸書，結果都不會改變。

  • 勉強(べんきょう) [0]：名 唸書
    • 勉強する [0]：動III 自動 辞書形 唸書
    • 勉強しよう [6]：意向形 要唸書
    • 勉強しようと [6]：無論是否 無論是否唸書
  • 勉強(べんきょう) [0]：名 唸書
    • 勉強する [0]：動III 自動 辞書形 唸書
    • 勉強すまい [6]：否定意向 不唸書
    • 勉強すまいと [6]：無論是否 無論是否唸書
    • 勉強します [6]：ます形 唸書
    • 勉強しまい [6]：否定意向 不唸書
    • 勉強しまいと [6]：無論是否 無論是否唸書
  • 結果(けっか) [0]：名 結果
  • は：副助 表 主題
  • 変(わ)わる [0]：動I 自動 改變
    • 変わらない [0]：ない形 不改變
