と、た (發現條件)

2026-03-17

普通形 と、動詞た形 可表示 發現條件，為 前句 動作發生了 之後，才發現 後句 的 結果。有 意想不到 的 語感。翻譯成 一看才發現。

接続

  • 普 + と、Vた
  • Vる + と、Vた
  • いA + と、Vた
  • なA + だ + と、Vた
  • N + だ + と、Vた

動詞

家に帰ると、空き巣に入られました。

回到家，一看才發現家裡被小偷闖入了。

と 表示 發現條件 時，後句不能有 意志、希望、請求、命令，因為 發現條件 時，後句 已經發生， 為 客觀現象，而 意志、希望、請求、命令 為 主觀意識，必須改用 動詞条件形

  • 家(いえ) [2]：名 家
  • に：格助 表 目的地
  • 帰(かえ)る [1]：動I 自動 辞書形 回家
    • 帰ると [1]：發現條件 回到家一看才發現
  • 空(あ)き巣(す) [0]：名 闖空門的小偷
  • に：格助表 動作者
  • 入(はい)る [1]：動I 自動 辞書形 進入
    • 入られる [4]：受身形 被進入
    • 入られます [5]：ます形 被進入
    • 入られました [5]：た形 被進入了

い形容詞

彼の顔が赤いと、熱があった。

他臉很紅，一看才發現他發燒了。

後句因為 已經發生，所以要用 動詞た形

  • 彼(かれ) [1]：代 他
  • の：格助 表 所屬
  • 顔(かお) [0]：名 臉
  • が：格助 表 主語
  • 赤(あか)い [0]：い形容 紅的
    • 赤いと [0]：發現條件 臉紅一看才發現
  • 熱(ねつ)がある [4]：慣 發燒
    • 熱があった [4]：た形 發燒了

な形容詞

店の中がにぎやかだと、パーティーをしていた。

店裡很熱鬧，一看才發現正在辦派對。

  • 店(みせ) [2]：名 店
  • の：格助 表 方位
  • 中(なか) [1]：名 裡面
  • が：格助 表 主語
  • にぎやか [2]：な形容 熱鬧的
    • にぎやかだと [2]：發現條件 熱鬧一看才發現
  • パーティー [1]：名 party、派對
  • を：格助 接受 する 的 賓語
  • する [0]：動III 他動 辞書形 舉行
    • して [0]：て形 舉行
    • している [0]：現在進行 在舉行
    • していた [0]：過去進行 在舉行

名詞

ここが出口だと、人がたくさん並んでいた。

這裡是出口，一看才發現很多人在排隊。

  • ここ [0]：代 這裡
  • が：格助 表 主語
  • 出口(でくち) [1]：名 出口
    • 出口だと [1]：發現條件 是出口一看才發現
  • 人(ひと) [2]：名 人
  • が：格助 表 主語
  • たくさん [0]：副 很多
  • 並(なら)ぶ [0]：動I 自動 辞書形 排隊
    • 並んで [0]：て形 排隊
    • 並んでいる [0]：現在進行 在排隊
    • 並んでいた [0]：過去進行 在排隊
