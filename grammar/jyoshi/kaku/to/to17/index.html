と (假定條件)

2026-03-17

接助詞 と 可表示 假定條件，為 依據常理 之 確信推測，只要前者 符合條件，後者 就會發生。翻譯成 如果 就會。

接続

  • 普 + と
  • Vる / Vない + と
  • いA / いA い くない + と
  • なA + だ / なA ではない/じゃない + と
  • N + だ / N ではない/じゃない + と

動詞

勉強を長時間続けると、効率が落ちます。

如果長時間持續唸書，效率就會下降。

と 表示 假定條件 時，後句不能有 意志、希望、請求、命令，因為 假定條件 為 依據常理 之 確信推測，屬於 客觀現象，而 意志、希望、請求、命令 為 主觀意識，必須改用 動詞条件形

  • 勉強(べんきょう) [0]：名 唸書
  • を：格助 接受 続ける 的 賓語
  • 長時間(ちょうじかん) [3]：名 長時間
  • 続(つづ)ける [0]：動II 他動 持續
    • 続けると [0]：假定條件 如果持續就
  • 効率(こうりつ) [0]：名 效率
  • が：格助 表 主語
  • 落(お)ちる [2]：動I 自動 下降
    • 落ちます [3]：ます形 下降

急がないと、間に合いませんよ。

如果不快點的話，就會來不及了喔。

  • 急ぐ(いそぐ) [2]：動I 自動 辞書形 趕快
    • 急がない [3]：ない形 不趕快
    • 急がないと [3]：假定條件 如果不趕就
  • 間(ま)に合(あ)う [3]：動I 自動 辞書形 趕上
    • 間に合います [5]：ます形 趕上
    • 間に合いません [4]：ない形 趕不上
  • よ：終助 表 提醒

い形容詞

値段が高いと、買う人が少なくなります。

如果價格很高，購買的人就會變少。

  • 値段(ねだん) [0]：名 價格
  • が：格助 表 主語
  • 高(たか)い [2]：い形容 昂貴的
    • 高いと [2]：假定條件 如果昂貴的就
  • 買(か)う [0]：動I 他動 購買
  • 人(ひと) [2]：名 人
  • が：格助 表 主語
  • 少(すく)ない [3]：い形容 少的
    • 少なく [3]：副 少地
  • なる [1]：動I 自動 辞書形 變成
    • なります：ます形 變成了

な形容詞

説明が丁寧だと、分かりやすい。

如果說明很仔細，就會容易理解。

  • 説明(せつめい) [0]：名 說明
  • が：格助 表 主語
  • 丁寧(ていねい) [1]：な形容 仔細的
    • 丁寧だと [1]： 假定條件 如果仔細的就
  • 分(わ)かる [2]：動I 自動 辞書形 理解
    • 分かります [4]：ます形 理解
    • 分かりやすい [5]：容易地 容易理解

名詞

週末だと、どこの観光地も混雑します。

如果是週末的話，哪個觀光地都會很擁擠。

  • 週末(しゅうまつ) [0]：名 週末
    • 週末だと：假定條件 如果是週末就
  • どこ [1]：代 哪裡
  • の：格助 表 修飾
  • 観光地(かんこうち) [3]：名 觀光地
  • も：副助 表 都
  • 混雑(こんざつ) [0]：名 擁擠
    • 混雑する [0]：動III 自動 辞書形 擁擠
    • 混雑します [6]：ます形 擁擠
