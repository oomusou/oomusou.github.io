も 與其他格助詞並用

2025-05-30

當 副助詞 も 接在 格助詞 か 或 を 的 名詞 之後時，が 和 を 必須 省略。其他 格助詞 (如 に、で、から、まで、と) 則必須 保留。へ 則可 省略 亦可保留。

接続

  • N + に + も
  • N + で + も
  • N + から + も
  • N + まで + も
  • N + と + も

例文

日本語も英語も勉強する。

日文、英文都要唸。

原本應為 日本語も英語をも勉強する。，但 を 在 も 之前必須省略

  • 日本語(にほんご) [0]：名 日語
  • も：副助 表 並立
  • 英語(えいご) [0]：名 英語
  • 勉強(べんきょう)する [0]：動III 自動 唸書

作文の宿題は3時間もかかる。

竟然花了 3 小時寫作文習題。

原本應為 作文の宿題は3時間がもかかる。，但 が 在 も 之前必須省略

  • 作文(さくぶん) [0]：名 作文
  • の：格助 以 名詞 形容 名詞
  • 宿題(しゅくだい) [0]：名 習題
  • は：副助 表 主題
  • 3 時間(さんじかん) [0]：名 3 小時
  • も：格助 表 竟然
  • かかる [2]：動I 自動 花費

他にもいろいろある。

其他還有很多各式各樣的。

に 在 も 之前必須保留

  • 他(ほか) [0]：名詞 別處
  • に：格助 表 存在位置
  • も：格助 表 也
  • いろいろ [0]：副 各式各樣
  • ある [1]：動III 自動 有

日本も行く。
日本へも行く。

也去日本。

へ 在 も 之前可省略亦可保留

  • 日本(にほん) [0]：地名 日本
  • も：格助 表 也
  • へ：格助 表 移動方向
  • 行(い)く [0]：動I 自動 去
