だけのことはある (真不虧)

2025-05-06

普通形 だけのことはある 可表示 稱讚，用於 正面描述，常搭配副詞 さすが 或 やっばり，翻譯成 真不虧。

接続

  • 普 + だけのことはある
  • V + だけのことはある
  • いA + だけのことはある
  • なA + な / である + だけのことはある
  • N + (である) + だけのことはある

動詞

なやなか予約が取れない。あのレストランは人気があるだけのことはある。

很難能取得預約。那家餐廳真不虧受歡迎。

  • なかなか [0]：副 很難地
  • 予約(よやく) [0]：名 預約
  • が：格助 表 主語
  • 取(と)る [1]：動I 他動 辞書形 取得
    • 取れる [2]：可能形 可取得
    • 取れない [2]：ない形 不可取得
  • あの [0]：連体 那個
  • レストラン [1]：名 restaurant，餐廳
  • は：副助 表 主題
  • 人気(にんき)がある [0]：慣 有人氣
    • 人気があるだけのことはある [9]：真不虧 真不虧有人氣

い形容詞

仕事を覚えるのが早いです。高橋さんは若いだけのことはある。

工作學得很快。高橋先生真不虧年輕。

  • 仕事(しごと)を覚(おぼ)える [0]：慣 學會工作
    • 仕事を覚えるの [0]：名 學會工作
  • が：格助 表 主語
  • 早(はや)い [2]：い形容 快的
  • です [0]：助動 是
  • 高橋(たかはし) [0]：姓氏 高橋
  • さん [0]：名 先生
  • は：副助 表 主題
  • 若(わか)い [2]：い形容 年輕的
    • 若いだけのことはある [6]：真不虧 真不虧年輕的

な形容詞

とても乗り心地がいいです。この車は高級なだけのことはある。

坐起來很舒服。這台車真不虧很高級。

  • とても [0]：副 很
  • 乗(の)り心地(ごこち) [3]：名 乘坐感
  • が：格助 表 主題
  • いい [1]：い形容 好的
  • です [0]：助動 是
  • この [0]：連体 這個
  • 車(くるま) [0]：名 車
  • は：副助 表 主題
  • 高級(こうきゅう) [0]：な形容 高級的
    • 高級なだけのことはある [8]：真不虧 真不虧高級的

名詞

体力がありますね。彼女はさすが元オリンピック選手だけのことはある

很有體力呢。她真不虧是前奧運選手。

  • 体力(たいりょく) [1]：名 體力
  • が：格助 表 主題
  • ある [1]：動I 自動 辞書形 有
    • あります [3]：ます形 有
  • ね：終助 表 尋求認同
  • 彼女(かのじょ) [1]：代 她
  • は：副助 表 主題
  • さすが [0]：副 真不虧
  • 元(もと) [1]：名 前
  • オリンピック [4]：名 olympic、奧林匹克
  • 選手(せんしゅ) [0]：名 選手
    • 選手だけのことはある [7]：真不虧 真不虧是選手
