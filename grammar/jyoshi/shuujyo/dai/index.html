だい (疑問)

2026-07-14

終助詞 だい 可表示帶有 親近感 的 疑問。多用於 詢問理由、場所、人物 或 狀態 等具體內容，語氣比一般 んだ 柔和，但帶有 男性 或 長輩 口吻。翻譯成 是 呢？。

難度

  • N3

情境

常見於 男性、長輩對晚輩、家人 或 親近關係 中的 口語 詢問，用來 溫和 追問具體資訊，正式場合 較 少 使用。

接続

  • 疑 + 普 + んだい
  • 疑 + V普 + んだい
  • 疑 + いA + んだい
  • 疑 + なA + な + んだい
  • 疑 + N + な + んだい

動詞

こんな時間にどこへ行くんだい？

這種時間你要去哪裡呢？

  • こんな [0]：連体 這樣的
  • 時間(じかん) [0]：名 時間
  • に：格助 表 時間點
  • どこ [1]：疑 哪裡
  • へ：格助 表 移動方向
  • 行(い)く [0]：動I 自動 辞書形 去
    • 行くんだい [0]：疑問 要去呢？

い形容詞

何がそんなに面白いんだい？

什麼事情那麼有趣呢？

  • 何(なに) [1]：疑 什麼
  • が：格助 表 主語
  • そんなに [0]：副 那麼
  • 面白(おもしろ)い [4]：い形容 有趣的
    • 面白いんだい [4]：疑問 有趣呢？

な形容詞

どうして今日はそんなに静かなんだい？

為什麼今天那麼安靜呢？

  • どうして [1]：疑 為什麼
  • 今日(きょう) [1]：名 今天
  • は：副助 表 主題
  • そんなに [0]：副 那麼
  • 静(しず)か [1]：な形容 安靜的
    • 静かなんだい [1]：疑問 安靜呢？

名詞

これは誰の本なんだい？

這是誰的書呢？

  • これ [0]：代 這個
  • は：副助 表 主題
  • 誰(だれ) [1]：疑 誰
  • の：格助 表 所屬
  • 本(ほん) [1]：名 書
    • 本なんだい [1]：疑問 是書呢？
