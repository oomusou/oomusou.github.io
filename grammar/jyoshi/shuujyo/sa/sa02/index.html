とさ (聽說)

2025-10-09

終助詞 とさ 可表示 聽說。

接続

  • 普 + とさ
  • Vる + とさ
  • いA + とさ
  • なA + だ + とさ
  • N + だ + とさ

動詞

あの山の上に昔お城があったとさ。

聽說那座山上以前有座城。

  • あの [0]：連体 那個
  • 山(やま) [2]：名 山
  • の：格助 表 方位
  • 上(うえ) [0]：名 上面
  • に：格助 表 存在位置
  • 昔(むかし) [0]：名 從前
  • お城(しろ) [0]：名 城堡
  • が：格助 表 主語
  • ある [1]：動I 自動 辞書形 有
    • あった [1]：た形 有了
    • あったとさ [1]：終助 表 聽說

い形容詞

あの山は冬になるととても寒いとさ。

聽說那座山一到冬天就非常冷。

  • あの [0]： 連体 那個
  • 山(やま) [2]：名 山
  • は：副助 表 主題
  • 冬(ふゆ) [2]：名 冬天
  • に：格助 表 發生時間
  • なる [1]：動I 自動 辞書形 成為
    • なると [1]：繼起 一成爲救
  • 寒(さむ)い [2]：い形容 寒冷的
    • 寒いとさ [2]：終助 表 聽說

な形容詞

彼女の家は立派だとさ。

聽說她家很氣派。

  • 彼女(かのじょ) [1]：代 她
  • の：格助 表 所屬
  • 家(いえ) [2]：名 家
  • は：副助 表 主題
  • 立派(りっぱ) [0]：な形容詞 氣派的
    • 立派だとさ [0]：終助 表 聽說

名詞

来週の月曜日学校休みだとさ。

聽說下週星期一學校休息。

  • 来週(らいしゅう) [0]：名 下週
  • の：格助 以 名詞 形容 名詞
  • 月曜日(げつようび) [3]：名 星期一
  • 学校(がっこう) [0]：名 學校
  • 休(やす)み [3]：名 休息
    • 休みだとさ [3]：終助 表 聽說
