使用 $computed 取代 computed

2021-10-01

隨著 Vue 3.2 提供 $ref Compiler Macro，亦提供 $computed，讓我們不必引用 computed 就可直接建立 Computed。

Version

Vue 3.2

Reactive Reference

2 會每秒鐘自動遞增加 1，右側值會連動改變。

<template>
  {{ count }} {{ plus }}
</template>

<script setup>
import { ref } from 'vue'

let count = ref (0)
let plus = $computed (_ => count.value + 1)

setInterval (_ => count.value++, 1000)
</script>

第 8 行

let count = ref (0)

使用 ref 建立 count Reactive Reference。

11 行

setInterval (_ => count.value++, 1000)

每一秒對 count 遞增。

第 9 行 

let plus = $computed (_ => count.value + 1)

將 count Reactive Reference 由 .value 取出後，再由 $computed 包成 plus computed。

Reactive Object

功能不變，但使用 Reactive Object 改寫。

<template>
  {{ state.count }} {{ plus }}
</template>

<script setup>
import { reactive } from 'vue'

let state = reactive ({ count: 0 })
let plus = $computed (_ => state.count + 1)

setInterval (_ => state.count++, 1000)
</script>

第 8 行

let state = reactive ({ count: 0 })

使用 reactive 建立 state Reactive Object。

第 9 行

let plus = $computed (_ => state.count + 1)

將 state Reactive Object 由 count 取出後，再由 $computed 包成 plus computed。

Reactive Variable

功能不變，但使用 Reactive Variable 改寫。

<template>
  {{ count }} {{ plus }}
</template>

<script setup>
let count = $ref (0)
let plus = $computed (_ => count + 1)

setInterval (_ => count++, 1000)
</script>

第 6 行

let count = $ref (0)

使用 $ref 建立 count Reactive Variable。

第 7 行

let plus = $computed (_ => count + 1)

直接將 count Reactive Varible 取出後，再由 $computed 包成 plus computed。

Conclusion

  • Vue 持續提供更精簡 syntax sugar，可不必引用 computed 就可建立 computed
  • $computed 可搭配 Reactive Reference、Reactive Object 與 Reactive Variable
  • $computed 回傳的依然是 reference，而非 variable，因此讀取時仍須 .value
