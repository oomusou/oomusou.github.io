使用 Event 從 Component 內傳出 Data

2021-04-29

由於 Component 是 Unidirection Dataflow，因此 Data 只能由外往內傳，若要由內往外傳，則要使用 Event。

Version

Vue 3.0.11

Composition API

當按下 + 對 counter 遞增時，會即時將 data 透過 event 由內往外傳。

App.vue

<template lang='pug'>
MyCounter(:startCount='outerCount', @countChange='onCountChange')
span {{ outerCount }}
</template>

<script setup>
import MyCounter from '/src/components/MyCounter.vue'

ref: outerCount = 3
let onCountChange = x => outerCount = x
</script>

第 7 行

import MyCounter from '/src/components/MyCounter.vue'

引用 MyCounter component。

第 2 行

MyCounter(:startCount='outerCount', @countChange='onCountChange')
span {{ outerCount }}

MyCounter 除了傳入 startCount prop 外，準備接收 countChange event 所傳出 data。

第 9 行

ref: outerCount = 3

使用 ref sugar 建立 outerCount state 且初始值為 3。

10 行

let onCountChange = x => outerCount = x

處理 accountChange event，將接收值寫入 outerCount state。

MyCounter.vue

<template lang='pug'>
button(@click='onClick') +
</template>

<script setup>
import { defineProps, defineEmit } from 'vue'

let props = defineProps({ startCount: 0 })
let emit = defineEmit(['countChange'])
ref: innerCount = props.startCount

let onClick = () => {
  innerCount++
  emit('countChange', innerCount)
}
</script>

第 8 行

let props = defineProps({ startCount: 0 })

使用 defineProps() 定義 startCount prop，且初始值為 0。

第 9 行

let emit = defineEmit(['countChange'])

使定 defineEmit() 定義 emit()，傳入為 Array，可同時定義多個 event。

10 行

ref: innerCount = props.startCount

使用 ref sugar 建立 outerCount state，並將 startCount prop 值指定到 innerCount state。

12 行

let onClick = () => {
  innerCount++
  emit('countChange', innerCount)
}

當按下 + button 時，除了 innerCount 遞增外，還將 innerCount 內部值透過 emit() 發出 countChange event 傳出去。

Point-free

結果不變，但使用 Point-free 改寫。

App.vue

<template>
MyCounter(:startCount='outerCount', @countChange='onCountChange')
span {{ outerCount }}
</template>

<script setup>
import { ref } from 'vue'
import { write } from 'vue3-fp'
import { pipe } from 'ramda'
import MyCounter from '/src/components/MyCounter.vue'

let outerCount = ref(3)

let onCountChange = pipe(
  write(outerCount)
)
</script>

14 行

let onCountChange = pipe(
  write(outerCount)
)

使用 pipe() 組合 onCountChange()：

  • write(outerCount)：將接收值寫入 outerCount state

MyCounter.vue

<template lang='pug'>
button(@click='onClick') +
</template>

<script setup>
import { ref, defineProps, defineEmit } from 'vue'
import { read, write, mark, eject } from 'vue3-fp'
import { pipe, inc } from 'ramda'

let props = defineProps({ startCount: 0 })
let emit = defineEmit(['countChange'])
let innerCount = ref(0)

let onClick = pipe(
  read(innerCount),
  inc,
  write(innerCount),
  eject(emit)('countChange')
)

pipe(
  mark('startCount', props),
  write(innerCount)
)()
</script>

14 行

let onClick = pipe(
  read(innerCount),
  inc,
  write(innerCount),
  eject(emit)('countChange')
)

使用 pipe() 組合 onClick()：

  • read(innerCount)：讀取 innerCount state
  • inc：使值遞增
  • write(innerCount)：寫入 innerCount state
  • eject(emit)('countChange')：使用 eject() 將 emit 轉成 curry function 發出 countChange event

21 行

pipe(
  mark('startCount', props),
  write(innerCount)
)()

使用 pipe() 組合 IIFE：

  • mark('startCount', props)：讀取 startCount prop
  • write(innerCount)：寫入 innerCount state

Conclusion

  • 使用 defineEmit() higher order function 建立 emit()
  • emit() 不方便 Point-free，使用 eject() 轉成 curry function 才能 Point-free
