Vue 3 之 onmounted()

2020-08-16

mounted hook 在 Vue 很常用，由於要透過 this 讀取 Model，只能使用 Function Expression，Vue 3 則可使用 unmounted() 將 Arrow Function 掛在 mounted hook 上。

Version

macOS Catalina 10.15.6
WebStorm 2020.2
Vue 3.0.0
Vue-fp 0.0.3

onMounted()

在 mounted hook 讀取 totalPrice reactive model 並增加 20。

<template>
  {{ totalPrice }}
</template>

<script>
import { onMounted, ref } from 'vue'

let totalPrice = ref(100)

let mount = () => totalPrice.value = totalPrice.value + 20

export default {
  name:'App',
  setup: () => {
    onMounted(mount)
    return { totalPrice }
  }
}
</script>

第 1 行

<template>
  {{ totalPrice }}
</template>

顯示 totalPrice reactive model。

第 6 行

import { onMounted, ref } from 'vue'

引用 onMounted() composition API。

14 行

setup: () => {
  onMounted(mount)
  return { totalPrice }
}

在 setup() 中使用 onMounted() 將 mount() 掛到 mounted hook。

onMounted()
(() -> *) -> void
將 arrow function 掛到 mounted hook

() -> *：一般 arrow function

void：不回傳值

第 10 行

let mount = () => totalPrice.value = totalPrice.value + 20

mount() 為 arrow function，以 .value 讀取 totalPrice reactive model，增加 20 後又寫入 totalPrice.value。

Function Pipeline

<template>
  {{ totalPrice }}
</template>

<script>
import { onMounted, ref } from 'vue'
import { pipe, add, always } from 'ramda'
import { unwrap, effect } from 'vue3-fp'

let totalPrice = ref(100)

let mount = pipe(
  unwrap(totalPrice),
  add(20),
  effect(totalPrice)
)

let setup = pipe(
  always(mount),
  onMounted,
  always({ totalPrice })
)

export default {
  name:'App',
  setup
}
</script>

18 行

let setup = pipe(
  always(mount),
  onMounted,
  always({ totalPrice })
)
  • 使用 always() 組合出 () => mount 傳入 onMounted() 掛到 mounted hook
  • 使用 always() 組合出 () => ({ totalPrice }) 回傳

12 行

let mount = pipe(
  unwrap(totalPrice),
  add(20),
  effect(totalPrice)
)
  • 使用 unwrap() 讀取 totalPrice reactive model
  • 使用 add() 增加 20
  • 使用 effect() 寫入 totalPrice reactive model

Example

來看看更接近現實的應用。

<template>
  {{ totalPrice }}
</template>

<script>
import { ref, onMounted } from 'vue'
import { pipe, filter, pluck, sum, always } from 'ramda'
import { propGte } from 'wink-fp'
import { effect } from 'vue3-fp'

let data = [
  { title: 'FP in JavaScript', price: 100 },
  { title: 'RxJS in Action', price: 200 },
  { title: 'Speaking JavaScript', price: 300 }
]

let totalPrice = ref(0)

let mount = pipe(
  always(data),
  filter(propGte('price', 200)),
  pluck('price'),
  sum,
  effect(totalPrice)
)

let setup = pipe(
  always(mount),
  onMounted,
  always({ totalPrice })
)

export default {
  name:'App',
  setup
}
</script>

11 行

let data = [
  { title: 'FP in JavaScript', price: 100 },
  { title: 'RxJS in Action', price: 200 },
  { title: 'Speaking JavaScript', price: 300 }
]

希望在 mounted 內對 data 做處理，然後顯示 totalPrice。

19 行

let mount = pipe(
  always(data),
  filter(propGte('price', 200)),
  pluck('price'),
  sum,
  effect(totalPrice)
)
  • 使用 always() 讀取 data
  • 使用 filter() 過濾 price 大於等於 200 所有資料
  • 使用 pluck() 取出 price property
  • 使用 sum() 加總
  • 使用 effect() 寫入 totalPrice reactive model

可發現 mount() 全部以 point-free 組合而成

Conclusion

  • Vue 2 由於要在 mounted hook 使用 this 讀寫 model，因此只能使用 function expression，也無法透過 Function Pipeline 組合出 mounted()，但這在 Vue 3 則完全改變，更可完全以 point-free 組合出來
