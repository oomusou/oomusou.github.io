使用 computed() 將 Object 轉成 Computed Setter

2020-11-05

Computed 除了是 Function 外，也可以是 Object，此時 Computed 就有了 Setter，Vue 3 一樣可使用 computed() 將 Object 轉成 Computed。

Version

Vue 3.0.0

Composition API

當 <input> 內容動態改變時，右方顯示也會動態更新。

<template>
  <input type="text" v-model="fullName">
  {{ fullName }}
</template>

<script setup>
import { ref, computed } from 'vue'

export let firstName = ref('Sam')
export let lastName = ref('Xiao')

export let fullName = computed({
  get: () => `${firstName.value} ${lastName.value}`,
  set: v => [firstName.value, lastName.value] = v.split(' ')
})
</script>

第 1 行

<template>
  <input type="text" v-model="fullName">
  {{ fullName }}
</template>

<input> 的 v-model 為 fullName，但 fullName 為 computed，由於 fullName 同時需要讀寫，因此必須使用 computed setter。

第 9 行

export let firstName = ref('Sam')
export let lastName = ref('Xiao')

建立 firstName 與 lastName state。

12 行

export let fullName = computed({
  get: () => `${firstName.value} ${lastName.value}`,
  set: v => [firstName.value, lastName.value] = v.split(' ')
})

Computed setter 為 Object，先建立 getter 與 setter function，最後使用 computed() 將 Object 轉成 computed。

computed()
{ get: () -> a, set: a -> () } -> Ref a
將 Object 轉成 computed

get : () -> a：提供 computed getter

set: a -> ()：提供 computed setter

Point-free

<template>
  <input type="text" v-model="fullName">
  {{ fullName }}
</template>

<script setup>
import { computed, ref } from 'vue'
import { head, last, pipe, split } from 'ramda'
import { formatN, seq } from 'wink-fp'
import { read, write } from 'vue3-fp'

let firstName = ref('Sam')
let lastName = ref('Xiao')

let setFirstName = pipe(
  head,
  write(firstName)
)

let setLastName = pipe(
  last,
  write(lastName)
)

let getFullName = pipe(
  read(lastName),
  formatN(2)('{0} {1}', read(firstName))
)

let setFullName = pipe(
  split(' '),
  seq(setFirstName, setLastName)
)

export let fullName = computed({ 
  get: getFullName, 
  set: setFullName 
})
</script>

35 行

export let fullName = computed({ 
  get: getFullName, 
  set: setFullName 
})

組合 fullName computed，它是由 getFullName getter 與 setFullName setter 組合而成。

25 行

let getFullName = pipe(
  read(lastName),
  formatN(2)('{0} {1}', read(firstName))
)

組合 computed getter：

  • 使用 read() 讀取 firstName 與 lastName state
  • 使用 formatN() 組合 String 回傳

20 行

let setFullName = pipe(
  split(' '),
  seq(setFirstName, setLastName)
)

組合 computed setter：

  • 使用 split() 將 argument 分解成 Array
  • 使用 seq() 兵分兩路寫入 firstName 與 lastName state

15 行

let setFirstName = pipe(
  head,
  write(firstName)
)

組合寫入 firstName state：

  • 使用 head() 讀取 Array 中第一筆
  • 使用 write() 寫入 firstName state

20 行

let setLastName = pipe(
  last,
  write(lastName)
)

組合寫入 lastName state：

  • 使用 last() 讀取 Array 中最後一筆
  • 使用 write() 寫入 lastName state

Conclusion

  • computed() 除了能接受 arrow function 外，也能接受 Object 組合 computed setter
  • Point-free 寫法雖然行數較多，但每一行都是 pure function 無 side effect，且職責明確
