使用 computed() 將 Function 轉成 Computed

2020-09-27

Computed 本質為 Function，理論上可由 Function Pipeline 與 Point-free 組合而成，但因為 Vue 2 使用 this 讀取 Model，因此無法使用，但 Vue 3 不需要 this，且只要將 Arrow Function 透過 computed() 轉成 Computed 即可。

Version

Vue 3.0.0

Composition API

Hello Sam 為 computed，而非 reactive reference。

<template>
  {{ greeting }}
</template>

<script setup>
import { ref, computed } from 'vue'

// name :: Ref String
export let name = ref('Sam')

// greeting :: Readonly Ref ReadOnly
export let greeting = computed(() => `Hello ${name.value}`)
</script>

第 6 行

import { ref, computed } from 'vue'

引用 computed() reactivity API 建立 computed。

第 8 行

// name :: Ref String
export let name = ref('Sam')

使用 ref() 建立 name reactive reference。

11 行 

// greeting :: Readonly Ref ReadOnly
export let greeting = computed(() => `Hello ${name.value}`)

Computed 本質為 function，先由 arrow function 建立，再由 computed() higher order function 將 arrow function 轉為 computed。

computed()
(() -> a) -> Readonly Ref Readonly a
將 arrow function 轉成 computed

() -> a：輸入為沒有任何 argument 的 arrow function

Readonly Ref Readonly a：回傳 computed

Point-free

<template>
  {{ greeting }}
</template>

<script setup>
import { computed, ref } from 'vue'
import { pipe } from 'ramda'
import { formatN } from 'wink-fp'
import { read } from 'vue3-fp'

// name :: Ref String
export let name = ref('Sam')

// formatName :: () -> ()
let formatName = pipe(
  read(name),
  formatN(2)('{0} {1}', 'Hello'),
)

// greeting :: Readonly Ref Readonly
export let greeting = computed(formatName)
</script>

14 行

// formatName :: () -> ()
let formatName = pipe(
  read(name),
  formatN(2)('{0} {1}', 'Hello'),
)
  • 使用 read() 讀取 name reactive reference
  • 使用 formatN() 實現 template string

20 行

// greeting :: Readonly Ref Readonly
export let greeting = computed(formatName)

使用 computed() 將普通 function 轉成 computed。

Conclusion

  • Computed 本質是 function，由於 Vue 3 不使用 this，可先使用 arrow function 實現 presentation logic，再使用新的 computed() 將 arrow function 轉成 computed
  • Vue 2 並非完全不能使用 Function Pipeline 組合 computed，只有在 computed 搭配 Vuex 的 state 才能使用，但因為 Vue 3 完全不使用 this，可完全使用 pipe() 組合 pure function
