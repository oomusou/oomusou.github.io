將 Prop 傳給 Computed 使內外連動

2020-11-05

雖然可將 Prop 傳給 State，但這會使其喪失與 Prop 連動可能，若需求是 Component 內部需與 Prop 連動，則要改傳給 Computed。

Version

Vue 3.0.0

Composition API

外部 counter 不斷遞增，且內部 component 的 counter 會連動 + 1。

App.vue

<template>
  <div>{{ outerCount }} <button @click="onClick">+</button></div>
  <MyCounter :start-count="outerCount"></MyCounter>
</template>

<script setup>
import { ref } from 'vue'
export { default as MyCounter } from '@/components/MyCounter'

export let outerCount = ref(3)

export let onClick = () => outerCount.value++
</script>

第 8 行

export { default as MyCounter } from '@/components/MyCounter'

直接將 MyCounter export 出去用於 HTML template 上。

10 行

export let outerCount = ref(3)

宣告 outerCount state，且初始值為 3。

12 行

export let onClick = () => outerCount.value++

當按下 + 時對 outerCount state 遞增。

MyCounter.vue

<template>
  {{ innerCount }}
</template>

<script setup="props">
import { computed } from 'vue'

export let innerCount = computed(() => props.startCount + 1)

export default {
  props: [
    'startCount'
  ]
}
</script>

第 8 行

export let innerCount = computed(() => props.startCount + 1)

直接透過 props 讀取 startCount 遞增後，傳給 computed() 建立 innerCount computed。

Point-free

App.vue

<template>
  <div>{{ outerCount }} <button @click="onClick">+</button></div>
  <MyCounter :start-count="outerCount"></MyCounter>
</template>

<script setup>
import { ref } from 'vue'
import { pipe, inc } from 'ramda'
import { read, write } from 'vue3-fp'
export { default as MyCounter } from '@/components/MyCounter'

export let outerCount = ref(3)

export let onClick = pipe(
  read(outerCount),
  inc,
  write(outerCount)
)
</script>

14 行

export let onClick = pipe(
  read(outerCount),
  inc,
  write(outerCount)
)
  • 使用 read() 讀取 outerCount state
  • 使用 inc() 遞增
  • 使用 write() 寫入 innerCount state

MyCounter.vue

<template>
  {{ innerCount }}
</template>

<script setup="props">
import { pipe, inc } from 'ramda'
import { computed } from 'vue'
import { mark } from 'vue3-fp'

let incCount = pipe(
  mark('startCount', props),
  inc
)  

export let innerCount = computed(incCount)

export default {
  props: [
    'startCount'
  ],
}
</script>

10 行

let incCount = pipe(
  mark('startCount', props),
  inc
)
  • 使用 mark() 讀取 startCount prop
  • 使用 inc() 遞增
  • 最後傳入 computed() 建立 innerCount computed

15 行

export let innerCount = computed(incCount)

使用 computed() 將普通 function 轉成 computed。

Conclusion

  • 由於 props 為 read only，若要對 prop 修改，可改傳給 state 或 computed，唯兩者特性不太相同，需視需求而決定該用何者
