使 Prop 也能 Reactive

2021-04-14

若 Prop 只用來對 Component 傳入初始值設定，只要在 mounted hook 接受並寫入 data 即可；但若日後還會將改變 Data 傳入 Prop，且希望 Component 能 Reactive 根據 Prop 改變，則要搭配 Computed 或 Watch。

Version

Vue 3.0.5

Computed

+ 不斷遞增 component 外部的 state，並以 prop 傳入 component 並 reactive 顯示。

Composition API

App.vue

<template>
  <button @click="onClick">+</button>
  <MyCounter :startCount="count"/>
</template>

<script setup>
import MyCounter from '/src/components/MyCounter.vue'

ref: count = 5

let onClick = () => count++
</script>

第 2 行

<button @click="onClick">+</button>
<MyCounter :startCount="count"/>
  • 使用 + button 對 count state 遞增
  • 並將 count state 傳入 <MyCounter> 的 startCount prop

第 7 行

import MyCounter from '/src/components/MyCounter.vue'

使用 import 引用 MyCounter。

第 9 行

ref: count = 5

定義 count state 並設定初始值為 5。

11 行

let onClick = () => count++

當 + button 按下時將執行 onClick() 對 count state 遞增。

MyCounter.vue

<template>
  <div>{{ count }}</div>
</template>

<script setup="props">
import { defineProps, computed } from 'vue'

let props = defineProps({
  startCount: 0
})

let count = computed(() => props.startCount)
</script>

第 2 行

<div>{{ count }}</div>

以 reactive 顯示 count computed。

第 8 行

let props = defineProps({
  startCount: 0
})

使用 defineProps() 定義 startCount prop。

12 行

let count = computed(() => props.startCount)

使用 computed() 根據 startCount prop 定義 count computed。

Function Pipeline

App.vue

<template>
  <button @click="onClick">+</button>
  <MyCounter :startCount="count"/>
</template>

<script setup>
import MyCounter from '/src/components/MyCounter.vue'
import { ref } from 'vue'
import { read, write } from 'vue3-fp'
import { pipe, inc } from 'ramda'

let count = ref(5)

let onClick = pipe(
  read(count),
  inc,
  write(count)
)
</script>

12 行

let count = ref(5)

使用 ref() 定義 count state 對 Function Pipeline 較方便。

14 行

let onClick = pipe(
  read(count),
  inc,
  write(count)
)

組合出 onClick()：

  • read()：讀取 count state
  • inc()：對值遞增
  • write()：寫入 count state

MyCounter.vue

<template>
  <div>{{ count }}</div>
</template>

<script setup="props">
import { defineProps } from 'vue'
import { mark, calculated } from 'vue3-fp'

let props = defineProps({
  startCount: 0
})

let count = calculated(
  mark('startCount', props)
)
</script>

13 行

let count = calculated(
  mark('startCount', props)
)

組合出 count computed：

  • mark()：讀取 startCount prop
  • calculated()：建立 count computed

為了讓 prop 能 reactive，最簡單的方式是使用 computed

Watch

功能不變，但使用 watch 改寫。

Composition API

App.vue

<template>
  <button @click="onClick">+</button>
  <MyCounter :startCount="count"/>
</template>

<script setup>
import MyCounter from '/src/components/MyCounter.vue'

ref: count = 5

let onClick = () => count++
</script>

App.vue 部分不變。

MyCounter.vue

<template>
  <div>{{ count }}</div>
</template>

<script setup="props">
import { defineProps, watch } from 'vue'

let props = defineProps({
  startCount: 0
})

ref: count = props.startCount

watch(() => props.startCount, x => count = x)
</script>

直接使用 computed 接 prop 雖然可行，但其缺點是不能有 state。

12 行

ref: count = props.startCount

直接使用 startCount state 定義 count state。

14 行

watch(() => props.startCount, x => count = x)

使用 watch() 定義 startCount prop 的 watch，一旦日後 prop 改變，將寫入 count state。

Function Pipeline

App.vue

<template>
  <button @click="onClick">+</button>
  <MyCounter :startCount="count"/>
</template>

<script setup>
import MyCounter from '/src/components/MyCounter.vue'
import { ref } from 'vue'
import { read, write } from 'vue3-fp'
import { pipe, inc } from 'ramda'

let count = ref(5)

let onClick = pipe(
  read(count),
  inc,
  write(count)
)
</script>

App.vue 部分不變。

MyCounter.vue

<template>
  <div>{{ count }}</div>
</template>

<script setup="props">
import { ref, defineProps, watch } from 'vue'
import { write, mark } from 'vue3-fp'
import { pipe } from 'ramda'

let props = defineProps({
  startCount: 0
})

let count = ref(0)

pipe(
  mark('startCount', props),
  write(count)
)()

watch(
  mark('startCount', props),
  write(count)
)
</script>

14 行

let count = ref(0)

使用 ref() 定義 count state 對 Function Pipeline 較方便。

16 行

pipe(
  mark('startCount', props),
  write(count)
)()

組合出 IIFE：

  • mark()：讀取 startCount prop
  • write()：寫入 count state

21 行

watch(
  mark('startCount', props),
  write(count)
)

組合出 startCount prop 的 watch：

  • mark()：讀取 startCount prop
  • write()：寫入 count state

若要使用 state，則要先將 prop 初始值給 data，再對 prop 進行 watch

immediate: true

功能不變，但使用 { immediate: true } 改寫。

Composition API

App.vue

<template>
  <button @click="onClick">+</button>
  <MyCounter :startCount="count"/>
</template>

<script setup>
import MyCounter from '/src/components/MyCounter.vue'

ref: count = 5

let onClick = () => count++
</script>

App.vue 完全不變。

MyCounter.vue

<template>
  <div>{{ count }}</div>
</template>

<script setup="props">
import { defineProps, watch } from 'vue'

let props = defineProps({
  startCount: 0
})

ref: count = 0

watch(
  () => props.startCount,
  x => count = x,
  { immediate: true }
)
</script>

14 行

watch(
  () => props.startCount,
  x => count = x,
  { immediate: true }
)

若覺得要先由 prop 定義 count state 初始值麻煩，也可以在 watch 的最後一個 argument 直接傳入 immediate 為 true，如此會一併定義 count state 初始值。

Function Pipeline

App.vue

<template>
  <button @click="onClick">+</button>
  <MyCounter :startCount="count"/>
</template>

<script setup>
import MyCounter from '/src/components/MyCounter.vue'
import { ref } from 'vue'
import { read, write } from 'vue3-fp'
import { pipe, inc } from 'ramda'

let count = ref(5)

let onClick = pipe(
  read(count),
  inc,
  write(count)
)
</script>

App.vue 部分不變。

MyCounter.vue

<template>
  <div>{{ count }}</div>
</template>

<script setup="props">
import { ref, defineProps, watch } from 'vue'
import { write, mark } from 'vue3-fp'

let props = defineProps({
  startCount: 0
})

let count = ref(0)

watch(
  mark('startCount', props),
  write(count),
  { immediate: true }
)
</script>

11 行

watch(
  mark('startCount', props),
  write(count),
  { immediate: true }
)

組合出 startCount prop 的 watch：

  • mark()：讀取 startCount prop
  • write()：寫入 count state
  • 在 watch() 最後一個 argument 傳入 { immediate: true } 直接定義 count state 初始值

Conclusion

  • 若要 prop 能 reactive，最簡單方式是使用 computed，唯沒有 data 可用
  • 若要有 data 也能 reactive， 則要對 prop 使用 watch，先以 prop 定義 data 初始值，再由 watch 將日後改變的 prop 修改 data
  • 若想一次由 prop 定義 data 初始值，也可直接在 watch 內傳入 { immediate: true }
