使用 Default Prop 設定型別

2020-11-05

Vue 的 State 可提供預設值代表型別方便日後維護，事實上 Component 的 Prop 也能提供預設值代表型別。

Version

Vue 3.0.0

Primitive

+ 在外層遞增 count state，並以 prop 傳進內層 component，會即時根據 prop 改變而顯示最新 state。

App.vue

<template>
  <MyCounter :start-count="count"/>
  <button @click="onClick">+</button>
</template>

<script setup>
import { pipe, inc } from 'ramda'
import { ref } from 'vue'
import { read, write } from 'vue3-fp'
export { default as MyCounter } from '@/components/MyCounter'

export let count = ref(8)

export let onClick = pipe(
  read(count),
  inc,
  write(count)
)
</script>

12 行

export let count = ref(8)

定義 count state，並設定其預設值為 8，可由其預設值得知 count state 型別為 Number。

MyCounter.vue

<template>
  <div>{{ count }}</div>
</template>

<script setup="props">
import { calculated, mark } from 'vue3-fp'

export let count = calculated(
  mark('startCount', props)
)

export default {
  props: {
    startCount: { default: 0 }
  },
}
</script>

13 行

props: {
  startCount: { default: 0 }
},

定義 startCount prop，並以 default 定義其預設值為 0，可由其預設值得知 startCount prop 型別為 Number。

Array

Add book 在外層新增 books state，並以 prop 傳進內層 component，會即時根據 prop 改變而顯示最新 state。

App.vue

<template>
  <BookShelf :books="books"/>
  <button @click="onClick">Add book</button>
</template>

<script setup>
import { append, pipe } from 'ramda'
import { ref } from 'vue'
import { read, write } from 'vue3-fp'
export { default as BookShelf } from '@/components/BookShelf'

export let books = ref([
  { title: 'FP in JavaScript', price: 100 },
  { title: 'RxJS in Action', price: 200 },
  { title: 'Speaking JavaScript', price: 300 }
])

export let onClick = pipe(
  read(books),
  append({ title: 'Programming Haskell', price: 400 }),
  write(books)
)
</script>

12 行

export let books = ref([
  { title: 'FP in JavaScript', price: 100 },
  { title: 'RxJS in Action', price: 200 },
  { title: 'Speaking JavaScript', price: 300 }
])

定義 books state 並提供其初始值。

BookShelf.vue

<template>
  <ul>
    <li v-for="(x, i) in shelf" :key="i">
      {{ x.title }} | {{  x.price }}
    </li>
  </ul>
</template>

<script setup="props">
import { always } from 'ramda'
import { calculated, mark } from 'vue3-fp'

export let shelf = calculated(
  mark('books', props)
)

export default {
  props: {
    books: { default: always([]) }
  }
}
</script>

18 行

props: {
  books: { default: always([]) }
}

定義 books prop，並以 default 定義其預設值為 [] Empty Array，可由其預設值得知 books prop 型別為 Array。

與 primitive 不同的是 default 不能直接回傳 []，而要回傳 arrow function，並由 function 回傳 []。

Conclusion

  • Prop 若使用 Array 表示，將無法提供任何型別資訊，較不利於日後維護，可學習 data 以預設值提供型別技巧，讓 prop 也能以預設值表示型別
  • Array 的預設值比較特別，須以 function 提供預設值 []
