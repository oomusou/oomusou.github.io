Tailwind CSS 與 Class Binding

2020-10-18

Tailwind 的 Utility 其實就是一堆預先定義好的 CSS Class，若要能根據 State 動態改變 Utility 使其 Reactive，就要使用 Vue 的 Class Binding。

Version

Vue 3.0.0
Tailwind CSS 1.9.0

String

Tailwind Rocks 使用 Tailwind 的 text-red-600 utility 成為紅色。

<template>
  <div :class="fontColor">Tailwind Rocks</div>
</template>

<script setup>
export let fontColor = 'text-red-600'
</script>

第 2 行

<div :class="fontColor">Tailwind Rocks</div>

並沒有在 class attribute 直接使用 text-red-600，而是綁定到 fontColor state。

第 6 行

export let fontColor = 'text-red-600'

將 fontColor state 指定為 text-red-600。

Object

<template>
  <div :class="{ 'text-red-600': true }">Tailwind Rocks</div>
</template>

第 2 行

<div :class="{ 'text-red-600': true }">Tailwind Rocks</div>

將 Object 直接綁定到 class attribute，key 為 Tailwind utility，value 為 true 則表示套用該 utility，false 則為否。

Binding to Property

<template>
  <div :class="{ 'text-red-600': isActive }">Tailwind Rocks</div>
</template>

<script setup>
import { ref } from 'vue'

export let isActive = ref(true)
</script>

第 2 行

<div :class="{ 'text-red-600': isActive }">Tailwind Rocks</div>

Object 的 value 為 isActive state。

第 8 行

export let isActive = ref(true)

true 與 false 由 isActive state 決定。

Binding to Object

<template>
  <div :class="fontColor">Tailwind Rocks</div>
</template>

<script setup>
import { ref } from 'vue'

export let fontColor = ref({ 'text-red-600': true })
</script>

第 2 行

<div :class="fontColor">Tailwind Rocks</div>

也可整個 Object state 與 class attribute 綁定。

第 8 行

export let fontColor = ref({ 'text-red-600': true })

fontColor state 為 Object。

Binding to Computed

<template>
  <div :class="changeFontColor">Tailwind Rocks</div>
</template>

<script setup>
import { computed } from 'vue'

export let changeFontColor = computed(() => ({ 'text-red-600': true }))
</script>

第 2 行

<div :class="changeFontColor">Tailwind Rocks</div>

若要最靈活，可使用 class binding 直接綁定 computed。

第 8 行

export let changeFontColor = computed(() => ({ 'text-red-600': true }))

Computed 唯一要求是要回傳 Object。

Point-free

<template>
  <div :class="changeFontColor">Tailwind Rocks</div>
</template>

<script setup>
import { objOf, T } from 'ramda'
import { calculated } from 'vue3-fp'

export let changeFontColor = calculated(
  T,
  objOf('text-red-600')
)
</script>

第 9 行

export let changeFontColor = computed(pipe(
  T,
  objOf('text-red-600')
))

既然 changeFontColor 為 function，就可使用 Ramda 組合使其 point-free。

Array

也可將 class 綁定到 Array。

<template>
  <div :class="[fontSize, fontColor]">Tailwind Rocks</div>
</template>

<script setup>
import { ref } from 'vue'

export let fontSize = ref('text-3xl')
export let fontColor = ref('text-red-600')
</script>

第 2 行

<div :class="[fontSize, fontColor]">Tailwind Rocks</div>

若要同時具備多個 utility，可將眾多 state 以 Array 組合綁定。

第 8 行

export let fontSize = ref('text-3xl')
export let fontColor = ref('text-red-600')

State 提供 utility。

Binding to Object Array

也可以使用 Object Array。

<template>
  <div :class="[{ fontSize: isFontSizeActive }, fontColor]">
    Tailwind Rocks
  </div>
</template>

<script setup>
import { ref } from 'vue'

export let fontSize = ref('text-3xl')
export let fontColor = ref('text-red-600')
export let isFontSizeActive = ref(false)
</script>

第 2 行

<div :class="[{ fontSize: isFontSizeActive }, fontColor]">
  Tailwind Rocks
</div>

若並不是所有 utility 都要使用，則可使用 Object Array，使用 property 的 value 決定是否 enable。

第 10 行

export let fontSize = ref('text-3xl')
export let fontColor = ref('text-red-600')
export let isFontSizeActive = ref(false)

另外提供 isFontSizeActive 決定是否啟用 text-red-600 utility。

Binding to Computed

<template>
  <div :class="changeFontStyle">Tailwind Rocks</div>
</template>

<script setup>
import { computed } from 'vue'

export let changeFontStyle = computed(() => ['text-3xl', 'text-red-600'])
</script>

第 2 行

<div :class="changeFontStyle">Tailwind Rocks</div>

最靈活依然是綁定 computed。

第 8 行

export let changeFontStyle = computed(() => ['text-3xl', 'text-red-600'])

只要該 computed 回傳 Array 即可。

Point-free

<template>
  <div :class="changeFontStyle">Tailwind Rocks</div>
</template>

<script setup>
import { always, split } from 'ramda'
import { calculated } from 'vue3-fp'

export let changeFontStyle = calculated(
  always('text-3xl, text-red-600'),
  split(',')
)
</script>

第 9 行

export let changeFontStyle = calculated(
  always('text-3xl, text-red-600'),
  split(',')
)

既然 changeFontStyle 為 function，就可使用 calculated() 與 Ramda 的 pure function 組合出 computed。

Conclusion

  • CSS binding 可使用 Object 或 Array，亦可使用 computed，只要能回傳 Object 或 Array 即可
  • 若要套用多 utility，除了使用 Array 外，也能使用 Object Array
