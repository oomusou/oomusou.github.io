由 Vue 改變 CSS Local Variable

2020-12-30

若有數值在 CSS 重複出現，我們可將其抽成 CSS Variable，將來只需維護一處即可，若能將 JavaScript Variable 與 CSS Variable 綁定，則可使用 JavaScript 動態改變 CSS 顯示。

Version

Vue 3.0.0

Background Color

<div> 的背景色為 紅色。

<template>
  <div class="bar"/>
</template>

<style scoped>
.bar {
  height: 30px;
  background-color: #f00;
}
</style>
  • background-color: #f000：設定 <div> 的背景為紅色

CSS Variable

<div> 背景一樣是紅色，但改用 CSS local variable。

<template>
  <div class="bar"/>
</template>

<style scoped>
.bar {
  height: 30px;
  --bgColor: #ff0000;
  background-color: var(--bgColor);
}
</style>
  • 以 -- 設定 bgColor variable
  • 並以 var() 使用 --bgColor

目前 --bgColor 只是 CSS scope 的 variable，JavaScript 並無法與之連動

Variable Binding

<div> 背景一樣是紅色，但改用 Vue 的 variable binding。

<template>
  <div class="bar"/>
</template>

<script setup>
import { ref } from 'vue'

export let bgColor = ref('#f00')
</script>

<style vars="{ bgColor }">
.bar {
  height: 30px;
  background-color: var(--bgColor);
}
</style>

11 行

<style vars="{ bgColor }">
.bar {
  height: 30px;
  background-color: var(--bgColor);
}
</style>

CSS 只使用了 var(--bgColor)，並沒有定義其值為 #f00。

在 <style> 使用 vars attribute，並直接 destructure 出 bgColor，此即為 CSS variable。

第 8 行

export let bgColor = ref('#f00')

在 Vue 定義 bgColor state 為 #f00。

第 2 行

<div class="bar"/>

HTML 不需如 Vue 2 需使用 style binding 與 computed 才能綁定 JavaScript variable 與 CSS variable。

Change Color

Vue 綁定 CSS variable 後，最大的價值在於可以使用 JavaScript 改變之。

按下 Change Color 後，<div> 背景色從 紅色 變成 綠色。

<template>
  <div class="bar"/>
  <button @click="onClick">Change Color</button>
</template>

<script setup>
import { ref } from 'vue'

export let bgColor = ref('#f00')

export let onClick = () => bgColor.value = '#0f0'
</script>

<style vars="{ bgColor }">
.bar {
  height: 30px;
  background-color: var(--bgColor);
}
</style>

11 行

export let onClick = () => bgColor.value = '#0f0'

在 Vue 的 bgColor data 改成 綠色，則 CSS variable 也會同步改成 綠色。

如此就能以 JavaScript 邏輯改變 CSS variable，不再如以往 CSS variable 只存在於 CSS scope，而無法與 JavaScript 連動

Point-free

使用 Point-free 重構。

<template>
  <div class="bar"/>
  <button @click="onClick">Change Color</button>
</template>

<script setup>
import { pipe, always } from 'ramda'
import { ref } from 'vue'
import { write } from 'vue3-fp'

export let bgColor = ref('#f00')

export let onClick = pipe(
  always('#0f0'),
  write(bgColor),
)
</script>

<style vars="{ bgColor }">
.bar {
  height: 30px;
  background-color: var(--bgColor);
}
</style>

13 行

export let onClick = pipe(
  always('#0f0'),
  write(bgColor),
)
  • 使用 always() 將 #0f0 包成 function
  • 使用 write() 寫入 bgColor state

Conclusion

  • 可將 CSS variable 之初始值都寫在 Vue 的 data 統一管理，避免初始值因寫在 <style> 內而忘記

  • Vue 3 使得 JavaScript 與 CSS Variable 綁定變成異常簡單，HTML template 完全不變，不需使用 style binding 與 computed，只要在 <style> 加上 vars attribute，並從其 destructure 出 CSS variable 即可
