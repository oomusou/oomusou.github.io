讀取 RESTful API 的 GET 回傳資料

2020-11-06

讀取 RESTful API 的 GET 回傳資料在實務上一定會碰到，由於 Composition API 沒有使用 this，在 Vue 3 將有不同處理方式。

Version

Vue 3.0.0
Axios 0.20.0

Composition API

按下 Get Book 會從 API 獲得 title 與 price。

<template>
  <button @click="onClick">Get Book</button>
  <p></p>
  {{ title }} / {{ price }}
</template>

<script setup>
import { ref } from 'vue'
import axios from 'axios'

let fetchBook = x => axios.get(`http://localhost:3000/books/${x}`)
  .then(x => x.data)

export let title = ref('')
export let price = ref('')

export let onClick = () => {
  fetchBook(1)
    .then(x => (title.value = x.title, x))
    .then(x => (price.value = x.price, x))
    .catch(e => console.log(e))
}
</script>

11 行

let fetchBook = x => axios.get(`http://localhost:3000/books/${x}`)
  .then(x => x.data)

API function，使用 axios.get() 從 API 取得 data。

14 行

export let title = ref('')
export let price = ref('')

定義 title 與 price state，初始值皆為 Empty String。

17 行

export let onClick = () => {
  fetchBook(1)
    .then(x => (title.value = x.title, x))
    .then(x => (price.value = x.price, x))
    .catch(e => console.log(e))
}

由於 fetchBook() 回傳 Promise，因此使用 then() 寫入 side effect。

由於不必透過 this 讀取 title 與 price state，因此 onClick() 可安心使用 arrow function，不必再使用 function expression。

Point-free

<template>
  <button @click="onClick">Get Book</button>
  <p></p>
  {{ title }} / {{ price }}
</template>

<script setup>
import { ref } from 'vue'
import { effect } from 'vue3-fp'
import { pipe, andThen as then, otherwise, prop, thunkify } from 'ramda'
import { get } from 'axios-fp'
import { formatN, log } from 'wink-fp'

let fetchBook = pipe(
  formatN(1)('http://localhost:3000/books/{}'),
  get,
  then(prop('data'))
)

export let title = ref('')
export let price = ref('')

export let onClick = pipe(
  thunkify(fetchBook)(1),
  then(effect('title', title)),
  then(effect('price', price)),
  otherwise(log)
)
</script>

14 行

let fetchBook = pipe(
  formatN(1)('http://localhost:3000/books/{}'),
  get,
  then(prop('data'))
)
  • 使用 formatN() 組合 API URL
  • 使用 get() 讀取 API，為 axios.get() 的 free function 版本
  • 使用 prop() 讀取 data property

23 行

export let onClick = pipe(
  thunkify(fetchBook)(1),
  then(effect('title', title)),
  then(effect('price', price)),
  otherwise(log)
)

由於 onClick() 不必使用 this 寫入 title 與 price state，整個 function 甚至可以使用 Function Pipeline 透過 pure function 組合出來，且完全 point-free。

  • 使用 thukify() 將 fetchBook() 轉為回傳 Thunk 的 function
  • 使用 effect() 從 title property 取得 data 並寫入 title state

Conclusion

  • Vue 3 Composition API 由於沒有使用 this，使得 Function Pipeline 與 Point-free 成為可能，event handler 都可以直接以 pure function 組合出來
