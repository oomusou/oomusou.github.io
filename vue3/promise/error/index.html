使用 Promise 處理 Error Response

2021-05-21

RESTful API 除了回傳 200 HTTP Status 與內容外，也會回傳 Error Response 與 Data，Vue 該如何正確處理呢 ?

Version

Vue 3.0.11

Mock API

mock/getBook.js

假設 /book/:id 會回傳 HTTP status：400，且還包含 code 與 errMsg。

Composition API

順利讀出 HTTP status、code 與 errMsg。

16 行

  • 由 e.response.status 讀取 HTTP status
  • 由 e.response.data.code 讀取回傳 data 的 code
  • 由 e.response.data 讀取回傳 data 的 errMsg

Data 部分都在 e.response.data 底下

Point-free

結果不變，但使用 Point-free 改寫。

16 行

從 e 解構出 status、code 與 errMsg。

Conclusion

  • HTTP status 可從 e.response.status 讀取；而 data 則都在 e.response.data 之下
