Vue 2 與 Vue 3 主要差異

2024-01-24

Vue 3.0 在 Sep.18, 2020 發表，Vue 2.7 也在 Dec.31, 2023 正式終止支援，目前最新版本是 Vue 3.4，Vue 3 已經到了成熟期，API 與生態系都趨於穩定，是學習 Vue 3 的好時機。

Version

Vue 3.4

Vue 2 x Vue 3

  • Bundle Tool
    • Vue 2：Webpack
    • Vue 3：Vite
  • API
    • Vue 2：Options API (偏 OOP)
    • Vue 3：Composition API (偏 FP)
  • Reactivity
    • Vue 2：Object.defineProperty()
    • Vue 3：Reactive API (Proxy)
  • HTML Template
    • Vue 2：不變
    • Vue 3：不變
  • API Fetch
    • Vue 2：Axios
    • Vue 3：Axios
  • Routing
    • Vue 2：觀念相同
    • Vue 3：觀念相同 (Composition API)
  • State Management
    • Vue 2：Vuex
    • Vue 3：Pinia (Composition API)
