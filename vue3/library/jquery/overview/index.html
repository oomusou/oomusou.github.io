Vue + jQuery 初體驗

2021-01-23

雖然使用 jQuery 的機會不多，但還是可在 Vue 使用 jQuery。

Version

Vue 3.0.0
jQuery 3.5.1

Setup

$ yarn add jquery

使用 Yarn 安裝 jQuery。

jQuery

使用 jQuery 控制 CSS class 顯示 Hello jQuery。

<template>
  <div class="box">Hello jQuery</div>
</template>

<script setup>
import S from 'jquery'
import { onMounted } from 'vue'

let mount = () => S('.box').addClass('show')

onMounted(mount)
</script>

<style scoped>
.box {
  display: none;
}

.show {
  display: block;
}
</style>

第 ６行

import S from 'jquery'

引用 jQuery 且 alias 成 S。

Vue 3 會警告 $ 已在內部使用，因此只能改成 S

第 9 行

let mount = () => S('.box').addClass('show')

onMounted(mount)

在 mounted hook 中使用 jQuery 找到使用 .box class 的 element，並動態新增 show class 使其顯示。

Function Pipeline

<template>
  <div class="box">Hello jQuery</div>
</template>

<script setup>
import S from 'jquery'
import { pipe, always } from 'ramda'
import { onMounted } from 'vue'

let addClass = sel => x => S(sel).addClass(x)

let mount = pipe(
  always('show'),
  addClass('.box')
)

onMounted(mount)
</script>

<style scoped>
.box {
  display: none;
}

.show {
  display: block;
}
</style>

10 行

let addClass = sel => x => S(sel).addClass(x)

以 currying 形式準備 addClass() 準備 Function Pipeline。

12 行

let mount = pipe(
  always('show'),
  addClass('.box')
)

onMounted(mount)
  • 使用 always() 準備要傳入的 show class
  • 使用 addClass() 對 .box element 動態新增 CSS class

Conclusion

  • 在 mounted hook 下基本上 HTML 與 CSS 都已經載入，因此不必如傳統 jQuery 需寫在 $(document).ready() 或 $(function() {}) 內
