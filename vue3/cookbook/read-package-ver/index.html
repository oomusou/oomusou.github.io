如何讀取 package.json 的 Property？

2023-01-13

我們經常在 package.json 儲存版本資訊，該如何在 Vue 中顯示版本資訊呢？

Version

Vue 3.2

Vite

package.json

Line 2

  • version：在 package.json 中儲存版本資訊

vite.config.js

Line 9

  • APP_VERSION：在 defineConfig() 的 define 下定義自己的變數，並使用 process.env.npm_package_version 讀取 package.json 中的 version property

App.vue

  • 可直接在 Vue 中讀取 APP_VERSION 變數

ES Module

App.vue

Line 7

  • 亦可使用 import 將 package.json 載入並直接解構出 version

Conclusion

  • 本文方法只適合 Vite，不適用於 Webpack
