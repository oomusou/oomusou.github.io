使用 v-model 綁定 Textbox

2024-01-30

當我們由 HTML 輸入資料，並由 JavaScript 取得時，就要使用 v-model Directive 將 HTML Element 綁定到 Vue 的 State，如此當 HTML 輸入變動時，JavaScript 就會 Reactive 地跟著改變。

Version

Vue 3.4

Input Event

<template>
  <input :value="name" @input="onInput" />
  {{ name }}
</template>

<script setup>
import { ref } from 'vue'

let name = ref('')

let onInput = event => {
  name.value = event.target.value
}
</script>

Line 11

let onInput = event => {
  name.value = event.target.value
}
  • 若只有 ref() 的 state，首先必須由 input event 從 HTML 取得 Textbox 的值
  • event.target：取得觸發 input event 的 Textbox，由 .value 取得 Textbox 的值

v-model

<template>
  <input v-model="name" />
  {{ name }}
</template>

<script setup>
import { ref } from 'vue'

let name = ref('')
</script>

Line 2

<input v-model="name" />
  • v-model：直接將 <input> 綁定到 name state，v-model 是 two way binding，任何在 HTML 對 state 變動會連動 JavaScript，任何在 JavaScript 對 state 變動也會連動 HTML

Options API

<template>
  <input v-model="name" />
  {{ name }}
</template>

<script>
export default {
  data: () => ({
    name: ''
  })
}
</script>
  • 最後附上相同功能的 Options API 供參考
  • State 宣告在 data() function
  • v-model 使用上不變

Conclusion

  • 藉由 v-model directive，我們就不必使用 input event 從 HTML 取得職
  • Vue 使用 event 時機並不多 (如 click event)，若在 Vue 大量使用 HTML event (input event, change event…)，很可能是沒善用 Vue 所提供的 directive
