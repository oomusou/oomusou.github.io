丁寧体

  • N です (丁寧体現在肯定)
  • N ではありません (丁寧体現在否定)
  • N でした (丁寧体過去肯定)

  • N は N じゃない (普通体現在否定)

  • N は N じゃなかった (普通体過去否定)
  • N で、 V (接続)
