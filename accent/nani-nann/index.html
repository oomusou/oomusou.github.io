何(なに)、何(なん)

2024-01-20

何 可念做 なに、亦可念做 なん，當 何 後面接 な行、た行、だ行 或 量詞 時唸 なん，其他時候唸 なに，但 何で 為例外，可念成 なに 或 なん，但意義不一樣。

接続

  • 何(なん) + な行
  • 何(なん) + た行
  • 何(なん) + だ行
  • 何(なん) + 數量

な行

何の本だ？

是什麼書？

  • 何(なん) [1]：代 什麼
  • の：格助 以 名詞 形容 名詞
  • 本(ほん) [1]：名 書
  • だ：助動 是

た行

寝る前に、何と言う？

睡覺之前要說什麼？

  • 寝(ね)る [0]：動II 自動 睡覺
  • 前(まえ) [1]：名 前
  • に：格助 表 發生時間
  • 何(なん) [1]：代 什麼
  • と：格助 表 內容
  • 言(い)う [0]：動I 自他 說

だ行

それは何だ？

那是什麼？

  • それ [0]：代 那個
  • は：副助 表 主題
  • 何(なん) [1]：代 什麼
  • だ：助動 是

量詞

今何時ですか？

現在幾點？

  • 今(いま) [1]：名 現在
  • 何時(なんじ) [1]：名 何時
  • です [0]：助動 是
    • ですか [0]：疑問 是嗎？

其他

何を買う？

買什麼？

  • 何(なに) [1]：代 什麼？
  • を：格助 接受 買う 的 賓語
  • 買(か)う [0]：動I 他動 買

箱に何がありますか？

箱子裡有什麼？

  • 箱(はこ) [0]：名 箱子
  • に：格助 表 存在位置
  • 何(なに) [1]：代 什麼
  • が：格助 表 主語
  • ある [1]：動I 自動 辞書形 存在
    • あります [3]：ます形 存在
    • ありますか [3]：疑問 存在嗎？

将来、何になりたい？

將來你想成為什麼？

  • 将来(しょうらい) [1]：名 將來
  • 何(なに)：代 什麼
  • に：格助 表 變化結果
  • なる [1]：動I 自動 辞書形 變成
    • なります [3]：ます形 變成
    • なりたい [3]：想做 想變成

例外

何(なに)で学校に行きますか？

你用什麼方法去學校？

何 + で 為例外，で 雖然為 だ行，但 何 亦可念成 なに 或 なん

  • 何(なに) [1]：代 什麼
  • で：格助 表 方法
  • 学校(がっこう) [0]：名 學校
  • に：格助 表 目的地
  • 行(い)く [0]：動I 自動 辞書形 去
    • 行きます [3]：ます形 去
    • 行きますか [3]：疑問 去嗎？

何(なん)で学校に行きますか？

你為什麼去學校？

  • 何(なん) [1]：代 什麼
  • で：格助 表 原因
  • 学校(がっこう) [0]：名 學校
  • に：格助 表 目的地
  • 行(い)く [0]：動I 自動 辞書形 去
    • 行きます [3]：ます形 去
    • 行きますか [3]：疑問 去嗎？
