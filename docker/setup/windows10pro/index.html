如何安裝 Docker Desktop for Windows ?

2019-07-26

由於 Docker 在本質上仍是 Linux，只是多個 Container 跑在同一個 Linux 上，因此若要將 Docker 跑在 Windows，仍然需要一個 VM 在背景跑 Linux。Docker Desktop for Windws 內建支援 Windows 的 Hyper-V，只會在 Hyper-V 內跑一個極小的 Linux，如此其他 Container 就能跑在 Docker 上。

Version

Windows 10 Pro 1709 16299.371
Docker Desktop for Windows 18.03.0.ce (0520e24)

Docker for Windows

要能在 Windows 執行 Docker，首先必須安裝 Docker Desktop for Windows。

下載 Docker Desktop for Windows

  1. 到 Docker 官網
  2. 按 Get Docker 下載 Docker

執行 Installer

  1. 預設將 Docker for Windows Installer 下載到 Downloads 目錄
  2. 執行 Docker for Windows Installer

Configuration

  1. 可選擇是否在 desktop 增加 shortcut
  2. 不要勾選 Use Windows containers instead of Linux containers

Windows container 與 Linux container 正在整合中，同時只能 單獨使用 Windows container 或 Linux container，目前只有 Edge 版可以 同時使用 Windows container 與 Linux container

安裝 Docker for Windows 中。

按 Close and log out，即將登出 Windows 並重新登入

Hyper-V

  1. 重新登入後，Docker 發現 Hyper-V 沒有啟動而無法執行
  2. 按 Ok 後 Windows 將重新啟動並自動安裝 Hyper-V
$ docker -v

預設 Windows 10 每次啟動都會執行 Docker，在 Command Prompt 輸入 docker -v，若出現 Docker 版本編號，則表示 Docker 安裝成功。

Conclusion

  • 安裝 Docker for Windows 之後，就可在 terminal 執行 Docker command
