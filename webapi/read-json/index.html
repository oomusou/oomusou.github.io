Reading JSON File in the Browser

2022-11-15

If we use JSON files as configuration for our Web App, we can use fetch() or import to read JSON files in the Browser.

Version

Brave 1.45.123

JSON File

data.json

  • Standalone data.json file for configuration

  • data.json can be read as an Object

fetch()

Line 9

  • The JSON file can be read by fetch()
  • Since fetch() returns Promise, we can use Promise Chain to handle JSON files.

Line 9

  • Since fetch() returns Promise. We can also use async-await to handle Promise
  • Top-level await is not supported by Browser now; what we can do is use async IIFE to simulate top-level await

ES Module

Line 8

  • We have to add type="module" on <script> to enable ES module

Line 9

  • Use import statement to import the JSON file
  • Assert the file format is JSON type

Line 8

  • We have to add type="module" on <script> to enable ES module

Line 9

  • We can also use import() to dynamically import JSON file
  • import() returns Promise

Conclusion

  • We can use fetch() to read JSON files, but asynchronously
  • We can also use import to read JSON files synchronously or asynchronously
