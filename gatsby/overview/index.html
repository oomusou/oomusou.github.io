Gatsby 初體驗

2019-10-15

Gatsby 是以 React 為基礎的 JAMstack Framework，特色是以 GraphQL 為基礎，其 Ecosystem 有豐富的 Plugin 可用。

Version

macOS Catalina 10.15
Node 10.16.3
Gatsby CLI 2.8.1
Gatsby 2.4.4

Gatsby CLI

$ yarn global add gatsby-cli

使用 Yarn 安裝 Gatsby CLI。

Create New Project

$ gatsby new gatsby-hello

使用 Gatsby CLI 的 gatsby new 建立 Gatsby project。

可以發現 gatsby new 會從 https://github.com/gatsbjs/gatsby-starter-default.git 下載預設的 starter。

Run Project

$ cd gatsby-hello
$ gatsby develop

進入 gatsby-hello 目錄，使用 gatsby develop 執行 Gatsby。

Gatsby 將執行在 http://localhost:8000/。

Create New Project with Starter

$ gatsby new gatsby-material https://github.com/Vagr9K/gatsby-material-starter

gatsby new 除了可建立預設 project 外，也可根據指定 starter 建立 project。

根據 https://github.com/Vagr9K/gatsby-material-starter 建立 gatsby-material project，此為 Material-UI 的 starter。

Starter Library

可在 Gatsby 官網的 Starter Library 找到眾多 starter。

Conclusion

  • Gatsby 提供了專屬 CLI，除了可建立 project 外，也可根據 starter 建立 project
