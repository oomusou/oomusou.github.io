使用 PUT 修改資料

2021-10-26

若要修改資料，可使用 PUT 透過 Body 傳遞，並將 ID 在 URL 以 Params 表示。

Version

Express 4.17.1

app.put

第 8 行

  • 使用 PUT 修改資料時，id 在 URL 以 params 形式傳入
  • 使用 app.put 實現 PUT，將欲修改資料以 body 傳入

Conclusion

  • 以 app.put 實現 PUT
  • 以 req.params 讀取 id
  • 以 req.body 讀取 body

Reference

Express, app.put
