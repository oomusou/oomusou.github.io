如何計算 Array 平均值 ?

2020-11-21

計算 Array 的平均值為常見需求，但 Ramda 並沒有內建 avg()，但可自行組合。

Version

Ramda 0.27.1

converge()

  • 使用 sum() 計算 Array 總和
  • 使用 length() 計算 Array 的 element 個數
  • 使用 converge() 分別將 sum() 與 length() 結果傳給 divide() 計算平均

lift()

  • 使用 lift() 對 divide() 改變 signature
  • converge() 須以 Array 傳入 function，lift() 則以 variadic function 傳入 function

juxt()

  • 使用 juxt() 將 Array 先將過 sum() 與 length() 處理
  • 因為 juxt() 回傳為 Array，所以使用 apply() 將 divide() 改變能接受 Array

Wink-fp

Wink-fp 已經內建 avg() 可直接使用。

Conclusion

  • avg() 雖然是簡單應用，但可分別使用 converge()、lift() 與 juxt() 實現

Reference

stackoverflow, How to convert simple average function in javascript to pointfree form using Ramda ?

# Ramda

  • 使用 avg() 計算 Array 平均值
  • 使用 juxt() 讓 Argument 先經過 Function 處理
