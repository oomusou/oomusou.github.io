使用 lift() 對多 Array 做排列組合運算

2020-11-21

lift() 除了能將一般 Function 提升成能處理 Apply 的 function 外，還能將多個 Array 中的值做排列組合運算。

Version

Ramda 0.27.1

lift()

import { add, lift } from 'ramda'

lift(add)([1,2], [3, 4]) // ?

使用 lift() 對 lift() 加以提升後，發現新 function 能對兩個 Array 中所有 element 做排列組合運算：

  • 1 + 3 = 4
  • 1 + 4 = 5
  • 2 + 3 = 5
  • 2 + 5 = 6

ap()

import { add, ap } from 'ramda'

let lift = f => x => y => ap(ap([f], x), y)

lift(add)([1,2])([3, 4]) // ?

lift() 底層其實是呼叫 ap()，當原 function 的 arity 為 1 則呼叫一次 ap()，2 則呼叫兩次，以此類推。

因為 f() 為 curried function，因此第一次 ap() 執行完時，只是 iterate 完第一個 Array 中所有 element，此時還沒有結果，Array 中只是執行一半的 partial function。

執行第二次 ap() 時將以前次 partial function array 繼續 iterate 第二個 Array 中所有 element，此時 Array 中才有最後結果。

也因為經過兩次 ap() 的 iterate，相當於兩層 for loop，因此兩個 Array 中所有排列組合 element 都執行過 function。

Conclusion

  • 當需要對多個 Array 中所有排列組合經過相同 function 運算時，不必再使用兩層 for loop，只要將一般 function 透過 lift() 提升後即可
