WebStorm 安裝與設定

2020-11-22

WebStorm 基於 ECMAScript 生態設計，預設組態已經很好用，只要少數調整即可。

Version

ＷebStorm 2020.2.3

Installation

$ brew cask install jetbrains-toolbox

使用 Homebrew 安裝 JetBrains Toolbox 。

Script Script Location

Toolbox App Settings

  • 選擇 Generate shell scripts
  • 設定 Shell scripts location 為 /usr/local/bin

Soft Link for CLI

$ cd /usr/local/bin
$ ln -s webstorm ws

將來可使用 ws . 啟動 WebStorm。

Font Size

IDE

Preferences -> Appearance & Behavior -> Appearance

  • Theme：Darcula
  • Use custom font：Dialog
  • Size：16

Editor

Preferences -> Editor -> Font

  • Font：Fira Code Medium
  • Size：19
  • Line spacing： 1.2
  • Fallback font： Menlo
  • Enable font ligatures：true

Terminal

Preferences -> Editor -> Color Scheme -> Console Font

  • Use console font instead of the default：true
  • Font：Menlo
  • Size：17
  • Line spacing：1.0

Appearance

View -> Appearance

  • Toolbar：false
  • Tool Window Bars：false
  • Status Bar：false
  • Navigation Bar：false
  • Members in Navigation Bar：false
  • Details in Tree Views：false

Editor Tabs

Preferences -> Editor -> General -> Editor Tabs

  • Tab placement：None

Plugin

.env files support

支援 .env 格式檔案。

IDE Features Trainer

JetBrains 官方所設計的 WebStorm 教學課程。

IDEA Mind Map

在 WebStorm 內畫心智圖。

Indent Rainbow

讓縮排有顏色顯示更明顯。

JS GraphQL

支援 GraphQL。

Rainbow Brackets

使 () 與 {} 有顏色區別方便辨識。

Save Actions

存檔時啟動 reformat code 與 optimize import。

String Manipulation

各種字串格式轉換。

Tailwind CSS

支援 Tailwind CSS。

Tool Windows

View -> Tool Windows

重新設定 shortcut。

  • Git：⌘ + 0 顯示 Git window
  • Project：⌘ + 1 顯示 project window
  • Structure：⌘ + 2 顯示 structure window
  • Terminal：⌘ + 3 顯示 terminal
  • npm：⌘ + 4 顯示 NPM script
  • Run：⌘ + 5 顯示 NPM script 結果
  • Problems：⌘ + 6 顯示目前 file 所有 error

Project Tool

  • Open Files with Single Click：true
  • Always Select Openned File：true

Shortcut

Preferences -> Keymap -> Navigate

  • Go to File：⌘ P (較簡短，且與 VS Code 相同)

Disable Parameter Hint

Preferences -> Editor -> Inlay Hints -> JavaScript -> Parameter hints

  • Show parameter name hints：false

Preferences -> Editor -> Inlay Hints -> Vue -> Parameter hints

  • Show parameter name hints：false

在大部分語言使用 parameter hints 或許可增加可讀性，但 ECMAScript 習慣以 parameter 直接傳遞 object，因此 parameter hint 重要性不若以往，反而讓 code 變得更冗長

Conclusion

  • WebStorm 沒有辦法吃 macOS 的 system preferences 較為可惜，必須手動一一設定 font size
  • 將所有的 bar 關起來也不使用 tab，讓顯示區最大化增進工作效率
