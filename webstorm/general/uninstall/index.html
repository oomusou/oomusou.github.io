如何徹底移除 WebStorm ?

2019-11-19

除了使用 Toolbox 移除 WebStorm 外，其實一些設定檔仍然留在系統，本文列出所有設定目錄，可自行刪除徹底移除 WebStorm。

Version

macOS Catalina 10.15.1
WebStorm 2019.2.4

Windows

Windows 7, 8, 10

<SYSTEM DRIVE>\Users\<USER ACCOUNT NAME>\.<PRODUCT><VERSION>

Windows XP

<SYSTEM DRIVE>\Documents and Settings\<USER ACCOUNT NAME>\.<PRODUCT><VERSION>

Configuration

config

idea.config.path。

Caches

system

idea.system.path。

Plugins

config\plugins

idea.plugins.path。

Logs

system\log

idea.log.path。

Linux

~/.<PRODUCT><VERSION>

macOS

Configuration

~/Library/Preferences/<PRODUCT><VERSION>

idea.config.path。

Caches

~/Library/Caches/<PRODUCT><VERSION>

idea.system.path。

Plugins

~/Library/Application Support/<PRODUCT><VERSION>

idea.plugins.path。

Logs

~/Library/Logs/<PRODUCT><VERSION>

Idea.log.path。

Conclusion

  • 這些目錄也適用其他 JetBrains 產品

Reference

JetBrains, Directories used by the IDE to store settings, caches, plugins and logs
