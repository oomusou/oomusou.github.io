初めて (第一次)

2023-06-05

副詞 初めて 可表示 名詞 的 第一次，亦可表示 副詞 的 第一次地。

単語

はじめて [2]：名 第一次 副 第一次地 。

例文

第一次

彼はこれが初めての海外旅行だった。

這是他第一次出國旅行。

  • 彼(かれ) [1]：代 他
  • は：副助 表 主題
  • これ [0]：代 這個
  • が：格助 表 主語
  • 始(はじ)めて [0]：名 第一次
  • の：格助 以 名詞 形容 名詞
  • 海外旅行(かいがいりょこう) [5]：名 海外旅行
  • だ：助動 是
    • だった [1]：過去 是

私の初めての企画なので、部長に教えていただく。

這是我第一份企劃案，我請部長教我寫。

  • 私(わたし) [0]：代 我
  • の：格助 表 所有
  • 初(はじ)めて [0]：名 第一次
  • の：格助 以 名詞 形容 名詞
  • 企画(きかく) [0]：名 企劃
    • 企画な [0]：普 企劃
  • ので [0]：格助 表 原因
  • 部長(ぶちょう) [0]：名 部長
  • に：格助 表 對象
  • 教(おし)える [0]：動II 他動 辞書形 教
    • 教えて [0]：て形 教
    • 教えていただく[0]：我請 我請教

すみません、初めてなんだけど・・・

不好意思，我第一次來…(請幫我一下)。

  • すみません [4]：拜託 不好意思
  • 初(はじ)めて [2]：名 第一次
  • なんだけど [0]：接助 表 請求

第一次地

初めて富士山を見たことが忘れられない。

無法忘記第一次看到富士山。

  • 初(はじ)めて [2]：副 第一次
  • 富士山(ふじさん) [1]：名 富士山
  • を：助 接受 見る 的 賓語
  • 見(み)る [1]：動II 他動 辞書形 看
    • 見た [1]：た形 看
    • 見たこと [1]：名 看
  • が：格助 表 內容
  • 忘(わす)れる [0]：動II 他動 辞書形 忘記
    • 忘れられる [0]：可能形 能忘記
    • 忘れられない [0]：ない形 不能忘記
