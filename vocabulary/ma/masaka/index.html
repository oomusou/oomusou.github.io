まさか

2023-06-06

まさか 可表示 副詞 沒想到，可當 名詞 或動詞 使用。在口語常單獨使用 まさか 表示 不會吧！ 或 難道是？。

単語

まさか [1]：名 副 沒想到。

接続

  • まさか + V
  • まさか + とは + 思わなかった
  • まさか + の + N
