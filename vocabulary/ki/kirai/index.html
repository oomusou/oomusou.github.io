嫌い

2025-10-31

きらい [0]：な形容 討厭的。(不是 い形容)

接続

  • A + は + B + が + 嫌い + だ
  • A + は + Vる + の + が + 嫌い + だ
  • A + は + N + が + 嫌い + だ

文体

  • 丁寧形現在肯定：嫌いです
  • 丁寧形現在否定：嫌いではありません
  • 丁寧形過去肯定：嫌いでした
  • 丁寧形過去否定：嫌いではありませんでした
  • 普通形現在肯定：嫌いだ
  • 普通形現在否定：嫌いじゃない
  • 普通形過去肯定：嫌いだった
  • 普通形過去否定：嫌いじゃなかった

例文

山田さんは納豆が嫌いだと言った。

山田先生說過他討厭納豆。

きらい 雖然是 い 結尾，但不是 い形容詞，而是 な形容詞，因為 きらい 的 語源 為 嫌う -> 嫌います -> 嫌い，本為 名詞 兼 な形容詞，不是 い形容詞，只是剛好以 い 結尾

  • 山田(やまだ) [0]：名 山田
  • 納豆(なっとう) [3]：名 納豆
  • が：格助 表 主語
  • 嫌(きら)い [0]：な形容 討厭的
  • だ：助動 是
  • と：格助 表 內容
  • 言(い)う [0]：動I 自他 辞書形 說
    • 言った [0]：た形 說

安くてもグループ旅行が嫌いだ。

即使便宜我也討厭團體旅行。

  • 安(やす)い [2]：い形容 便宜的
    • 安くて [2]：て形 便宜的
    • 安くても [2]：即使 即使便宜的
  • グループ [2]：名 group、團體
  • グループ旅行(りょこう)：名 團體旅行
  • が：格助 表 主語
  • 嫌(きら)い [0]：な形容 討厭的
  • だ：助動 斷定
