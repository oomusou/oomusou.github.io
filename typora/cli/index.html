如何以 CLI 呼叫 Typora 開啟 Markdown ?

2019-09-05

隨著 CLI 的流行，在 Terminal 下以 code . 開啟 VS Code 非常方便，對於常用的 Markdown 檔案，是否也能在 CLI 以 t . 開啟 Typora 呢 ?

Version

macOS Mojave 10.14.6
iTerm2 3.3.2
Zsh 5.3
Typora 0.9.9.28 (3403)

Alias

.zshrc

alias t='open -a typora'

只要在你的 .zshrc 加上這個 alias，就可以在任何目錄

$ t .

以 Typora 開啟目前目錄

或

$ t foo.md

直接以 Typora 開啟某個檔案。

Conclusion

  • Typora 是非常好用的 Markdown 編輯器，有別於傳統左右視窗的 Markdown 編輯器，Typora 是 WYSISWYG，除了直接使用 Markdown 語法外，也可以搭配熱鍵，還支援自動 outline 功能
  • 透過簡單的 alias，就可以使用 CLI 方式開啟 Typora 了
