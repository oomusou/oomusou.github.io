使用 write() 寫入 State

2020-11-06

Vue 3 希望我們使用 .value 寫入 State 內部值，但這種方式不方便 Point-free，Vue3-fp 提供了 Curry Function 版本的 write()。

Version

Vue 3.0.0
Vue3-fp 0.0.5

Composition API

將 Sam 寫入 state 並顯示。

<template>
  {{ name }}
</template>

<script setup>
import { onMounted, ref } from 'vue'

export let name = ref('')

let mount = () => name.value = 'Sam'

onMounted(mount)
</script>

第 8 行

export let name = ref('')

使用 ref() 建立 name state，且其初始值為 Empty String。

10 行

let mount = () => name.value = 'Sam'

在 mount() 內以 .value 寫入 name state。

Point-free

<template>
  {{ name }}
</template>

<script setup>
import { always, pipe } from 'ramda'
import { onMounted, ref } from 'vue'

let write = ref => a => ref['value'] = a

export let name = ref('')

let mount = pipe(
  always('Sam'),
  write(name)
)

onMounted(mount)
</script>

第 9 行

let write = ref => a => ref['value'] = a

.value 並不適合 Point-free，故另外建立 curry function 的 effect()，並將寫入 data 放在最後一個 argument 方便 Point-free。

13 行

let mount = pipe(
  always('Sam'),
  write(name)
)
  • 使用 always() 將寫入值包成 function
  • 使用 write() 寫入 state

可發現使用 write() 後，擾人的 .value 都不見了。

Vue3-fp

<template>
  {{ name }}
</template>

<script setup>
import { always, pipe } from 'ramda'
import { onMounted, ref } from 'vue'
import { write } from 'vue3-fp'

export let name = ref('')

let mount = pipe(
  always('Sam'),
  write(name)
)

onMounted(mount)
</script>

Vue3-fp 已經內建 effect() 可直接使用。

write()
Ref a -> a -> a
將 value 寫入 state

Ref a：欲寫入的 state

a：欲寫入的 value

a：回傳 state，可繼續 Function Pipeline

Conclusion

  • 透過 .value 寫入 state 並不是適合 Point-free，因此另外建立了 write()
  • 很多人抱怨 Vue 2 是一堆 this，而 Vue 3 是一堆 .value，透過 Vue3-fp 的 effect() higher order function，就再也不用使用 .value 了
