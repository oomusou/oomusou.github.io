local_auth 啟動生物辨識登入

2024-10-15

生物辨識登入 已經成為手機登入的標配，local_auth 可 啟動 該裝置的生物辨識登入。

Version

Flutter 3.24
local_auth 3.2

local_auth

  • 安裝 local_auth package

Flutter

local_auth

Line 19

  • 偵測是否有可用的生物辨識
  • auth.canCheckBiometrics：硬體 是否支援生物辨識
  • auth.isDeviceSupported：用戶 是否有設定生物辨識
  • 當 canCheckBiometrics 與 isDeviceSupported 都為 true 時，才表示能啟動生物辨識登入

Line 36

  • _auth.authenticate：啟動生物辨識登入
    • localizedReason：iOS 必須傳入 localizedReason

Line 54

  • 整合 _hasBiometrics() 與 _authenticate() 啟動生物辨識登入
  • 當 isAuthAvailable 為 true 時，才會真正啟動生物辨識登入

Conclusion

  • 要啟動生物辨識前，必須先判斷裝置是否支持生物辨識，且用戶也已設定好生物辨識
