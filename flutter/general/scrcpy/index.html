scrcpy 將 Android 手機投影到 macOS

2024-10-24

Scrcpy 可將 Android 手機畫面投影到 macOS 上，特別適合開會分享畫面。

Version

scrcpy 2.7

Install Scrcpy

$ brew install scrcpy
  • 使用 Homebrew 安裝 scrcpy
export PATH=$HOME/bin:/usr/local/bin:$HOME/Flutter/3.24.0/bin:$HOME/Library/Android/sdk/platform-tools:$PATH
  • 由於 scrcpy 會使用 Android 的工具 adb，因此必須將 $HOME/Library/Android/sdk/platform-tools 放到 PATH 上
$ scrcpy -d
  • d：使用 USB 連線的 Android 手機

Conclusion

  • 因為平常 AVD 都會開著，此時若 USB 也連上 Android 手機，執行 scrcpy 時會抱怨不知道要連接哪一個裝置，-d 可明確告訴 scrcpy 要連上 USB 的 Android 手機
