設定 Flutter 開發環境

2020-11-15

Flutter 可同時開發 iOS 與 Android App，在 macOS 除了要安裝 Flutter SDK 外，還必須同時安裝 Xcode 與 Android Studio，最後可使用 Flutter Doctor 檢查開發環境是否設定成功。

Version

macOS Catalina 10.15.7
Flutter 1.22.4

Flutter SDK

從 Fluttter 官網下載最新版 SDK。

$ cd
$ unzip ~/Download/flutter_macos_1.22.4-stable.zip

將 SDK 解壓縮到 ~/flutter folder 下。

export PATH=$HOME/bin:/usr/local/bin:$PATH:$HOME/flutter/bin

將 $HOME/flutter/bin 設定到 path 下。

儲存後重新啟動 terminal。

Android Studio

使用 JetBrains Toolbox 安裝 Android Studio。

只要一直按 Next 安裝 Android toolchain 即可。

Android Studio -> Prefereces -> Pluggins

  • 安裝 Flutter pluggin，會順便安裝 Dart pluggin

安裝完重新啟動 Andriod Studio。

$ flutter doctor --android-licenses

接受 Android license，一路上按 y 繼續。

$ ln -s ~/Library/Application\ Support/Google/AndroidStudio4.1/plugins ~/Library/Application\ Support/AndroidStudio4.1

修正 Android Studio 4.1 symbolic link。

Xcode

由 App Store 安裝 Xcode。

$ sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer
$ sudo xcodebuild -runFirstLaunch

安裝完 Xcode 立即執行。

$ brew install cocoapods

安裝 cocoapods。

VS Code

安裝 Flutter extension。

Emulator

Android

在 Android Studio 開啟 Android simulator。

iOS

$ open -a Simulator

開啟 iOS simulator。

Flutter Doctor

$ flutter doctor

以 Flutter Doctor 檢查 Flutter 環境是否安裝成功。

Conclusion

  • 安裝 Flutter 開發環境尚稱平順，唯需同時安裝 Android Studio 與 Xcode，因此需要大量 SSD 空間
