Flutter 初體驗

2020-11-23

以 Flutter 建立最簡單的 Hello World，並在 iOS、Android 與 Web 執行。

Version

Flutter 2.2.0

Flutter Project

選擇 Create New Flutter Project。

Flutter SDK

  • 左側選擇 Flutter App
  • Flutter SDK path：設定 Flutter SDK 路徑

  • Project name：輸入 project 名稱
  • Platforms：預設會選擇 Android 與 iOS，自行選擇 Web

Hello World

lib/main.dart

import 'package:flutter/cupertino.dart';

main() {
  runApp(
    Center(
      child: Text(
        'Hello World',
        textDirection: TextDirection.ltr,
      ),
    ),
  );
}

  • main()：為 Flutter 起始 function，系統一開始以呼叫 main() 開始

  • runApp()：執行 Flutter

  • Center：使用 Center widget 將內容水平垂直置中，要將 widget 寫在 child property
  • Text：使用 Text widget 顯示 Hello World，且一定要指定 textDirection property，否則會錯誤

Flutter 建議在每個 widget 與 property 之後都加上 ,，如此存擋之後 Android Studio 會加以排版顯示出 widget tree 架構

iOS

在 iOS simulator 顯示。

Andriod

在 Android simulator 顯示。

Web

在 Chrome 顯示。

Conclusion

  • Flutter 以 widget 為主體，並以 widget tree 組合 UI
  • 相同的 codebase 可同時在 iOS、Android 與 Web 顯示
