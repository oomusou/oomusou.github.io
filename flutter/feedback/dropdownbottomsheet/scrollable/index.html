DropdownBottomSheet 顯示下方有捲軸選單

2024-10-29

Flutter 並沒有提供 DropdownBottomSheet 顯示下方有捲軸選單，必須自行使用 ModalBottomSheet 、 ListView 與 Scrollbar 實現。

Version

Flutter 3.24

Flutter

  • Android 與 iOS 都成功使用 DropdownBottomSheet 顯示下方有捲軸選單

DropdownBottomSheet

Line 12

  • options：定義下拉選單所顯示的資料

Line 36

  • SizedBox：ListView 本身並沒有 height 設定高度，需在外層使用 SizeBox 定義 height
    • height：可使用 MediaQuery 取得裝置高度動態決定 height 不寫死
  • Scrollbar：使 ListView 有捲軸
    • thumbVisibility：是否永遠顯示捲軸

Line 21

  • 由 options 產生 ListTile

Conclusion

  • ListView 本身無法設定高度，必須由外層的 SizedBox 設定
  • ListView 本身無法產生捲軸，必須由外層的 Scrollbar 產生

# Flutter

  • DropdownBottomSheet 顯示兩層選單
  • ModalBottomSheet 顯示由下方彈出 Modal
