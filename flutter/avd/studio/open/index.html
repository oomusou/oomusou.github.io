使用 Android Studio 啟動 AVD

2024-09-05

在 Android 有兩種啟動 Android Virtual Device 方式，結果稍微不一樣，但都支援 Flutter 的 Hot Reload。

Version

Android Studio 2024.1

Virtual Device Manager

  • 在 Welcome to Android Studio 下執行 Virtual Device Manager

  • 進入 Device Manager 後，選擇 AVD 執行

  • AVD 會以獨立的視窗執行

Device Manager in Project

  • 進入 Device Manager 後，選擇 AVD 執行

  • 在 Running Devices 按下 power 啟動 Android
  • AVD 以 Dock 形式執行在 Android Studio 內

Conclusion

  • 若想將 AVD 放在外接螢幕時，則建議在 project 外使用 Virtual Device Manager 啟動，如此可將 AVD 放在其他螢幕
  • 若想將 AVD 與 Android Studio 放在同一個螢幕，則建議在 project 內使用 Device Manager 啟動，如此同一個螢幕可以寫 Code 與看結果
  • 無論使用哪種方式，都可順利搭配 Hot Reload
