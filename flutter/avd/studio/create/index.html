使用 Android Studio 建立 AVD

2024-09-05

AVD 全名為 Android Virtual Device，為 Android 的模擬器，讓我們可測試不同 Android Device 與 API 版本，可在 Android Studio 內以 GUI 方式建立。

Version

Android Studio 2024.1

Virtual Device Manager

More Actions -> Virtual Device Manager

  • 使用 Virtual Device Manager 建立 AVD

Device Manager

  • AVD 管理介面

Create Vitual Device

  • +：建立 AVD

Select Hardware

  • 選擇要模擬的硬體

System Image

  • 選擇要使用的 image

SDK Component Installer

  • 下載 image

  • Image 下載完畢，按 Next 繼續

Android Virtual Device

  • 可自訂 AVD 名稱，按 Finish 完成

Run Device

  • |>：執行 AVD

  • Android 模擬器順利執行在 macOS

Conclusion

  • AVD 安裝過程尚稱平順，唯 Android Studio + AVD 同時開啟，記憶體已經接近 8GB，相當吃記憶體
