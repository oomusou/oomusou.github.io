Container 為 Widget 添加 Style

2024-09-05

當要對原本 Widget 加上 背景顏色、Margin、Padding、外框、Alignment、旋轉…時，可在 Widget 外加上 Container 實現。

Version

Flutter 3.24

Color

  • Hello World 的 背景色 為 土黃色
  • Text 並 沒有 提供屬性設定 color
import 'package:flutter/material.dart';

class Home extends StatelessWidget {
  const Home({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
        body: Center(
            child: Container(
                color: Colors.amber,
                child: const Text(
                    style: TextStyle(fontSize: 30), 'Hello World'))));
  }
}
  • 將 Text 包在 Container 內
  • color：設定 Container 的顏色為 Text 的背景色

Padding

  • Hello World 的四周圍保留有 padding
  • Text 並 沒有 提供屬性設定 padding
import 'package:flutter/material.dart';

class Home extends StatelessWidget {
  const Home({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
        body: Center(
            child: Container(
                color: Colors.amber,
                padding: const EdgeInsets.all(20),
                child: const Text(
                    style: TextStyle(fontSize: 30), 'Hello World'))));
  }
}
  • 將 Text 包在 Container 內
  • padding：設定 Container 對內的 padding

Margin

  • Hello World 對內除了保留有 padding 外，對外還保有 margin
  • Text 並 沒有 提供屬性設定 margin
import 'package:flutter/material.dart';

class Home extends StatelessWidget {
  const Home({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
        body: Center(
            child: Container(
                color: Colors.blue,
                child: Container(
                  color: Colors.amber,
                  padding: const EdgeInsets.all(20),
                  margin: const EdgeInsets.all(20),
                  child:
                      const Text(style: TextStyle(fontSize: 30), 'Hello World'),
                ))));
  }
}
  • 將 Container 包在 Container 內
  • margin：設定 Container 對外的 margin

Decoration

  • Hello World 的四周圍除了有 padding 外，四個角還有弧度
  • Text 並 沒有 提供屬性設定 borderRadius
import 'package:flutter/material.dart';

class Home extends StatelessWidget {
  const Home({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
        body: Center(
            child: Container(
      decoration: BoxDecoration(
        color: Colors.blue,
        borderRadius: const BorderRadius.all(Radius.circular(18.0)),
        border: Border.all(width: 1.0, color: Colors.blue),
      ),
      padding: const EdgeInsets.symmetric(vertical: 10, horizontal: 50),
      child: const Text(style: TextStyle(fontSize: 30), 'Hello World'),
    )));
  }
}
  • 將 Text 包在 Container 內
  • decoration：設定 Container 的 style
    • borderRadius：設定 border 的弧度
    • border：設定 border 的寬度、顏色

Alignment

  • Hello World 的位置 垂直靠左
  • Text 並 沒有 提供屬性設定 alignment
  • Container 的 長寬 受 alignment 影響，會自動研展成 外層容器 的 長寬
import 'package:flutter/material.dart';

class Home extends StatelessWidget {
  const Home({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
        body: Center(
            child: Container(
                alignment: Alignment.centerLeft,
                color: Colors.amber,
                child: const Text(
                    style: TextStyle(fontSize: 30), 'Hello World'))));
  }
}
  • 將 Text 包在 Container 內
  • alignment：設定子元素對於 Container 的 alignment，但 Container 的 width 與 height 會被影響成為外層容器的 width 與 height

Width、Height

  • Hello World 的位置 垂直靠左
  • Text widget 並 沒有 提供屬性設定 alignment
  • 重新設定 Container 的 width 與 height
import 'package:flutter/material.dart';

class Home extends StatelessWidget {
  const Home({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
        body: Center(
            child: Container(
                alignment: Alignment.centerLeft,
                width: 200,
                height: 100,
                color: Colors.amber,
                child: const Text(
                    style: TextStyle(fontSize: 30), 'Hello World'))));
  }
}
  • 將 Text 包在 Container 內
  • alignment：設定子元素對於 Container 的 alignment，但 Container 的 width 與 height 會被影響成為外層容器的 width 與 height
  • width：重新設定 Container 的 width
  • height：重新設定 Container 的 height

雖然重新設定 width 與 height 可以解決 width 與 height 被 alignment 更改的事實，但寫死 width 與 height 顯然不是最好的方法

Child Container

  • Hello World 的位置 垂直靠左，且 Container 雖著 Hello World 的長寬而變，並沒有寫死 width 與 height
import 'package:flutter/material.dart';

class Home extends StatelessWidget {
  const Home({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
        body: Center(
            child: Container(
                alignment: Alignment.centerLeft,
                child: Container(
                    color: Colors.amber,
                    child: const Text(
                        style: TextStyle(fontSize: 30), 'Hello World')))));
  }
}
  • 將 Text 包在 Container 內，並再次將 Container 包在 Container 內
  • 外層 Container 負責 alignment，內層 Container 負責 color

Transform

  • Hello World 的 Container 有傾斜角度
import 'package:flutter/material.dart';

class Home extends StatelessWidget {
  const Home({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
        body: Center(
            child: Container(
                color: Colors.amber,
                transform: Matrix4.rotationZ(0.2),
                child: const Text(
                    style: TextStyle(fontSize: 30), 'Hello World'))));
  }
}
  • 將 Text 包在 Container 內
  • transform：設定旋轉角度

Conclusion

  • 若沒對 Container 設定 有效 屬性，Android Studio 會提出警告
