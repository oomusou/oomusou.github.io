我的 macOS 基本設定

2024-10-01

macOS 內建設定大都已經很好用，只需少數客製化設定，特別將自己慣用設定記錄下來，適合初學者參考。

Version

macOS Sequoia

Wi-Fi

  • インターネット共有に接続する前に確認：オフ

Bluetooth

  • AirPods Pro
    • この Mac に接続：この Mac に前回接続していた場合

通知

  • 画面のロック中に通知を許可：オフ
  • アプリケーションの通知
    • JetBrains Toolbox
    • Setapp

  • バナー：オン
  • 通知音を鳴らす：オフ

サウンド

  • 起動時にサウンドを再生：オフ
  • ユーザインターフェイスのサウンドエフェクトを再生：オフ

一般

ソフトウェアアップデート

  • 自動アップデート：オン
  • ベータアップデート：オフ

自動アップデート

  • macOSのアップデートをインストール：オン
  • App Storeからのアプリケーションアップデートをインストール：オン

言語と地域

  • 優先する言語：日本語

外観

  • 外観モード：ダーク
  • アクセントカラー ：レッド
  • 強調表示色：レッド
  • サイドバーのアイコンサイズ：大
  • ウインドウで壁紙の色合い調整を許可：オフ
  • スクロールバーを表示
    • スクロール時に表示：オン
  • スクロールバーをクリックして
    • クリックされた部分へジャンプ：オン

アクセシビリティ

ズーム機能

  • キーボードショートカットを使って拡大縮小：オン
  • トラックパッドのジェスチャを使って拡大縮小：オフ

ディスプレイ

  • 透明度を下げる：オフ
  • アニメーション画像自動再生：オン

テキスト

テキストサイズ

  • テキストサイズ：カスタム
  • メニューバーのサイズ：大

  • テキストサイズ：17 pt
  • Finder：17 pt

ポインタ

  • カーソルのサイズ：3
  • ポインタの枠線のカラー：黒い
  • ポインタの塗りつぶしカラー：白い

ポインタコントロール

トラックパッドオプション

  • スクロールの速さ：6
  • ドラッグにトラックパッドを使用：オン
  • ドラッグ方法：3本指のドラッグ

マウスオプション

  • スクロールの速さ：6

コントロールセンター

コントロールセンターモジュール

  • 集中モード：メニューバーに非表示
  • サウンド：メニューバーに非表示
  • 再生中：メニューバーに非表示

バッテリー

  • メニューバーに表示：オフ

メニューバーのみ

  • Spotlight：メニューバーに非表示

時計

  • スタイル：アナログ

Siri と Spotlight

  • Siriに頼む：オフ

Spotlight

  • 検索結果
    • PDF書類：オン
    • アプリケーション：オン
    • システム設定：オン
    • その他：オン
    • 計算機：オン
    • 書類：オン
    • 変換：オン
    • 定義：オン

Spotlight の プライバシー

  • プライバシー：oomusou/MyBlog/source/_post

デスクトップとDock

Dock

  • 画面上の位置：右
  • ウインドウをしまうときのエフェクト：スケールエフェクト
  • Dockを自動的に表示/非表示：オン
  • 起動済みのアプリケーションにインジケータを表示：オフ
  • アプリの提案と最近使用したアプリをDockに表示：オフ

デスクトップとステージマネージャ

  • 項目を表示
    • デスクトップに：オフ
    • ステージマネージャ使用時に：オフ
  • 壁紙をクリックしてデスクトップを表示：ステージマネージャ使用時のみ
  • ステージマネージャ：オフ
  • デフォルトのWebブラウザ：Safari.app

ウィジェット

  • ウィジェットを表示
    • デスクトップに：オフ
    • ステージマネージャ使用時に：オフ
  • デフォルトのWebブラウザ：Safari

ウインドウ

  • ウインドウを画面の端にドラッグしてタイル表示：オフ
  • エキーを押しながらドラッグしてタイル表示：オフ
  • タイル表示されたウインドウを隙間を入れて配置：オフ

Mission Control

  • 最新の使用状況に基づいて操作スペースを自動的に並べ替える：オフ
  • アプリケーションの切り替えで、アプリケーションのウインドウが開いている操作スペースに移動：オン
  • ウインドウをアプリケーションごとにグループ化：オン
  • ディスプレイごとに個別の操作スペース：オン

ホットコーナー

  • 左下：ディスプレイをスリープさせる

ディスプレイ

  • 文字を拡大
  • 輝度を自動調節：オフ
  • True Tone：オフ

詳細設定

  • ポインタとキーボードを近くにあるすべてのMacまたはiPad間で移動することを許可：オン
  • ディスプレイの端を通過させて近くにあるMacまたはiPadに接続：オフ
  • 近くにあるMacまたはiPadに自動的に再接続：オフ

壁紙

  • カラー：ブラック
  • すべての操作スペースに表示：オン

バッテリー

  • 低電力モード：バッテリー使用時のみ

バッテリーの状態

  • バッテリー充電の最適化：オフ

オプション

  • バッテリー使用時はディスプレイを少し暗くする：オフ
  • 電源アダプタ使用時はディスプレイがオフのときに自動でスリープさせない：On
  • ネットワークアクセスによるスリープ解除：しない
  • バッテリー使用時のビデオストリーミングを最適化：オン

ロック画面

  • 使用していない場合はスクリーンセーバを開始：しない
  • 電源アダプタ接続時に使用していない場合はディスプレイをオフにする：しない
  • スクリーンセーバの開始後またはディスプレイがオフになったあとにパスワードを要求：しない

キーボード

  • 環境光が暗い場合にキーボードの輝度を調整：オフ
  • fn キーを押して：入力ソースを変更

テキスト入力

  • 入力ソース：編集

すべての入力ソース

  • メニューバーに入力メニューを表示：オン
  • Caps LockキーでABC入力モードと切り替える：オン
  • 文頭を自動的に大文字にする：オフ

日本語-ローマ字入力

  • タイプミスを修正：オン
  • 候補表示
    • フォント：BIZ UGGothic Bold
    • フォントサイズ：18
  • 推測候補表示：オン
  • 数字を全角入力：オフ

注音 - 繁体字

  • 候補表示：
    • 方向：縦方向
    • フォントサイズ：18
  • 予測入力を表示：On
  • 単語頻度を調整：On

音声入力

  • 音声入力：オフ

トラックパッド

ポイントとクリック

  • 軌跡の速さ：7
  • クリック：0
  • タップでクリック：オン

スクロールとズーム

  • 拡大/縮小：オフ

その他のジェスチャ

  • 通知センター：オン
  • アプリ Expose：4本指で下にスワイプ

Conclusion

  • 這些設定相當個人向，不見得適合所有人，但最少是我工作起來很順的設定方式
