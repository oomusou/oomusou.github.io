我的 Finder 基本設定

2024-08-12

本文記錄我的 Finder 基本設定。

Version

macOS Sonoma

設定

一般

  • 新規Finder ウインドウで次を表示：ダウンロード

タグ

  • Tag 全部刪除

サイドバー

  • AirDrop：オン
  • アプリケーション：オン
  • ダウンロード：オン
  • oomusou：オン
  • iCloud Drive：オフ
  • 共有：オフ

詳細

  • すべてのファイル名拡張子を表示：オン
  • 30日後にゴミ箱から項目を削除：オン
  • 検索実行時：：このMacを検索

よく使う項目

  • ダウンロード
  • アプリケーション
  • Setapp
  • oomosou
  • Dropbox
  • Code

  • comparison
  • spoken

表示

  • タブバーを表示：オン
  • パスバーを表示：オン

表示オプション

リストオプション

  • 表示順序：名前
  • アイコンサイズ：Large
  • 変更日：オン
  • サイズ：オフ
  • デフォトルとして使用

アイコンオプション

  • 表示順序：名前
  • アイコンサイズ：80x80
  • グリッド間隔：100x100
  • デフォトルとして使用
