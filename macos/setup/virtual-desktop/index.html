我的 Virtual Desktop 基本設定

2024-01-10

macOS 提供了 Virtual Desktop 機制，但開太多 desktop 又會造成管理上的雜亂，我只會開 3 個 desktop 使用。

Version

macOS Sonoma

Desktop 1

  • 查詢資料
  • Safari、KKTV、ミュージック、ポッドキャスト、Kaiteki、TV

Desktop 2

  • 主工作區
  • Telegram、WebP Converter、Term2、Sublime Text、ForkLift
  • Typora、Moji辞書、Gitfox、Firefox、WebStorm、VSCode

Desktop 3

  • 看結果或其他
  • 辞書 by 物書堂、辞書、Instgram、Facebook、Twitter
  • 写真、Reeder、やさしい日本語、Chrome

Conclusion

  • Virtual Desktop 除了在 Trackpad 用 四指左右滑 外，也可用 Ctrl + 左右鍵 切換
