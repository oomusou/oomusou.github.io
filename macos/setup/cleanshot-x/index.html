我的 CleanShot X 基本設定

2024-10-30

本文記錄我的 CleanShot X 基本設定。

Version

CleanShot X 4.6.1

Setapp

  • 使用 Setapp 安裝 CleanShot X

General

  • Start at login：On
  • Play sounds：Off
  • Show icon：Off
  • Export Location：Downloads
  • Show Quick Access Overlay：Only recording
  • Open Annotate tool：Only Screenshot

Wallpaper

  • Padding：0
  • Capture window shadow：Off

Shortcuts

  • All-in-one：⇧ ⌘ 5
  • Capture Area：⇧ ⌘ 4
  • Capture Fullscreen：⇧ ⌘ 3
  • Capture Window：⇧ ⌘ 2
  • Start / Stop Recording：⇧ ⌘ 7

# macOS

  • GestureDetector 取代 ListTile 顯示兩層選單
  • GestureDetector 實現 IconButton
