我的 Karabiner 基本設定

2025-01-31

自從 M2 Macbook Air 重新將 Fn 鍵回歸取代 Touch Bar 後，如何善用 Fn 鍵就成為重要的課題，畢竟 macOS 不像 Windows 常常會使用到 Fn 鍵。我們可使用 Karabiner-Elements 重新定義 Fn 鍵取代 Touch Bar 切換 app，並保留 ⌘ + Fn 執行原本的功能。

Version

Karabiner-Elements 14.12.0

Installation

  • 使用 Homebrew 安裝 Karabiner

Fn Keys

  • F1：Safari
  • F2：Android Studio
  • F3：WebP Converter
  • F4：iTerm2
  • F5：Typora
  • F6：FortLift
  • F7：Sublime Text
  • F8：MOJi 辞書
  • F9：ChatGPT
  • F10：WebStorm
  • F11：Gitfox
  • F12：Firefox
  • ⌘ F1：Decrease Volumn
  • ⌘ F2：Increase Volumn
  • ⌘ F3：Mission Control
  • ⌘ F4：Zoom
  • ⌘ F5：Capture Text by TextSnipper
  • ⌘ F6：Capture Screen by CleanShot X
  • ⌘ F7：Record Screen by CleanShot X
  • ⌘ F8：Play/Pause
  • ⌘ F9：Fast Forward
  • ⌘ F10：Mute
  • ⌘ F11：Decrease Volumn
  • ⌘ F12：Increase Volumn

Karabinar Configuration

karabiner.json

karabiner.json 位於 /Users/oomosou/.config/karabiner 目錄下。

Shell Command

Line 19

  • rules：若要使用 Complex Modification，要統一寫在 rules 下

Line 20

  • from.key_code : 設定 Fn
  • to.shell_command : 設定 Fn 要開啟的 app

Shortcut

Line 184

  • manipulators.from.key_code：設定 Fn 鍵
  • manipulators.to.key_code：設定 shortcut 主鍵
  • manipulators.to.modifiers：設定 shortcut 的輔助鍵
  • manipulators.type：shortcut 一定要設定為 basic

System Function

Line 36

  • manipulators.from.key_code：設定 Fn
  • manipulators.from.modifiers.mandatory：設定 Fn 的 modifier key
  • manipulators.to.consumer_key_code：設定系統功能

Conclusion

  • 事實上 Karabiner-Elements 的功能非常複雜，本文只使用了最基本的重新定義 Fn 鍵
  • Fn 鍵若要執行 shell command 可使用 fn_function_keys ，亦可使用 Complex Modification，本文統一使用 Complex Modification
  • F6 與 F7 因為很少用到原本功能，因此重新定義成 Capture Text 與 Sleep
  • F3 與 F4 目前能維持原本的 Mission Control 的 Spotlight，若將來有其他需求，亦可率先選擇這兩個按鍵自行定義
  • 當 Fn 鍵可搭配 ⌘ 後，則可同時保留 切換 app 與 原本功能，若有更多的 app 要切換，可搭配 ⌥，如此實用性已經可超越 Touch Bar
