一些在 macOS 增加字型大小的方法

2023-10-28

macOS 並沒有單一設定增加字型大小，而是分散在各處。

Version

macOS Sonoma

Display

  • Larger Text

Appearance

  • Sidebar icon size：Large

Accessibility

Display

  • Menu bar size：Large

Pointer

  • Pointer size：2

Finder

  • Icon size：Large
  • Text size：16

System Settings -> Accessibility -> Display

  • Text Size

  • Text Size：17 pt
  • Finder：17 pt

Safari

Advanced

  • Accessibility: Never use font size smaller than：12

Conclusion

  • 本文只是針對 Apple 自家的 macOS、Finder、Safari 進行討論，有些 app 可能要單獨進行設定
  • Sonoma 在 Accessibility 多了 Text Size 設定，可再針對 Finder 設定超越 16 的字型大小

  • 讓 Finder 字型大小超過 16
