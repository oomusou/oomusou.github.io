重建 Spotlight Index

2023-10-27

當 Spotlight 搜尋不到 App，或 System Settings 無法快速搜尋設定，則表示 Spotlight Index 有問題，可能需要重新建立 Index。

Version

macOS Sonoma

Rebuild Index

$ mdutil -sa
  • 顯示目前 Spotlight Index 的狀況，注意是否為 disabled
$ sudo mdutil -Ea
$ sudo mdutil -ai off
  • 刪除所有 index
  • 將 Spotlight Index 服務關閉
$ sudo mdutil -ai on
  • 將 Spotlight Index 服務開啟

如此就會重新 index，需要一些時間

Conclusion

  • 一般使用不會需要重新建立 Spotlight Index，目前是更新到 Sonoma 14.1 時，才發現 Spotlight 無法使用，連在 System Settings 下也無法快速搜尋設定，後來才驚覺 Spotlight Index 已經被停止，需重新啟動重新建立 Index
