我的 macOS 基本設定

2020-12-07

macOS 內建設定大都已經很好用，只需少數客製化設定，特別將自己慣用設定記錄下來，適合初學者參考。

Version

macOS Big Sur 11.0.1

General

Appearance 選擇 Dark。

Desktop & Screen Saver

Screen Saver

  • Start after：never 不啟動 screen saver

Dock

  • 選擇 Magnification 當 hover 過去可放大 icon
  • 選擇 Automatically hide and show the Dock 節省空間
  • 不選 Show recent applications in Dock 避免污染 dock

Mission Control

  • 不選 Automatically rearrange Spaces based on most recent use 避免桌面順序被改變
  • 選擇 Group windows by application 使 Mission Control 可顯示較多 app
  • 取消 Mission Control 與 Application windows 的 shortcut，因為 ⌃ ↑ 與 ·⌃ ↓ 太常使用，將這兩個組合釋放出來

Siri

不選 Show Siri in menu bar 節省 menu bar 空間。

按下 Siri Suggestions & Privacy...。

Safari

  • Show Siri Suggestions in App：false
  • Learn from this App：false

如此 Safari 在 favorites 下就不會顯示 Siri Suggestions

Language & Region

新增繁體中文注音輸入法 Zhuyin。

不選 Show Input menu in menu bar 節省 menu bar 空間。

Notifications

Turn on Do Not Disturb

  • 選擇 From 12:00 AM 到 12 : 00 AM，也就是整天都關閉 notification
  • 選擇 When the display is sleeping
  • 選擇 When mirroring to TVs and projectors

Accessibility

調整 Cursor size 至中間。

Enable dragging 選擇 three finger drag。

Screen Time

關閉 Screen Time。

Security & Privacy

選擇 Allow your Apple Watch to unlock your Mac。

Software Update

選擇 Automatically keep my Mac to update

Network

不選擇 Ask to join Personal Hotspots 自動連上 iPhone。

Sound

不選擇 Play user interface sound affects 避免在辦公室出現奇怪聲音。

Trackpad

  • 選擇 Tap to Click 以輕拍 trackpad 取代 click
  • 將 Click 選擇 Light

Displays

  • 選擇 Larger Text 顯示最大字型

  • 不選 Show mirroring options in the menu bar when available 節省 menu bar 空間

Battery

Battery

  • Show battery status in menu bar：false，將以 Endurance 取代
  • Slightly dim the display while on battery power：true
  • Optimize video streaming while on battery：true
  • Optimized battery charging：true

Power Adapter

  • Turn display off after：Never

  • Prevent computer from sleeping automatically when the display is off：true

Date & Time

選擇 Analog 節省 menu bar 空間。

Finder

Finder -> Peferences -> Tags

  • New Finder windows show：Downloads

Safari

General

  • Open “safe” files after downloading：false

Search

  • Smart Search Field: includes Safari Suggestions：false

Advanced

  • Smart Search Field：Show full website addres：true
  • Show Develop menu in menu bar：true

Conclusion

  • 這些設定相當個人向，不見得適合所有人，但最少是我工作起來很順的設定方式
