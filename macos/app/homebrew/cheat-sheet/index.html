Homebrew Cheat Sheet

2023-05-08

本文整理 Homebrew 常用指令。

Version

Homebrew 4.0.16

Install Homebrew

$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
$ (echo; echo 'eval "$(/opt/homebrew/bin/brew shellenv)"') >> /Users/oomusou/.zprofile
$ eval "$(/opt/homebrew/bin/brew shellenv)"

會安裝 Command Line Tools，需要一段時間。

Update Homebrew

$ brew update

更新 Homebrew 到最新版本。

Install App

$ brew install xxx
$ brew install --cask xxx

使用 Homebrew 安裝 app。

Upgrade App

$ brew upgrade xxx

使用 Homebrew 更新特定 app 到最新版本。

Upgrade All Apps

$ brew upgrade

使用 Homebrew 更新所有 app。

Uninstall App

$ brew uninstall xxx

使用 Homebrew 移除特定 app。

List All Apps

$ brew list

列書 Homebrew 所安裝的所有 app。

Cleanup Homebrew

$ brew cleanup --pruce=all

刪除所有暫存檔。
