  • Vimari：Free Vim on Safari

  • Dropover：Paid 將檔案拖曳到此管理

  • Reeder：Paid RSS 讀取器
  • Negative：Free 支援 Dark Mode 的 PDF Reader

  • iTerm2：Free macOS 最好用的 terminal

Web

  • Dropbox：Paid macOS 最好用的網路儲存空間
  • Telegram：Free macOS 最好用的 IM
  • Sublime Text：Free 我沒用很多，只用來修改簡單 markdown 與一般文字處理

  • JetBrain Toolbox：Free 管理 JetBrains 工具

  • Chrome：Free 市佔率最高的跨平台瀏覽器，用於開發
  • Firefox：Free 很好用的跨平台瀏覽器，用於日文學習
  • Parallel Desktop：Paid macOS 最佳 VM，可穩定執行 Windows 與 Linux
  • Karabiner：Free 可重新設定 Fn 鍵取代 TouchBar

  • Mosaic：Paid 可使用鍵盤快速設定視窗位置，並設定出不規則左右大小

  • One Switch：Paid 可在 menu bar 調整一些常用系統設定

  • Sip：Paid 色彩擷取工具
  • ForkLift：Paid 取代 Finder 的檔案管理工具
  • RapidAPI：Paid 符合 macOS 風格的 API 測試工具
