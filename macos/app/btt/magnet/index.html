使用 BetterTouchTool 取代 Magnet

2023-06-25

對於沒有 Touch Bar 的 MacBook，Magnet 算最容易使用的視窗管理工具，若已經安裝了 BetterTouchTool，其實可直接使用 BetterTouchTool 模擬 Magnet，減少安裝一個工具。

Version

BetterTouchTool 4.068

Magnet

Magnet 其實最常用的功能只有 3 個：

  • 視窗移到左半部：⌥ ⌃ ←
  • 視窗移到右半部：⌥ ⌃ →
  • 視窗最大化：⌥ ⌃ ⏎

這三個功能都可直接在 BetterTouchTool 設定 shortcut 達成。

Left Window

  • Shortcut：⌥ ⌃ ←
  • Trigger：Window Resize & Move (Maximize Window Left Half)

Right Window

  • Shortcut：⌥ ⌃ →
  • Trigger：Window Resize & Move (Maximize Window Right Half)

Maximize Window

  • Shortcut：⌥ ⌃ ⏎
  • Trigger：Window Resize & Move (Maximize Window)

Conclusion

  • BetterTouchTool 如此設定後，在使用上完全與 Magnet 相同
