BetterTouchTool 安裝與設定

2020-07-20

BetterTouchTool 為 Setapp 所提供輔助設定 Keyboard、Trackpad、Touch Bar … 的強力工具，由於功能太多，很難簡單介紹 BetterTouchTool，本文分享我在工作上所使用的設定。

Version

macOS Catalina 10.15.5
BetterTouchTool 3.389

Keyboard

新增控制 window 位置的 shortcut。

Trackpad

Safari 可在 trackpad 使用雙指 swipe right 與 swipe left 控制前進後退，但 Finder 與 ForkLift 卻不行，可用 BetterTouchTool 加以補強。

Touchbar

Touchbar 好用在於其客製化，可將常用控制與常用 app 直接放在 Touchbar 上加速自己的 workflow。

由左到右依次為：

  • Switch app
  • Fresh
  • Mission Control
  • App Expose
  • Close App
  • Connet to AirPods Pro
  • Sleep Display
  • Now Playing (Music / Safari)
  • Capture Screen

Installation

從我的 GitHub 下載 My BetterTouchTool Setting。

Preset: Default -> Import

載入MyBTT.bttpreset。

Conclusion

  • BetterTouchTool 的最大優點就是客製化超高，初學者可以此作為 template 改善自己的 workflow
