使用 Fn 切換 App

2023-11-11

自從 M2 Macbook Air 重新將 Fn 回歸取代 Touch Bar 後，如何善用 Fn 鍵就成為重要的課題，畢竟 macOS 不像 Windows 常常會使用到 Fn 鍵。我們可使用 Karabiner-Elements 重新定義 Fn 鍵取代 Touch Bar 切換 app。

Version

macOS Sonoma
Karabiner-Elements 14.12.0

Fn Keys

  • F1：Telegram
  • F2：Safari
  • F3：WebP Converter
  • F4：iTerm2
  • F5：Sublime Text
  • F6：FortLift
  • F7：Typora
  • F8：MOJi 辞書
  • F9：Gitfox
  • F10：Chrome
  • F11：Slidepad
  • F12：Firefox

Shell Command

Line 72

"fn_function_keys": [
  • fn_functions_keys : 若要使用 Fn 執行 shell command，必須寫在 fn_function_keys 下

Line 73

{
  "from": {
    "key_code": "f1"
  },
  "to": [
    {
      "shell_command": "open -a '/Applications/Telegra m.app'"
    }
  ]
},
  • from.key_code : 設定 Fn
  • to.shell_command : 設定 Fn 要開啟的 app

Shortcuts

Line 19

"rules": [
  • rules：若要使用 Fn 執行 shortcut，必須在 rules 寫 Complex Modification

Line 20

{
  "description": "Use F11 to Open Slide",
  "manipulators": [
    {
      "type": "basic",
      "from": {
        "key_code": "f11",
      },
      "to": [
        {
          "key_code": "p",
          "modifiers": [
            "shift",
            "control"
          ]
        }
      ]
    }
  ]
}
  • manipulators.type：shortcut 一定要設定為 basic
  • manipulators.from.key_code：設定 Fn
  • manipulators.to.key_code：設定要執行的 shortcut
  • manipulators.to.modifiers：設定輔助組合鍵

Conclusion

  • 事實上 Karabiner-Elements 的功能非常複雜，本文只使用了最基本的重新定義 Fn 鍵
  • Fn 若要執行 shell command，需寫在 fn_function_keys 下
  • Fn 若要執行 shortcut，則要使用 Complex Modification 寫在 rules 下
