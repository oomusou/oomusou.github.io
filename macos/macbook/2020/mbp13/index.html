MacBook Pro 2020 13.3" 使用心得

2023-12-03

Macbook Pro 2020 13” 與 Macbook Pro 2019 13” 從規格看起來似乎差異不大，但實際用起來仍有些在規格上看不到的差異，也順便談談選購 Macbook 的一些迷思。

Version

MacBook Pro 2020 13.3”

Keyboard

剪刀腳鍵盤

Macbook Pro 2020 最大變動首推鍵盤，從 2016 的蝴蝶鍵盤又改成 2015 之前的剪刀腳鍵盤，這使得 2020 較 2019 厚 0.02 inch，但視覺上是無感。

至於蝴蝶鍵盤與剪刀腳鍵盤的在使用上各有優缺點：

蝴蝶鍵盤

  • 優點：省力、厚度較薄、輸入較快
  • 缺點：較吵、容易因入塵而損壞、不耐打

剪刀腳鍵盤

  • 優點：安靜、耐打
  • 缺點：較費力、輸入較慢、厚度較厚

大部分人應該比較喜歡剪刀腳鍵盤，Apple 也全系列都改成剪刀腳鍵盤了。

Caps Lock

Caps Lock 位置並沒有變，但因為使用了剪刀腳鍵盤後，有了微妙變化。

因為中文會使用 Caps Lock 切換中英文輸入法，之前在蝴蝶鍵盤時代，中英文切換大概有 3 成失敗機會，因為 Caps Lock 位於左側邊緣，常會使用小指或無名指去按，因受力不均勻導致 Caps Lock 沒按成功而導致中英文切換失敗。

Macbook Pro 2020 改成剪刀腳之後，中英文切換的成功率幾乎 100%，擾人的中英文切換問題總算得到解決。

ESC

Macbook Pro 2020 總算迎回實體 ESC 鍵，若你是 Vim 愛用者，則實體 ESC 就非常重要，個人 Vim 用的不多所以感覺還好，但這也導致 Touch Bar 較 2019 短，會少放一個 icon。

倒 T 方向鍵

Macbok Pro 從 2016 的方向鍵改成左右較大，2020 又改成左右較小的倒 T 方向鍵，個人用起來則無感。

Trackpad

MacBook Pro 2020 的 Trackpad 也有了微妙變化，Apple 似乎調整了 forch touch 的回饋參數，導致 forch touch 用起來感覺更加真實。

在 Macbook Pro 2019 之前我並沒有使用 force touch，但 2020 愛上 force touch 的真實回饋感而使用。

Speaker

Macbook Pro 的 speaker 為 notebook 中最佳無庸置疑，但 Macbok Pro 2020 又更為進步，除了聲音更大外，立體感更加明顯，在相同音樂下常會聽到以前 Macbook Pro 2019 所聽不到的聲音，可媲美 Macbook Pro 16” 的 speaker。

32 GB Ram

Macbook Pro 2019 將 4 Core CPU 帶入 13” 後，2020 更進一步將 32 GB Ram 也帶進 13”，至此 13” 已經有原本 15” 才有的工作能力，可安心在 Parallel Desktop 下執行多個 VM。

CPU

若以 benchmark 比較 2019 與 2020 兩代 CPU，可發現 Intel CPU 進步不大，但實際上以 Node.js 執行 Hexo 對 700 多篇 markdown 轉譯成 HTML 時，仍可有感 Macbook Pro 2020 的 CPU 有變快。

Myth

接下來談談幾個選購 Macbook Pro 迷思。

Touch Bar

Touch Bar 與蝴蝶鍵盤為 Macbook Pro 2016 最具代表性設計，隨著剪刀腳鍵盤回歸，不少人覺得 Touch Bar 也是雞肋該廢除。

其實 Touch Bar 必須搭配 BetterTouchTool 才能發揮威力，可將 macOS 常用功能與常用 app 常駐在 Touch Bar，可大幅增進工作效率。

Space Gray

Macbook Pro 2016 迎來 space gray 之後，我在 2017 與 2019 仍使用 Macbook 的代表色 silver，但 2020 使用 space grary 之後，發現其與 macOS dark mode 超級搭配，有種軟體與硬體渾然天成感覺，這種黑又不會太黑的感覺很棒。

Macbook Air

隨著 Macbook Air 的進步，Air 與 Pro 在規格上越來越接近，因此很多人猶豫到底要買 Air 還是 Pro。

Air 與 Pro 有兩項決定性的差異：

  • Touch Bar：若你是 Touch Bar 愛用者，則一定要選 Pro
  • 兩個風扇：Air 只有單風扇，Pro 有雙風扇，若你要讓 Macbook 能溫度更低，則一定要選 Pro

Conclusion

  • MacBook 從 2016 改成蝴蝶鍵盤後導致很多人不願升級，但 2020 重回剪刀腳鍵盤後，可以不用再猶豫蝴蝶鍵盤了
  • MacBook Pro 2020 很可能是 Intel CPU 的末代 MacBook Pro，而 ARM CPU 的 Macbook 與其生態系預估也要三年才能穩定，所以 Macbook Pro 2020 仍然是很直得投資的 production 機種，兼具 Intel CPU 穩定與 32 GB ram 高效
  • MacBook Pro 2020 唯一遺憾是沒有如 Macbook Pro 16” 窄邊框，否則將是 Touch Bar 世代中最完美機種
