NVM Cheat Sheet

2020-12-19

NVM 是 Node 版本管理程式，可用來安裝/移除不同版本的 Node，還可以切換 Node 版本，若你大量依賴 Node 生態系，NVM 一定要安裝。

All Node Version

顯示所有 NVM 支援 Node 版本。

All LTS Version

顯示所有 NVM 支援 Node LTS 版本。

Install Latest Node

安裝最新版 Node。

Install Latest LTS

安裝最新 LTS 版 Node。

Install Version

安裝某個版本的 Node。

Uninstall Version

移除某個版本的 Node。

List All Installed Version

顯示系統所有安裝的 Node 版本。

Use Version

使用 version 切換到某個版本的 Node。

Switch Version

切換到 alias。

Add Alias

為某個 Node 版本取別名。

Remove Alias

移除 alias 。

Set Default Version

設定預設 Node 版本 (當 termial 啟動時自動切換的版本)。

Clear Cache

清除 cache。
