設定 Ubuntu Desktop Resolution

2020-06-06

若要在 Macbook Pro 13” 的 Parallel Desktop 內跑 Ubuntu Ubuntu Focal Fossa 20.04，到底什麼 Resolution 才是最佳設定呢 ?

Version

macOS Catalina 10.15.5
Parallel Desktop 15.1.4 (47270)
Ubuntu Desktop 20.04

Parallel Desktop

Configuration -> Hardware -> Graphics

  • Memory：512 MB (Recommended)
  • Resolution：(More Space)

當 Resolution 選擇 More Space 時，PD 才會以 Mabcook Pro 13” 原生的 2560 x 1600 送出，否則 Ubuntu 會出現 pointer cursor 過大情形

Ubuntu

Settings -> Displays

  • Resolutoin：選擇 2048 x 1280 (16:10)
  • Scale：選擇 200%

若 Scale 選 100%，會造成字體過小

Conclusion

  • Ubuntu 若依照這種 resolution 設定，則顯示相當完美，能充分發揮 Retina 螢幕優勢，且 Ubuntu 中英文顯示都很銳利
