如何在 Ubuntu 安裝 Docker ?

2020-06-10

Docker 本來就是 Linux 的技術，只要在 Ubuntu 裝上 Docker Engine，就可以在 Ubuntu 內使用 Docker。

Version

Ubuntu Focal Fossa 20.04
Docker 19.03.11

Install by Repository

$ sudo apt update
$ sudo apt install apt-transport-https ca-certificates curl gnupg-agent software-properties-common
$ sudo curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -
$ sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable"
$ sudo apt update
$ sudo apt install docker-ce docker-ce-cli containerd.io

Docker 官網介紹由 repository 安裝 Docker。

第 2 行

$ sudo apt install apt-transport-https ca-certificates curl gnupg-agent software-properties-common

安裝允許 apt 使用 HTTPS 所需 package。

第 3 行

$ sudo curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -

新增 Docker 官方的 GPG key。

第 4 行

$ sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable"

新增 repository。

第 5 行

$ sudo apt update

更新剛剛所新增的 respository。

第 6 行

$ sudo apt install docker-ce docker-ce-cli containerd.io

安裝 Docker 所需 package。

Snap

$ sudo snap install docker

亦可使用 Snap 安裝 Docker。

Conclusion

  • Docker 官網的方式較麻煩，可改用 Snap 安裝簡單安裝
