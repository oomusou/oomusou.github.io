Setting Up Docker on Ubuntu

2021-11-24

Docker is based on Linux, so we simply install the Docker engine on Ubuntu, and then we can use Docker on Ubuntu.

Version

Ubuntu 21.10
Docker 19.03.11

Parallel Desktop

Configure -> Hardware -> CPU & Memory

  • Hypervisor : Parallels
    • Adaptive Hypervisor : Yes
    • Enable nested virtualization : Yes

Snap

$ sudo snap install docker

Install Docker by Snap.

Docker

$ docker --version

Check Docker version.

Docker Compose

$ docker-compose --version

Check Docker Compose version.

Hello World

$ sudo docker run hello-world

Run hello-world to test Docker.

Conclusion

  • Installation steps on official Docker docs are more complicated; Snap is very simple
