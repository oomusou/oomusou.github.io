使用 Vite 安裝 Sanctuary

2021-04-20

Vue 3 在 Vue CLI 能正常執行 Sanctuary，在 Vite 卻不行，但有 Work Around 可暫時解決。

Version

Vue 3.0.5
Sanctuary 3.1.0
Vite 2.1.5

Vue Project

$ yarn create @vitejs/app vue3-sanctuary --template vue

使用 Vite 直接建立 Vue。

Add Sanctuary

$ yarn add sanctuary

使用 Yarn 安裝 Sanctuary。

Component

<template>
{{ result }}
</template>

<script setup>
import { ref } from 'vue'
import { write } from 'vue3-fp'
import { pipe, Left, Right, map, add, either, show, I } from 'sanctuary'

let div = x => y =>
  y === 0 ?
  Left ('Can not divide by zero') :
  Right (x / y)

let result = ref (0)

let transform = x => pipe ([
  div (x),
  map (add (1))
])

pipe ([
  transform (2),
  either (show) (I),
  write (result)
]) (1)
</script>

使用 Sanctuary 具代表性的 Either Monad。

10 行

let div = x => y =>
  y === 0 ?
  Left ('Can not divide by zero') :
  Right (x / y)

若除數為 0 則失敗回傳 Left()，成功則回傳 Right()。

22 行

pipe ([
  transform (2),
  either (show) (I),
  write (result)
]) (1)

組合出 IIFE：

  • either()：取出 Either Moand 內部值
  • write()：寫入 result state

執行會出現 Cannot read property 'custom' of undefined 錯誤訊息。

這只有 Vite 會有錯，Vue CLI 則正常

Work Around

$ yarn add --dev util

使用 Yarn 安裝 util。

node_modules/object-inspect/index.js

將 32 行從

var inspectCustom = require('./util.inspect').custom;

改成

var inspectCustom = require('./util.inspect.js').custom;

Vite Config

vite.config.js

import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

// https://vitejs.dev/config/
export default defineConfig({
  plugins: [vue()],
  define: {
    'process.env': {},
  }
})

第 7 行

define: {
  'process.env': {},
}

定義 process.env 為 {}。

HTML

index.html

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8"/>
  <link rel="icon" href="/favicon.ico"/>
  <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
  <title>Vite App</title>
  <script>
    window.global = window
  </script>
</head>
<body>
<div id="app"></div>
<script type="module" src="/src/main.js"></script>
</body>
</html>

第 8 行

<script>
  window.global = window
</script>

新增 JavaScript 將 window 指定給 window.global。

Sanctury 可正常執行。

Conclusion

  • 此為 Vite 的 bug，本文為 work around 讓 Sanctuary 暫時能在 Vite 執行
  • Sanctuary 在 Vue CLI 則正常執行 Vue 3
