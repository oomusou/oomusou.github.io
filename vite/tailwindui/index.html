使用 Vite 安裝 TailwindUI

2021-04-07

TailwindUI 基於 TailwindCSS，讓我們只要組合 Utility 就能建立 Component，不必再寫 CSS，且搭配 PurgeCSS 後，還可將沒用到的 CSS 拿掉。

Version

Vue 3.0.5
TailwindCSS 2.1.1

Vue Project

$ yarn create @vitejs/app vue3-tailwindui21 --template vue

使用 Vite 建立 Vue project。

Add TailwindCSS

$ yarn add --dev tailwindcss postcss autoprefixer @tailwindcss/forms

安裝 TailwindCSS 相關 package，會順便安裝 Vue 相關 package。

@tailwindcss/forms 為 Tailwind 官方所提供 plugin，由於 Tailwind 預設將所有 element 都 reset，因此若使用 form 相關 element 將使用大量 utility，而 @tailwindcss/forms 已經提供預設 style，可降低 utility 使用量

Add Config Files

$ npx tailwindcss init -p

在根目錄自動建立 tailwind.config.js 與 postcss.config.js。

Tailwind Config

tailwind.config.js

module.exports = {
  mode: 'jit',
  purge: ['./index.html', './src/**/*.{vue,js,ts,jsx,tsx}'],
  darkMode: false, // or 'media' or 'class'
  theme: {
    extend: {},
  },
  variants: {
    extend: {},
  },
  plugins: [
    require('@tailwindcss/forms'),
  ]
}

第 2 行

mode: 'jit',

在 development 階段啟動 JIT compiler。

第 3 行

purge: ['./index.html', './src/**/*.{vue,js,ts,jsx,tsx}'],

設定 PurgeCSS 管理檔案，將移除沒使用到的 CSS。

11 行

plugins: [
  require('@tailwindcss/forms'),
]

使用 @tailwindcss/forms plugin。

Add CSS File

src/index.css

@tailwind base;
@tailwind components;
@tailwind utilities;

新增 src/index.css。

Tailwind 會在 build-time 以標準 CSS 取代。

Main

main.js

import { createApp } from 'vue'
import App from './App.vue'
import './index.css'

createApp(App).mount('#app')

第 3 行

import './index.css'

在 main.js 引入 index.css。

Component

App.vue

<template>
  <input type="text" class="border-gray-300 rounded-md shadow-sm focus:ring-indigo-500 focus:border-indigo-500">
</template>

使用 Tailwind 實現 <input type="text">，會發現所有 style 都使用簡單的 utility 描述。

Development Mode

$ yarn dev

啟動 DevServer。

可發現 Tailwind 已經成功套用。

Production Mode

$ yarn build

使用 production mode 編譯 Vue 並啟動 PurgeCSS。

啟動 PurgeCSS 後，Tailwind 只剩下 8.13 kb，壓縮後更只剩下 2.01 kb。

Conclusion

  • Tailwind 2 在 Vue CLI 下比較 tricky，但在 Vite 卻異常簡單，推薦使用 Vite 取代 Vue CLI
  • 原本 JIT compiler 是獨立在 @tailwindcss/jit，目前已經整合進 Tailwind 2.1，只要在 tailwind.config.js 加上 mode: 'jit' 即可啟動
