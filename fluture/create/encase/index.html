使用 encase 建立回傳 Future 的 Function

2021-07-22

一般我們所寫的 Function 都只能回傳 ECMAScript 的原生 Type，而無法回傳 Future 這類 Monadic Type，透過 encase 可將一般 function 轉成成能回傳 Future 的 Function。

Version

Fluture 14.0.0

compose

data 為 Future，使用 chain 將 Future 與回傳 Future 的 function 綁定。

第 9 行

使用 compose 組合 resolve 與 inc 產生回傳 Future 的 inc_。

encase

第 9 行

也可使用 Fluture 的 encase，直接將一般 function 轉成回傳 Future 的 function。

encase :: Throwing e a r -> a -> Future e r
將一般 function 轉成回傳 Future 的 function

Throwing e a r：一般 function，可能產生 exception

a -> Future e r：回傳 Future 的 function

Conclusion

  • encase 使我們能將一般 function 轉成回傳 Future 的 function，而不需組合 resolve，非常好用

Reference

Fluture, encase
